### 問題の発見
- **和紙固有の課題**: 露光時間を延長（6.75分→10分→11分）してもInput 240-255が紙白になってしまい、階調表現が不可能

### 重要な発見: プリントとネガでは発想が逆

**最も重要な学び:**
デジタルネガとプリントでは**完全に逆の発想**が必要

```
2段階反転の原理:
デジタル画像 → ネガ（反転） → 光 → プリント（反転）

K値が高い = インク多い = ネガが濃い = 光を遮断 = プリントが薄い
K値が低い = インク少ない = ネガが薄い = 光を通す = プリントが濃い
```

**間違った発想（v1-v3）:**
- プリントで240-255が圧縮されている
- → ネガでも圧縮すべき（❌ 逆効果）
- → K値範囲を狭める（28000-29400）
- → 結果: さらに表現不可能に

**正しい発想（v5-v6）:**
- プリントで240-255が圧縮されている
- → ネガの濃度差が不足している
- → K値範囲を**拡張**すべき（✓ 正解）
- → 27000-32000に広げる
- → 結果: 階調表現の可能性

### v14-washi-v6 最終仕様

**設計パラメータ:**
```
Zone 1 (0-24):    v14と同じ（4148 K）
Zone 2 (24-240):  均等補間（106 K/step）
Zone 3 (241-255): ハイライト拡張（333 K/step）

K[240] = 27000  (v14: 29400, -2400 K)
K[255] = 32000  (v14: 33800, -1800 K)
240-255範囲 = 5000 K  (v14: 4400 K, +13.6%拡張)
```

**技術的成果:**
- 240-255のK値範囲を5000 Kに拡張（v14の4400 Kより広い）
- Zone 3勾配: 333.3 K/step（Zone 2の3.15倍）
- K[239]=26894 → K[240]=27000（連続性確保）
- 拡張効果: v14比1.14倍のネガ濃度差
