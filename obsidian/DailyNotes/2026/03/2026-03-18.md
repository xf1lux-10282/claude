## 📋 3/18 QTRカーブ最適化：露光レンジ拡大プロジェクト

### 問題の本質

**現象:**
- Input 240: 1.17D（良好）
- Input 248: 1.21D（良好）
- Input 255: 1.69D（問題！紙白1.25D制約超過）

**原因分析:**
1. Pd:Pt モル比が14:1（ほぼPd 93%）
2. Pd優位の配合 → 露光レンジが非常に狭い
3. H₂O₂過剰（10cc）→ 硬調化
4. 複合的な問題

---

### 解決策：新感光液システム（2つの配合を段階的にテスト）

#### **ステップ1: 新シュウ酸第二鉄の作成**

```
【材料】
シュウ酸〔無水〕: 5.5g
シュウ酸鉄II（二水和）: 6.0g
過酸化水素水30%: 5cc ← 10ccから半減（重要！）
蒸留水: 20cc ← 15ccから+5cc

【作成手順】
1. ビーカーにシュウ酸5.5g + シュウ酸鉄II 6.0g
2. 蒸留水15cc加えて溶解
3. 過酸化水素水30% 5ccをゆっくり添加（泡に注意）
4. 完全反応後、蒸留水で30ccに調整
5. 遮光瓶保存、24時間以上熟成

【化学量論的根拠】
H₂O₂理論量: 1.7cc
安全マージン（3倍）: 5.1cc
→ 5ccは適切（従来10ccは過剰）
```

### 理論的根拠

**Mike Wareの研究データより:**
- Pd 100%: 露光レンジ 2.0-2.4段（低コントラスト）
- Pt:Pd = 3:1: 露光レンジ 1.6-2.2段（中コントラスト）
- Pt 100%: 露光レンジ 1.2-1.5段（高コントラスト）

**現在の問題:**
- 現在配合（Pd:Pt = 14:1）: ほぼPd 100%相当
- しかしFe過剰（0.6cc）+ H₂O₂過剰（10cc）により硬調化
- Input 255で急激な濃度上昇（1.69D）

**改善戦略:**
- 推奨配合1: Pd:Pt = 3:1 + Fe削減 → バランス型
- 推奨配合2: Pd:Pt = 2:1 + Fe大幅削減 → 最大露光レンジ型
