## Obsidian Vault構成変更

### 変更理由
- iOSのObsidianはSyncthing（Mobius Sync）フォルダを直接開けない制約がある
- iCloudをマスターにすることでMac・iPhone間の同期が安定

### 新しい構成
```
iCloud (マスター)
    ↓ 自動同期
Mac Obsidian ←→ iPhone Obsidian
    ↓
NAS (バックアップ)
```

### 実施した作業

#### Mac側
1. Syncthingのobsidian同期を停止・削除
2. Macローカルの古いObsidianVault（`~/Documents/ObsidianVault`）を削除
3. NASのデータをiCloudにコピー（WEBフォルダ除外、約151MB）
4. Mac版ObsidianでiCloud Vaultを開く

#### iCloudのVaultパス
```
~/Library/Mobile Documents/iCloud~md~obsidian/Documents/obsidian
```

### 除外したフォルダ
- `WEB/` - ポートフォリオサイトのバックアップ（306MB）、iPhoneでは不要

### 残りの作業
- [x] VSCodeの作業フォルダをiCloudに変更
- [ ] iPhoneでiCloud Vaultを開く
- [ ] NASへの定期バックアップ設定（任意）

  --name syncthing-obsidian \
  --restart unless-stopped \
  -p 8385:8384 \
  -p 22001:22000/tcp \
  -p 22001:22000/udp \
  -p 21028:21027/udp \
  -v /volume1/docker/syncthing/config:/var/syncthing/config \
  -v /volume1/obsidian:/data/obsidian \
  syncthing/syncthing:latest
