# AP2管理WEB（社内業務管理アプリ・叩き台）

Alternative Process Lab / Ap2lab の **社内向け業務管理アプリ** の試作（叩き台）です。

## ファイル
- `ap2lab-manager.html` … 単一ファイルで動くWebアプリ。ダブルクリック、またはスマホのブラウザで開けます。

## 使い方
1. ファイルを開くと「ログイン画面」が出ます。
2. **4人の誰としてログインするか**を選ぶと、その人に見える画面だけが表示されます（権限分けの体感）。
3. 右上「切替」でいつでも別の人に切り替えできます。

## 4ロールの権限
| ロール | 見える範囲 |
|--------|-----------|
| オーナー（あなた） | 全部（経営数値まで） |
| 経営パートナー（管理者） | 全部 |
| 予約・顧客担当 | 予約・顧客・作業（経営数値は非表示） |
| 作業担当 | 割当作業と進捗のみ（顧客・売上は非表示） |

## 盛り込んだ要素
- 予約管理（暗室A/B・スタジオ、Web予約／店頭受付、Square事前決済／店頭精算）
- 作業・進捗（進捗バー付きTodo）
- 顧客リスト
- 経営ダッシュボード（売上・稼働率・Square入金ステータス）
- ホーム画面アイコン（PWA）を見据えたスマホUI

## 位置づけ
これは **「Glideで作ったらこう見える」という完成イメージの再現** です。
データはサンプルで、本物のログイン・保存はまだありません。
見た目と権限分けの感触を掴み、仕様を固めるための叩き台として使います。
仕様が固まったら、同じ構成を Glide で本番化します。
