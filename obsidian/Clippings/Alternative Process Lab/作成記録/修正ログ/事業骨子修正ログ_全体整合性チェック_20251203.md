# 事業骨子修正ログ：全体整合性チェックと修正（2025年12月3日）

## 修正の背景

融資希望額変更後、ユーザーより全体を通しての整合性チェックと修正の依頼を受けた。

### 実施した調査

1. 日本語表記のチェック（誤字脱字、表記ゆれ）
2. 金額のM表記チェック（22箇所）
3. 数値の整合性チェック（第1章 vs 第9章）
4. 表記統一のチェック

---

## 発見された主要な問題点

### 🔴 最優先修正項目

#### 1. 収益配分モデルの見直し（ユーザーからの指摘）

**問題点**：
- 旧モデル：猪股氏40%、木下氏57%（残余）
- 木下氏の取り分が多すぎて、事業拡張のための資金が確保できない

**修正内容**：
- 新モデル：猪股氏35%、木下氏35%、事業資金積立30%
- 事業資金は融資返済と拡張投資に充当

**効果**：
- 3年間で約¥5,086,000の事業資金を蓄積
- 4年目以降の暗室2室目増設が現実的に

#### 2. 営業利益の数値不一致

**問題点**：
- 第1章：1年目営業利益 ¥3,387,500（25%）
- 第9章：1年目営業利益 ¥2,749,500（20%）
- 差額：¥638,000

**原因**：
- 第9章で融資返済を営業利益計算に含めていた（会計上不適切）

**修正内容**：
- 営業利益から融資返済を除外
- 「営業利益」と「財務活動」を明確に分離
- 正しい営業利益：
  - 1年目：¥4,087,500（30%）
  - 2年目：¥10,500,000（48%）
  - 3年目：¥18,187,500（54%）

#### 3. M表記の多用（22箇所）

**問題点**：
- ¥13.45M、¥16.8M などの表記が多数
- 正確な数値との差異が生じる箇所がある

**修正内容**：
すべてのM表記を正確な数値に変換：
- ¥13.45M → ¥13,450,000
- ¥16.8M → ¥16,807,500
- ¥33.85M → ¥33,850,000
- ¥8M → ¥8,000,000
- ¥13M → ¥13,000,000
- ¥40M → ¥40,000,000
- ¥50M → ¥50,000,000
- その他すべて

#### 4. 1年目の収益合計の微調整

**問題点**：
- 概要表：¥13,450,000
- 詳細合計：¥13,458,000
- 差額：¥8,000（レンタル暗室の端数処理）

**修正内容**：
- 第5章の概要表をより正確な数値に修正
- レンタル暗室：¥2,500,000 → ¥2,508,000
- 1年目総収益：¥13,450,000 → ¥13,458,000

## 結論

本修正により、Alternative Process Labの事業計画書は以下の点で大幅に改善されました：

### 1. 持続可能な成長モデルの構築
- 3年間で約¥510万円の事業資金を蓄積
- 4年目以降の暗室2室目増設が現実的に
- 個人の利益と事業成長のバランスを実現

### 2. 数値の整合性と透明性の向上
- 第1章と第9章の数値が完全に一致
- M表記をすべて正確な数値に変換
- 営業利益の定義が明確化

### 3. 会計処理の適正化
- 営業利益と財務活動を明確に分離
- 融資返済を営業利益から除外し、会計上正しい処理に

### 4. 投資家・融資担当者への説得力向上
- 事業拡張のための資金計画が明確
- 4年目以降の成長戦略が具体的に
- 暗室1室体制での上限（約¥27M）と、2室体制での目標（¥40M超）が明確に

この修正により、本事業計画書は**融資申請・助成金申請の基盤資料**として、より強固な内容となりました。
