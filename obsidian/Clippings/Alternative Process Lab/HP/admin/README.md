# AP2 Lab イベント管理システム

## 概要

このシステムは、AP2 Labのウェブサイト上のイベント情報を簡単に管理するための管理画面です。

## 機能

### 1. イベントの追加・編集
- イベントタイトル、種別、募集状況、開催日時などを入力
- 技法の複数選択が可能
- 説明文の入力
- 画像URLの指定

### 2. イベント一覧表示
- 登録されたイベントを日付順に表示
- 各イベントの編集・削除が可能

### 3. データのエクスポート/インポート
- **JSONエクスポート**: イベントデータをJSON形式でバックアップ
- **JSONインポート**: バックアップから復元
- **HTML生成**: event.htmlに貼り付けるHTMLコードを生成

### 4. 自動非表示機能
- イベント開催日の翌日11:59に自動的に非表示
- event.htmlのページ読み込み時にJavaScriptで判定

## 使い方

### 初回セットアップ

1. ブラウザで `admin/event-manager.html` を開く
2. イベント情報を入力して「保存」をクリック

### イベントの追加

1. フォームに必要な情報を入力
   - **イベントタイトル**: イベントの名前
   - **イベント種別**: Workshop / Seminar / Other
   - **募集状況**: 募集中 / キャンセル待ち / 募集終了
   - **開催日**: YYYY-MM-DD形式
   - **開催時間**: 例: 13:00-17:00 (任意)
   - **技法**: 該当する技法にチェック
   - **説明文**: イベントの説明
   - **画像URL**: 画像ファイル名 (任意)

2. 「保存」ボタンをクリック

### イベントの編集

1. 一覧から編集したいイベントの「編集」ボタンをクリック
2. フォームに情報が読み込まれるので、修正
3. 「保存」ボタンをクリック

### イベントの削除

1. 一覧から削除したいイベントの「削除」ボタンをクリック
2. 確認ダイアログで「OK」をクリック

### HTMLコードの生成とevent.htmlへの反映

1. 「HTML生成」ボタンをクリック
2. `event_cards_YYYY-MM-DD.html` ファイルがダウンロードされる
3. ダウンロードしたファイルを開き、HTMLコードをコピー
4. `event.html` を開く
5. `<!-- Event Grid -->` セクション内の `<div class="event-grid">` の中身を全て削除
6. コピーしたHTMLコードを貼り付け
7. `event.html` を保存

### データのバックアップ

1. 「JSONエクスポート」ボタンをクリック
2. `ap2lab_events_YYYY-MM-DD.json` ファイルがダウンロードされる
3. このファイルを安全な場所に保存

### データの復元

1. 「JSONインポート」ボタンをクリック
2. バックアップしたJSONファイルを選択
3. 確認ダイアログで「OK」をクリック

## データの保存場所

イベントデータはブラウザの **localStorage** に保存されます。

### 注意事項

- データはブラウザごとに保存されます（Chrome、Safari、Firefoxなど別々）
- ブラウザのキャッシュをクリアするとデータが消える可能性があります
- **定期的にJSONエクスポートでバックアップを取ることを推奨します**

## 自動非表示機能の仕組み

### 動作仕様

- イベント開催日の**翌日11:59**を過ぎると自動的に非表示
- 例: 2026-06-01のイベント → 2026-06-02 11:59 以降に非表示

### 技術的な実装

1. `event.html` のJavaScriptがページ読み込み時に実行
2. 各イベントカードの `data-date` 属性を確認
3. 現在日時と比較して、過ぎたイベントをDOMから削除

### 制限事項

- **クライアント側の処理**: ページを開いた時にのみ判定
- サーバー側の自動処理ではない
- ユーザーのブラウザの時刻設定に依存

## トラブルシューティング

### イベントが保存されない

- ブラウザでJavaScriptが有効になっているか確認
- ブラウザのコンソールでエラーがないか確認
- プライベートモード/シークレットモードでは使用しない

### データが消えた

- バックアップJSONファイルから復元
- ブラウザのキャッシュクリアを避ける

### HTML生成がうまくいかない

- イベントが1件以上登録されているか確認
- ブラウザのダウンロード設定を確認
