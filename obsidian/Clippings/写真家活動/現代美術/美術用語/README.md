# 現代美術用語集

## 概要

現代美術を理解し、批評的に思考するための基礎的概念を体系的にまとめた用語集です。アートキュレーター、研究者、学生、およびアート愛好者の実践的参照資料として構成されています。

## 分類体系

### 1. 理論的概念
- コンテクスト（Context）
- インスタレーション（Installation）
- サイトスペシフィック（Site-Specific）
- パフォーマンス（Performance）
- コンセプチュアル・アート（Conceptual Art）
- 曼荼羅（Mandala）——視覚哲学と空間論

### 2. 批評・解釈の概念
- 批評的言説（Critical Discourse）
- 表象（Representation）
- イデオロギー（Ideology）
- フェミニズム批評（Feminist Criticism）
- ポストコロニアル批評（Postcolonial Criticism）

### 3. 制度・システム
- アートワールド（Art World）
- 文化的資本（Cultural Capital）
- アートマーケット（Art Market）
- オルタナティブ・スペース（Alternative Space）
- 公共芸術（Public Art）

### 4. メディア・技法
- ニューメディア（New Media）
- デジタルアート（Digital Art）
- ビデオアート（Video Art）
- ミクストメディア（Mixed Media）
- インタラクティブ・アート（Interactive Art）

### 5. キュレトリアルな概念
- 展示論（Exhibition Theory）
- 空間論（Spatial Theory）
- 観者論（Viewer Theory）
- 美術館論（Museum Studies）
- アーカイブ論（Archive Theory）

## 使用方針

**学術的厳密性**: 各用語の定義は学術的根拠に基づき、複数の視点を含む
**実践的応用**: 理論的説明に加え、具体的な応用例や実践への示唆を含む
**批評的視点**: 中立的記述にとどまらず、批評的・問題提起的な観点を含む
**継続的更新**: 現代美術の動向に合わせた定期的な内容更新

## 編集方針

- 各用語は独立したMarkdownファイルとして管理
- 相互参照による概念間の関係性を明示
- キュレーターの実践的視点を重視
- 具体例と理論の両方を含む構成
