## 使い方のヒント

### アーティストトークの準備
1. 作品のテーマを明確化（例: 時間、喪失、身体性）
2. 該当する哲学者のディレクトリーを開く
3. 「〜をXX的に語る」のセクションを参照
4. 自分の言葉に翻訳

### キュレーターとの対話
- 哲学者の名前を出すことで、作品の理論的強度を示せる
- ただし、衒学的にならないよう注意
- 「私はXXの影響を受けています」より、「XXが語ったYという概念は、私の作品のZという側面と共鳴します」

### ステートメント執筆
- 哲学的概念を直接使いすぎない
- しかし、概念を理解した上で、自分の言葉で表現
- 例: 「アウラ」とは書かずに「唯一性と距離の感覚」と書く

## 注意事項

### 哲学は「道具」
- 哲学は作品を語る言語を提供する
- しかし作品自体が最も雄弁
- 哲学に依存しすぎない

### 批判的距離
- ハイデガーのナチス関与
- プラトンの本質主義
- これらを知った上で、批判的に使用
