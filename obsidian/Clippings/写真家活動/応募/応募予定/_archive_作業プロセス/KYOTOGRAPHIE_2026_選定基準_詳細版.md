# KYOTOGRAPHIE 2026 選定基準 - 詳細版

**作成日**: 2026年3月22日
**目的**: ステートメントとの完璧な整合性を実現する選定基準の確立

---

## 🎯 全体方針: ステートメントの論理を作品で「構築」する

### ステートメントの核心

```
写真はこれらの条件を記録せず、物質的に構築する。
Photography does not document these conditions. It materially constructs them.
```

→ **選定される作品は、条件を「見せる」のではなく「構築」するものでなければならない**

---

## 📐 三部作の論理構造

### I. 月（可視性の条件）

**ステートメントの要求**:
```
可視性は光と闇の交渉の中で生じる。プラチナの光は像を紙の内部に留める。
眼は完全には見えないものを見ようとする。現れるものは部分的である。

Visibility emerges through negotiation between light and darkness.
Platinum's light embeds the image within the paper.
The eye adapts, striving to see what never fully appears.
What appears is partial.
```

**物質的構築の要件**:
1. **プラチナの特性**: 内部への吸収、沈み込み、固定的
2. **光と闇の緊張**: どちらか一方ではなく「交渉」
3. **部分的な現れ**: 完全には見えない、かろうじて見える
4. **眼の適応**: 見る者の努力を要求する

**選定基準（月3点）**:

**基準1: 光と闇の緊張度**
- 最も暗いもの（闇が優勢）: 1点
- 中間的なもの（緊張が最高）: 1点
- 相対的に明るいもの（光が優勢）: 1点
→ 3点で「交渉」のスペクトラムを構築

**基準2: プラチナの物質性**
- 内部への沈み込みを想起させる
- 表面ではなく深度を感じさせる
- 固定的だが不完全

**基準3: 構図と視覚的リズム**
- 中央配置 vs オフセンター
- 月の大きさの微妙な差異
- 3点が並んだときのリズム

**除外基準**:
- 月が「はっきり見える」もの（可視性の条件が機能している = NG）
- 闇または光のどちらかが完全に優勢なもの（交渉がない = NG）

---

### II. 雪（判断の条件）

**ステートメントの要求**:
```
判断は境界に依存する。雪は境界の機能を失わせる。
面は、光を反射し、見る角度によって異なる。
区別は可能だが確定には至らず、知覚は宙吊りになる。

Judgment depends on boundaries.
Snow suspends their function.
The surface reflects light and shifts with the viewer's position.
Distinction becomes possible yet cannot be fixed.
Perception remains suspended before conclusion.
```

**物質的構築の要件**:
1. **ゼラチンシルバーの特性**: 反射、表面性、角度依存
2. **境界の機能不全**: 区別できるが確定しない
3. **知覚の宙吊り**: 結論に至らない
4. **雪の作用**: 境界を失わせる

**選定基準（雪4点）**:

**基準1: 境界の機能不全のタイプ**
- タイプA「完全な消失」: 境界が見えない（ミニマル） - 1点
- タイプB「痕跡による探求」: 境界を探そうとするが失敗する - 1点
- タイプC「要素の断片」: 樹木・枝などで区別を試みるが確定しない - 1点
- タイプD「複数要素の宙吊り」: 複数の要素が関係を結べない - 1点

**基準2: ゼラチンシルバーの物質性**
- 反射性を想起させる（白の明度）
- 表面性を感じさせる
- 角度による見え方の変化を暗示

**基準3: 視覚的なバランス**
- ミニマル vs 要素あり: 2:2
- 縦構図 vs 横構図のバランス
- 4点が並んだときの緊張感

**除外基準**:
- 境界が「はっきり見える」もの（判断の条件が機能している = NG）
- 雪がない、または雪の効果が明確でないもの
- 意味が確定してしまうもの（例: 明確な物語性）

---

### III. 花（意味の条件）

**ステートメントの要求**:
```
意味は対象に内在しない。枠組みが揺らぐとき、可視なものは意味へ収まらない。
像は明瞭だが定着しない。

Meaning does not reside in objects.
When frameworks falter, the visible resists resolution.
The image is clear yet refuses to fix.
```

**物質的構築の要件**:
1. **プラチナの特性**: 固定性、明瞭さ、内部への沈み込み
2. **意味の不安定性**: 明瞭だが意味が定まらない
3. **枠組みの崩壊**: 文化的コード（生け花等）からの逸脱
4. **可視性 vs 意味**: 見えているのに理解できない

**選定基準（花3点）**:

**基準1: 枠組みの崩壊度**
- 最も「異様」なもの: 文化的枠組みから最も遠い - 1点
- 「中間的」なもの: 花/枯れ枝だと認識できるが意味が揺らぐ - 1点
- 「一見正常」なもの: 構図は整っているが何かがおかしい - 1点
→ 3点で意味の不安定性のスペクトラムを構築

**基準2: 容器と要素のバリエーション**
- 白い花瓶: 1点
- 透明な容器/皿: 1点
- その他（球体の組み合わせ等）: 1点
→ 異なる「枠組み」の試み

**基準3: プラチナの物質性 vs 意味の不安定性**
- 「明瞭に見える」（プラチナの固定性）
- しかし「意味が定まらない」（対象の不安定性）
- この矛盾が最も強いもの

**除外基準**:
- 意味が確定してしまうもの（例: 美しい生け花）
- 不明瞭なもの（プラチナの明瞭さに反する）
- 装飾的すぎるもの（「安全」に見える）

## 📊 選定プロセスの手順

### ステップ1: 月3点の選定

1. **12点を明度でソート**: 暗い → 明るい
2. **光と闇の緊張度を評価**: 1-10点
3. **3点を選定**:
   - 最も暗いグループから1点
   - 中間的な緊張度から1点
   - 相対的に明るいグループから1点
4. **構図のバランス確認**: 中央/オフセンター

### ステップ2: 雪4点の選定

1. **15点を境界の機能不全でカテゴライズ**:
   - A: 完全な消失
   - B: 痕跡による探求
   - C: 要素の断片
   - D: 複数要素の宙吊り
2. **各カテゴリーから1点選定**
3. **視覚的バランス確認**: ミニマル2点 vs 要素あり2点

### ステップ3: 花3点の選定

1. **22点を枠組みの崩壊度で評価**: 1-10点
2. **容器のタイプでグループ化**: 白い花瓶/透明/その他
3. **3点を選定**:
   - 各グループから1点
   - 枠組みの崩壊度が異なる3点
4. **意味の不安定性を確認**: 明瞭だが意味が定まらない

### ステップ4: 全体の整合性確認

1. **10点を並べる**: 月3 + 雪4 + 花3
2. **素材の対比が明確か**: プラチナ vs ゼラチンシルバー
3. **三条件の論理が伝わるか**: 可視性 → 判断 → 意味
4. **展示が想像できるか**: キュレーター視点

### ステップ5: 最終調整

- 必要に応じて1-2点を入れ替え
- ステートメントとの整合性を再確認
- 選定理由を記録
