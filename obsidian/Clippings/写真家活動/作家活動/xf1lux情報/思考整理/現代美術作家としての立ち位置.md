# 現代美術作家としての立ち位置

**基本姿勢**: 写真は手段でもありメディウムでもある

> 「画家が筆を持つのと同じで、カメラを持っている」

## 自己認識

- プラチナプリント作家
- 現代美術作家（写真を扱う）
- 展示場所: 限定しない（写真専門美術館、現代美術館、両方）

## 目標モデル: 杉本博司

### 杉本さんが「現代美術作家」として認識されている理由

1. **哲学的問いから始まる**
   - 海景: 「人類が最初に見た風景とは？」
   - 劇場: 「映画の全時間を1枚に圧縮したら？」
   - 建築: 「焦点をぼかすと本質が見えるか？」
   - → 写真は「問いに答えるための手段」

2. **シリーズという形式**
   - 1枚の傑作ではなく、システマティックな探求
   - On Kawara、Sol LeWittと同じコンセプチュアル・アート

3. **写真の限界への挑戦**
   - ピントを合わせない（建築）
   - 長時間露光で「見えないもの」を写す（劇場）
   - → 写真メディウムの探求でもある

4. **制作方法の明確さ**
   - 大判カメラ、長時間露光
   - プロセスが作品の一部

5. **歴史との対話**
   - 19世紀写真（Talbot、Le Gray）への参照
   - でも「写真史」ではなく「時間の哲学」として語る

### 杉本モデルと木下の対応

| 杉本博司 | 木下大輔 |
|---------|---------|
| 「人類最初の風景」 | 「感覚の共有可能性」 |
| 海景シリーズ | 三部作（BLUE MIND、RUTEN、Tsukiyukihana） |
| 長時間露光 | 密着プリント（データの物質化） |
| 時間の探求 | 感覚の探求 |
| Talbot、Le Gray | デジタルとアナログの架橋 |

### 杉本さんとの差別化

- **杉本**: 時間（量としての時間、概念的） / 観る人は「考える」
- **木下**: 感覚（質としての経験、現象学的） / 観る人は「感じる」

**補完関係、競合ではない**

## 絵画・陶芸作家との完全な対等性

### 各作家の根本的問い

```
Gerhard Richter（絵画）:
「絵画は現実を表現できるのか？」

Edmund de Waal（陶芸）:
「器は記憶を保持できるのか？」

木下大輔（写真）:
「写真は感覚を伝達できるのか？」
```

**完全に対等**

## 村上隆の戦略からの学び

- 「アート史における句読点を作る」の重要性
- **写真史ではなく美術史で戦うべき**
- 琳派を内面化しながら前面に出さない戦略

## 同じ問いを探求している作家たち

- **James Turrell**（光のインスタレーション）: 知覚の限界を探る
- **Olafur Eliasson**（現象の再現）: 知覚の不確定性
- **Wolfgang Laib**（自然素材）: 儀式的な制作プロセス
- **Ann Veronica Janssens**（霧）: 知覚の不確定性

### 決定的な差別化

```
Turrell、Eliasson: 人工的環境で知覚を「操作」
→ スタジオで光や虹を作る

木下: 写真で「記録」
→ 実在する場所、実在する瞬間
→ 「そこに本当にあった感覚」
```

**写真の指標性が、問いの核心になる**

## 重要な原則

### 制作優先

> 「60-70代になるまで売れなくても、何百枚、1000枚以上描き続けた」（村上隆）

**最も重要なこと**:
- 制作を続けること
- 作品を蓄積すること
- 実践を深めること

### 健全なパターン

- ❌ 新しい理解 → それに合わせて作品を作る（頭でっかち）
- ✓ 既にやっていること → その意味を発見する（説得力がある）
