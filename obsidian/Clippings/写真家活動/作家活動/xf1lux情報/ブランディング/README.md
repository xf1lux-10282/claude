## このディレクトリについて

このディレクトリには、作家活動におけるブランディング戦略、ポジショニング、マーケティング戦略に関する資料を保存しています。

## ファイル構成

### 戦略文書
- `ブランディング戦略_マスタープラン.md` - 全体戦略の統合版
- `杉本博司型戦略_最適化版.md` - 杉本博司を参考にした具体的戦略
- `村上隆から学ぶ戦略的示唆.md` - 村上隆の戦略分析

### ポジショニング
- `3つの顔_ポジショニング戦略.md` - 市場別のポジショニング
- `差別化ポイント.md` - 競合との差別化

### 実行計画
- `Phase1_基盤強化_2026-2027.md` - 短期アクションプラン
- `Phase2_国際展開_2027-2029.md` - 中期計画
- `Phase3_ブランド確立_2029-2035.md` - 長期ビジョン

### 参考資料
- `競合分析.md` - 杉本博司、村上隆等の分析
- `市場調査メモ.md` - アート市場の動向

## 使用方法

1. 新しいチャットを開始する際は、まず `ブランディング戦略_マスタープラン.md` を参照
2. 具体的な施策を検討する際は、該当するPhaseのファイルを確認
3. ポジショニングを明確にする際は、`3つの顔_ポジショニング戦略.md` を参照
