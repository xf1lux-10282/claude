# 3つの顔：市場別ポジショニング戦略

**作成日**: 2026-01-13
**目的**: 市場に応じたポジショニングの使い分け

---

## 戦略の原則

同じ作品・作家でも、**ターゲット市場によって強調すべき要素が異なる**。

重要なのは：
- ✓ 嘘をつかない（すべて真実）
- ✓ しかし強調点を変える
- ✓ 相手が理解できる言語で語る

---

## A. 国際現代美術市場向け

### ポジショニング

```
「Post-Digital Materialist addressing Anthropocene」
ポスト・デジタル時代における物質性の探求者、人新世に応答する
```

### キーワード

**使うべき言葉**:
- Anthropocene（人新世）
- Temporal Crisis（時間的危機）
- Material Turn（物質論的転回）
- Climate Testimony（気候証言）
- Post-Digital Materiality
- Hyperobjects（超対象）
- New Materialism（新唯物論）

**避けるべき言葉**:
- ❌ "Traditional Japanese technique"（伝統的日本技法）
- ❌ "Beautiful seasons"（美しい季節）
- ❌ "Cultural heritage"（文化遺産）

### ステートメント例（Short Version）

```
I materialize time.

Climate change has destabilized the thousand-year-old
system of seasonal perception. The osmanthus fragrance
that once announced my October birthday now signals
year's end.

Using platinum—a material that endures 500 years—
I record vanishing seasons. This is not documentation.
It is a critical intervention into time itself.

Through the "New 24 Solar Terms," I reconstruct
distorted seasons into new language, making visible
the temporal crisis we unconsciously experience daily.
```

### 強調すべき要素

1. **哲学的基盤**
   - Karen Barad（新唯物論）
   - Timothy Morton（超対象）
   - Henri Bergson（時間哲学）

2. **同時代性**
   - 気候変動という最も重要な問題への応答
   - データ（東京100年間の気温上昇）
   - 科学との対話

3. **批評性**
   - 「新24節気」という言語的介入
   - 受動的記録ではなく、能動的再構築
   - 展示空間での身体的体験（気温上昇の可視化）

4. **物質性**
   - 500年残るプラチナの倫理性
   - デジタルからアナログへの批評的回帰
   - 手作業・瞑想性

5. **日本美術（補助的に）**
   - 方法論の一つとして
   - 「日本的だから」ではなく「時間を捉えるのに最適だから」

## B. 写真芸術市場向け

### ポジショニング

```
「Contemporary Platinum Printer exploring Time」
時間を探求する現代プラチナプリント作家
```

### キーワード

**使うべき言葉**:
- Platinum/Palladium Print
- Post-Digital Photography
- Materiality and Process
- Archival Permanence（500年の寿命）
- Handcraft in Digital Age
- Tonal Range（階調表現）
- Alternative Process

**使える言葉**:
- ✓ "19th-century technique"（19世紀技法）
- ✓ "Hand-coated"（手塗り）
- ✓ "Unique prints"（唯一無二のプリント）

### ステートメント例（Short Version）

```
I create platinum prints exploring the materialization
of time.

In an era of digital immediacy, I choose a 19th-century
process—hand-coating each print, a practice requiring
days of devotion. Platinum endures 500 years,
transforming ephemeral moments into permanent matter.

My current work documents seasons displaced by climate
change. Each print is a temporal refugee, a witness to
beauty vanishing from its proper time.

This is not nostalgia. It is material philosophy.
```

### 強調すべき要素

1. **技法の卓越性**
   - プラチナプリントの歴史（19世紀〜）
   - 手塗りの技術
   - デジタルネガの精密性
   - 和紙との組み合わせ

2. **プロセスの重要性**
   - 制作に2-3日かかる瞑想的時間
   - 夜間のみの暗室作業
   - 化学反応との対話
   - 予測不可能性の美学

3. **ポスト・デジタル戦略**
   - デジタルを経由してアナログへ
   - 複製可能性の時代における唯一性
   - ベンヤミンの「アウラ」の回復

4. **系譜**
   - 杉本博司（時間の探求）
   - 井津建郎（プラチナ + 日本的主題）
   - Irving Penn（プラチナの現代的応用）
   - Edward Weston（古典的プラチナ）

5. **テーマ**
   - 時間の物質化
   - 500年という寿命の意味
   - 消えゆくものへの応答

## C. 日本・アジア市場向け

### ポジショニング

```
「現代における季節の記録者」
「日本的時間感覚を現代的に継承する作家」
```

### キーワード

**使うべき言葉**:
- 二十四節気
- 陰翳礼讃
- もののあはれ
- 月雪花
- 季節感覚の変容
- 文化的記憶
- 継承と革新

**使える言葉**:
- ✓ "日本の美学"
- ✓ "伝統と現代"
- ✓ "茶道の精神"
- ✓ "琳派の美学"

### ステートメント例（Short Version）

```
失われゆく季節を、500年残る形で記録する。

10月の誕生日を告げていた金木犀の香りは、
年々遅れ、いまでは年の終わりを告げるものになった。

千年以上続いた二十四節気という季節の体系が、
わずか一世代で崩壊しつつある。

プラチナプリントという永続的な技法で、
変容する季節の姿を記録し、
新24節気という新しい季節言語を構築する。

これは、次世代への文化的記憶の継承である。
```

### 強調すべき要素

1. **日本文化との深い対話**
   - 二十四節気の歴史的重要性
   - 谷崎潤一郎『陰翳礼讃』
   - 酒井抱一「本歌取り」
   - 茶花の伝統

2. **季節感覚の変容**
   - 個人的記憶（金木犀）
   - 文化的喪失（季語、俳句、茶道）
   - 次世代への継承

3. **日本美学の現代的再解釈**
   - もののあはれ（儚さへの感受性）
   - 琳派の「たらしこみ」との共鳴（予測不可能性）
   - 陰翳の階調表現

4. **技法の継承**
   - 19世紀の技法を21世紀に
   - 手仕事の価値
   - Alternative Process Labでの教育活動

5. **普遍性へ**
   - 日本だけの問題ではない
   - グローバルな気候変動
   - 人類共通の課題

## 使い分けの実践例

### 例1: 同じ作品の異なる説明

**作品**: 月雪花「花」シリーズ

#### A. 国際現代美術市場向け
```
"Flowers as Temporal Refugees"

Tea flowers photographed at night under moonlight-like
illumination. Each bloom has been displaced from its
proper season by climate change—arriving weeks early
or late, or ceasing to bloom entirely.

Printed in platinum (500-year lifespan), these images
become fossils of extinct seasons. The exhibition
spatially manifests Tokyo's century of warming:
each 1°C rise positions prints 5cm higher, forcing
viewers to constantly adjust their gaze—a bodily
experience of climate disruption.
```

#### B. 写真芸術市場向け
```
"Platinum Prints: Moonlit Tea Flowers"

Hand-coated platinum/palladium prints on handmade washi.
Each print requires 2-3 days of devotional labor:
coating, exposing, washing repeatedly in water.

The deep tonal range of platinum captures subtle
gradations inspired by Tanizaki's "In Praise of Shadows."
Night-only shooting under controlled illumination
creates contemplative, timeless quality.

Edition of 8. Archival permanence: 500+ years.
```

#### C. 日本・アジア市場向け
```
「月雪花 花シリーズ：季節の翳り」

二十四節気に基づき、鎌倉の茶房で選ばれた茶花を撮影。
しかし各花は、本来の開花時期から数週間ずれて咲いている。
気候変動によって追いやられた「時間的難民」である。

プラチナプリントの深い階調は、谷崎潤一郎『陰翳礼讃』の
思想を現代に継承する。失われゆく季節の美を、
500年残る物質に封じ込める。

これは記録であり、鎮魂歌であり、次世代への遺産である。
```

## 実践的ガイドライン

### 1. 相手を観察する

**初対面での質問**:
- "What brought you to this work?"
- どの要素に興味を示すか？
  - 技法？ → B（写真芸術市場）
  - テーマ？ → A（現代美術）
  - 日本文化？ → C（日本・アジア）

### 2. 柔軟に切り替える

同じ会話の中でも、相手の反応に応じて強調点を変える。

**例**:
```
相手「プラチナプリントは初めて見ました」
→ B（技法の説明）へ

相手「気候変動をテーマにした作品ですか？」
→ A（人新世・時間の危機）へ

相手「二十四節気って中国起源ですよね？」
→ C（日本での洗練、文化的継承）へ
```

### 3. 常に一貫性を保つ

**核心的メッセージは変わらない**:
- 時間を物質化する
- 500年残る物質で記録する倫理性
- 消えゆく季節への応答

**変わるのは強調点のみ**:
- A: 気候危機・批評性
- B: 技法・プロセス
- C: 日本文化・継承

## まとめ：3つの顔の統合

これらは別々の人格ではなく、**同じ作家の異なる側面**。

```
        A: 国際現代美術
        (哲学・批評性)
              ↑
              |
    あなた（核心）
    時間の物質化
         ↙        ↘
B: 写真芸術    C: 日本・アジア
(技法・物質性)  (文化・継承)
```

すべては「時間の物質化」という核心から派生している。
