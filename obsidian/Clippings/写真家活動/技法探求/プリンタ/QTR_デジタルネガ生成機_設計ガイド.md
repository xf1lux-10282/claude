## 概要

このドキュメントは、プラチナプリント用の**デジタルネガ生成専用機**を構築するための技術ガイドです。

### 基本コンセプト

従来のデジタルネガ制作:
```
画像 → カーブ補正 → プリント → ネガ
(毎回補正が必要)
```

デジタルフィルム方式:
```
プリンター + インク + QTR = 感材
画像 → 感材 → ネガ
(フィルムのように特性が固定)
```

### 目標

- **再現性の確保**: モニター環境や個人の感性に依存しない
- **作家の意思の保持**: 補正工程で意図が溶けない
- **安定した階調**: 特にハイライト域の滑らかさ
- **高いUV濃度**: プラチナプリントに必要な Dmax 2.0以上

## デジタルネガとデジタルフィルムの違い

### 従来のデジタルネガ

**アプローチ**:
- 画像ごとにトーンカーブやグラデーションマップで補正
- プリンター/モニター環境に合わせて調整
- 感性による「合わせ込み」作業

**問題点**:
1. モニター環境で補正値が変わる
2. 撮影環境で補正が異なる
3. プリンター(刷る人)の感性に支配される
4. 再現性が不安定

### デジタルフィルム方式

**アプローチ**:
- プリンター+インク+QTRカーブを「感材」として規格化
- 一度特性曲線を確立したら固定
- 画像側は「入力規格」だけを守る

**銀塩フィルムとの対応**:

| 要素 | 銀塩フィルム | デジタルフィルム |
|------|------------|----------------|
| 感材 | フィルム乳剤 | プリンター+インク+QTR |
| 特性 | H&D曲線 | QTRカーブ |
| 感度 | ISO | デジタルISO(露光時間) |
| 調整 | N現像(N-1, N, N+1) | QTRカーブバリエーション |
| 粒子 | 銀粒子 | ディザパターン |

### フィルムの特性曲線との類似

銀塩フィルム:
```
露光量 → 現像 → ネガ濃度
(H&D曲線: トー/直線部/ショルダー)
```

デジタルフィルム:
```
デジタル値(0-255) → QTRカーブ → ネガ濃度
(逆Sカーブ: ハイライト圧縮/中間調リニア/シャドウ圧縮)
```

## QTR (QuadTone RIP) の役割

### QTRとは

**QuadTone RIP**は、モノクロ専用の RIP (Raster Image Processor) ソフトウェアです。

**主な機能**:
1. プリンタードライバを迂回した直接制御
2. 各インクチャンネルの吐出量を個別制御
3. カスタムカーブによる階調設計

### QTRの構造

```
入力画像(0-255)
  ↓
QTRカーブ(特性曲線)
  ↓
各チャンネルへの吐出率
  ↓
Ch1: ULG = 30%
Ch2: LG  = 50%
Ch3: MG  = 20%
Ch4: PK  = 0%
  ↓
ディザ処理
  ↓
プリント
```

### QTRカーブ = 特性曲線

QTRカーブは、銀塩写真の**H&D曲線(特性曲線)**に相当します。

**定義するもの**:
- 入力濃度(0-255) → 各インクチャンネルの吐出割合
- 結果としてのネガ濃度
- 階調の形(Sカーブ、リニア等)

**重要な点**:
- カーブは画像側ではなくプリンター側に持つ
- 一度確立したら固定(=感材の特性)
- 画像は補正なし・γ1.0で入力

## インク構成の設計思想

### 通常のインクジェット vs ピエゾグラフィ vs デジタルフィルム

#### 通常インクジェット
```
黒 + CMY(カラーインク)
↓
グレーを合成
```

**問題**:
- 微妙な色被り
- 階調の粗さ
- ハイライトが濁る

#### ピエゾグラフィ(階調型)
```
ULG/LG/MG/DG/VDG/PK
(インク濃度が異なる)
↓
濃度差で階調を作る
```

**利点**: 非常に滑らかな階調
**欠点**: インク管理が複雑

#### デジタルフィルム方式(ノズル分散型)
```
Gray × 6-7 (同一インク)
PK × 2
↓
ノズル数で階調を作る
```

**利点**:
- インク管理が簡単
- 粒状性が低い
- 安定性が高い
- デジネガに最適

### P800での理想インク配置

#### 9スロット構成

元の純正インク配置:
```
Slot1: PK  (Photo Black)
Slot2: MK  (Matte Black)
Slot3: LK  (Light Black)
Slot4: LLK (Light Light Black)
Slot5: C   (Cyan)
Slot6: LC  (Light Cyan)
Slot7: VM  (Vivid Magenta)
Slot8: VLM (Vivid Light Magenta)
Slot9: Y   (Yellow)
```

#### デジネガ専用配置(推奨)

```
Slot1: ULG  (Ultra Light Gray)  ← LLK
Slot2: LG   (Light Gray)        ← LK
Slot3: MG   (Mid Gray)          ← MK
Slot4: MG   (Mid Gray)          ← C
Slot5: DG   (Dark Gray)         ← LC
Slot6: DG   (Dark Gray)         ← VM
Slot7: VDG  (Very Dark Gray)    ← VLM
Slot8: PK   (Photo Black)       ← Y
Slot9: PK   (Photo Black)       ← PK
```

**まとめ**:
- ULG × 1
- LG  × 1
- MG  × 2
- DG  × 2
- VDG × 1
- PK  × 2

#### 各インクの役割

**ULG (Ultra Light Gray)**
- 担当階調: 0-20
- 役割: ハイライトの滑らかさ
- 重要度: ★★★★★

**LG (Light Gray)**
- 担当階調: 20-40
- 役割: ハイライト〜中間調の接続
- 重要度: ★★★★★

**MG × 2 (Mid Gray)**
- 担当階調: 40-150
- 役割: 中間調の滑らかさ
- 重要度: ★★★★☆
- 2本使用の理由: ノズル分散による粒状性低減

**DG × 2 (Dark Gray)**
- 担当階調: 150-220
- 役割: シャドウ手前の階調
- 重要度: ★★★★☆
- 2本使用の理由: シャドウの安定性

**VDG (Very Dark Gray)**
- 担当階調: 220-240
- 役割: 深いシャドウ
- 重要度: ★★★☆☆

**PK × 2 (Photo Black)**
- 担当階調: 240-255
- 役割: 最大濃度(Dmax)確保
- 重要度: ★★★★★
- 2本使用の理由: UV遮光能力の最大化

### なぜPKを2本入れるか

プラチナプリントでは**UV濃度**が重要です。

**インクジェットの限界**:
```
1ノズル列 = インク吐出量の上限
```

**PK × 2の効果**:
```
黒インクのドット密度 ≈ 2倍
↓
Dmax増加
↓
UV遮光能力向上
```

**実測例**:
- PK × 1: Dmax ≈ 1.8
- PK × 2: Dmax ≈ 2.2

### なぜGrayを複数使うか(同一インク)

**銀塩フィルムとの類似**:

銀塩フィルムの乳剤層:
```
小粒子層
中粒子層
大粒子層
↓
粒子分布で階調を作る
```

デジタルフィルムのチャンネル:
```
Gray Ch1
Gray Ch2
Gray Ch3
Gray Ch4
Gray Ch5
Gray Ch6
↓
ノズル分散で階調を作る
```

**効果**:
1. 粒状性の低減
2. バンディングの防止
3. 滑らかなグラデーション
4. ノズル抜けの影響分散

## 特性曲線とカーブ設計

### デジタルフィルムのISO概念

銀塩フィルムには**ISO(感度)**があります。
デジタルフィルムも同じ概念を持つべきです。

#### デジタルISO の定義

```
デジタルISO =
一定の露光条件(UV量・感光液・紙)に対して、
標準露光で狙いのハイライト密度を得るために
必要なネガ濃度スケール
```

**ゾーンシステムとの対応**:
- Zone I: やっと乗る濃度
- Zone VIII: ハイライトの基準
- 標準露光で Zone VIII が適正になるように設定

#### システム感度の固定

一度以下を固定します:
- UV露光時間
- 感光液の調合
- 紙の種類
- ネガフィルム(Pictorico等)
- プリンター設定
- QTRカーブ

→ これが**デジタルフィルムのISO**になります

### N現像の概念

銀塩写真:
```
N現像: 標準
N+現像: コントラスト増
N-現像: コントラスト減
```

デジタルフィルム:
```
Nカーブ: 標準(基準)
N+カーブ: シャドウ強調
N-カーブ: ハイライト保持
```

**運用**:
- 通常作品: Nカーブ固定
- 個別作品: 露光を微調整(±1/3〜2/3 stop相当)
- 特殊な場合のみ: N+/N- カーブを使用(2-3種類だけ用意)

### Sカーブの設計

#### なぜSカーブが必要か

**人間の視覚特性**:
- Weber-Fechner の法則
- 暗部の変化に敏感
- 明部では変化を感じにくい

**結果**:
```
リニア階調 → フラットに見える
Sカーブ → 自然に見える
```

#### プラチナプリントの特性

プラチナプリントは銀塩より:
- ハイライトが長い
- シャドウが急

**理想のネガカーブ(逆Sカーブ)**:
```
入力値
│
│        ／
│      ／
│    ／
│  ／
│／
└────────── ネガ濃度

構造:
・ハイライト: 圧縮(0-20を小さな濃度差に)
・中間調: リニア(20-180をそのまま)
・シャドウ: 軽い圧縮(180-255を緩やかに)
```

### 256ステップ測定法

#### γ1.0での測定の重要性

**補正なし・γ1.0で出力する理由**:
```
プリンター + フィルム + プラチナ紙
すべての特性をそのまま測定できる
```

もし途中で補正すると:
```
測定値が歪む
↓
正しいカーブが作れない
```

#### 測定手順

1. **γ1.0のネガ作成**
   ```
   0-255の256ステップ
   補正なし
   リニアガンマ
   ```

2. **プラチナプリント**
   ```
   同一条件で露光
   ・UV露光時間固定
   ・感光液固定
   ・紙固定
   ・現像・クリアリング固定
   ```

3. **濃度測定**
   ```
   方法: スキャナーでRGB値取得
   変換: L値 → I濃度
   計算: Density = log10(255 / pixel_value)
   ```

4. **特性曲線の作成**
   ```
   横軸: 入力値(0-255)
   縦軸: プラチナ濃度
   ```

5. **逆カーブの生成**
   ```
   理想濃度 → 必要ネガ濃度
   の逆算
   ```

6. **QTRカーブへの変換**
   ```
   入力値 → 各チャンネル吐出率
   ```

### カーブ設計の実例

#### 実運用カーブ: xf1シリーズ (PX1V + Kチャンネルのみ)

**xf1-v7-NA (ナチュラル)**:

```
実用範囲: Input 0-255 全範囲

主要ポイント:

```

**xf1-v5-HC (ハイコントラスト)**:

**xf1シリーズの特徴**:
- 線形補間(np.linspace)による完全均一Gradient
- バンディングゼロ達成
- Kチャンネルのみで高品質実現
- CISS改造不要

#### 理想的な階調分担(9チャンネル、P800等)

```
入力値    担当インク    役割
0-20      ULG          ハイライト専用
20-40     LG           ハイライト〜中間調
40-90     MG           中間調前半
90-150    MG           中間調後半
150-200   DG           シャドウ手前
200-230   DG           シャドウ
230-245   VDG          深いシャドウ
245-250   PK           最大濃度
250-255   PK           Dmax確保
```

#### ハイライトの処理

**重要**: 0-30の階調が最も重要

プラチナプリントでは:
```
ハイライトが長い
↓
0-30の微小な差が
プリント階調に大きく影響
```

**設計**:
- ULG と LG だけを使用
- PK は一切使わない(粒状性の問題)
- 非常に緩やかなカーブ

#### シャドウの処理

**目標**: Dmax 2.0以上

**手段**:
- PK × 2 を使用
- 吐出量を最大化
- ただし階調は保持

## ディザ設定

### ディザとは

**定義**:
```
1ドットを
どの位置に
どの頻度で配置するか
を決めるアルゴリズム
```

インクジェットの階調生成:
```
濃度 = インク量 + ドット密度
```

### ディザ方式の種類

#### Ordered Dither(規則的ディザ)

**特徴**:
- 規則的なパターン
- 計算が単純

**デメリット**:
- 粒状が見える
- パターンが目立つ
- **デジネガではノイズになる**

#### Error Diffusion Dither(誤差拡散)

**特徴**:
- ランダムに近い配置
- ドットパターンが見えない

**デメリット**:
- 処理が重い

**デジネガでは必須**:
```
ネガのドット
↓
プラチナプリントの粒子
として拡大される
```

### デジネガに最適な設定

#### 推奨設定

```
Resolution : 1440 dpi
Dither     : Error Diffusion
Ink limit  : QTRカーブで調整
Mode       : 1440 × 1440
```

#### なぜ2880ではないか

**2880 dpi の問題**:
```
ドットが重なる
↓
インク拡散
↓
ネガ濃度誤差
```

**1440 dpi の利点**:
```
適度なドット間隔
↓
濃度安定
↓
粒状性とのバランス
```

### ディザと銀塩粒子の対応

**銀塩フィルム**:
```
粒子配置 → 粒状性
```

**デジタルフィルム**:
```
ディザパターン → 粒状性
```

**つまり**:
```
ディザ = デジタル粒子
```

```

・破綻しない範囲で
・ガンマ補正なし

・16bit TIFF
・グレースケール
・埋め込みプロファイルなし
```

#### ネガ出力

**Print-Tool使用 (PX1V実運用)**:
```
1. QTRカーブ選択:
   - 標準: PX1V-PtPd-xf1-v7-NA
   - 高コントラスト: PX1V-PtPd-xf1-v5-HC
2. 解像度: 1440 × 1440
3. ディザ: Error Diffusion
4. カラーマネジメント: OFF
5. 出力
```

#### プラチナプリント

**固定条件 (xf1シリーズ基準)**:
- 露光機: PGI UV-Printer
- 露光時間: 7分00秒
- 感光液調合:
  - シュウ酸第二鉄: 0.60 cc
  - パラジウム溶液: 0.675 cc
  - 塩化白金酸ナトリウム 20%: 0.25 cc
  - モル比 Pt:Pd ≈ 1:2.1
- 紙: Arches Platine 310gsm
- 現像: クエン酸カリウム飽和溶液 1-2分
- クリアリング: EDTA 1% 5分
- 水洗: 5分 + 20-30分

**微調整**:
- 個別作品で露光時間を±15-30秒調整
- それ以外は変えない

### N+/N- カーブの使用

**通常**: Nカーブ固定

**特殊な場合のみ**:
- N+カーブ: ハイコントラストシーン
- N-カーブ: ローコントラストシーン

**2-3種類だけ用意**:
- これ以上増やすと感材の意味がなくなる

## トラブルシューティング

### CISS関連

#### ノズル欠け(最頻出)

**症状**:
- 特定ラインが出ない
- ランダムに欠ける

**原因の切り分け**:

**毎回同じチャンネルが欠ける**:
```
→ そのチャンネルの詰まり
→ ヘッド側の問題
```

**ランダムに欠ける**:
```
→ 気泡・負圧
→ CISS供給側の問題
```

**時間で回復/悪化**:
```
→ 温度・内圧
→ ベント・沈殿の問題
```

**対策**:
1. タンク高さ調整(ヘッドより5-8cm下)
2. ベント栓の確認
3. チューブ内気泡の除去
4. ダミーカートリッジの清掃
5. 定期的なノズルチェック

#### 濃度ムラ

**症状**:
- 日によって濃度が変わる
- ベタ面にムラ

**原因**:
- 負圧の変動
- タンク高さ
- 温度変化

**対策**:
1. タンク位置の固定
2. 室温の安定
3. ベントの定期清掃

### 階調の問題

#### ハイライトが急激

**原因**:
- ULG/LG の吐出不足
- カーブ設計の問題

**対策**:
1. 0-30階調のカーブを再設計
2. ULG専用チャンネルの追加
3. 256ステップ再測定

#### シャドウが締まらない

**原因**:
- UV濃度不足
- PK吐出量不足

**対策**:
1. PK × 2 の確認
2. インクのUV遮光性確認
3. カーブのPK割当増加

#### 粒状性が目立つ

**原因**:
- ディザ設定
- Gray分散不足

**対策**:
1. Error Diffusion確認
2. Gray複数チャンネル使用
3. 解像度確認(1440推奨)

### カーブの問題

#### 全体的にフラット

**原因**:
- Sカーブ不足
- リニア過ぎる

**対策**:
1. ハイライト圧縮を増やす
2. 逆Sカーブの強化
3. プラチナ側の特性再確認

#### 階調が飛ぶ

**原因**:
- カーブが急峻すぎる
- チャンネル分担の問題

**対策**:
1. 256ステップ再測定
2. 滑らかなカーブ再設計
3. チャンネル割当の再調整

## 参考: 測定とキャリブレーション

### スキャナー測定の注意点

#### リニアスキャン設定

**必須**:
- 自動補正OFF
- ガンマ補正OFF
- 露出補正OFF

**理由**:
```
内部ガンマがあると
濃度カーブが歪む
```

#### スキャナーキャリブレーション

より高精度にするには:
```
1. 既知濃度のステップウェッジをスキャン
2. スキャナーの特性曲線を測定
3. 補正式を作成
4. プラチナプリント測定に適用
```

### I濃度系の使用

**I濃度 (ISO visual density)**:
- 人間の視覚感度に合わせた濃度
- 白黒写真で標準的に使用

**UV濃度との関係**:
- UV濃度計は非常に高価
- 実務ではI濃度を代替指標として使用
- 相関は完全ではないが実用上は有効

## まとめ

### デジタルフィルムの本質

```
プリンター + インク + QTRカーブ = 感材

特性曲線を固定
↓
作家の意思を工程に保持
↓
再現性の確保
```

### 銀塩写真との完全な対応

| 要素 | 銀塩 | デジタルフィルム |
|------|------|----------------|
| 感材 | フィルム | プリンター+インク |
| 感度 | ISO | デジタルISO |
| 特性 | H&D曲線 | QTRカーブ |
| 粒子 | 銀粒子 | ディザ |
| 調整 | N現像 | N±カーブ |

### 重要原則

1. **感材を固定する**
   - プリンター設定
   - インク構成
   - QTRカーブ
   - 解像度・ディザ

2. **画像は規格に合わせる**
   - γ1.0
   - 補正なし
   - Scene-referred

3. **測定で管理する**
   - 256ステップ
   - 定量的評価
   - 感性に依存しない

4. **階調設計の優先順位**
   - ハイライト(0-30) > 中間調 > シャドウ
   - プラチナはハイライトが命
