## 改造の概要

### 改造の目的

Epson SureColor P800を**プラチナプリント用デジタルネガ生成専用機**に改造します。

### 改造の内容

**ハードウェア**:
1. カラーインクをすべてグレー系に置き換え
2. CISS(Continuous Ink Supply System)導入
3. 外部タンクからの連続供給

**ソフトウェア**:
1. QuadToneRIP導入
2. モノクロ専用カーブ作成
3. Print-Tool運用(Mac)

### 改造の結果

**変更点**:
- カラープリント: 不可能になる
- モノクロ専用機になる

**得られるもの**:
- 高いUV濃度(Dmax 2.2以上)
- 滑らかな階調(特にハイライト)
- 低粒状性
- 安定した再現性

#### 階調重視型を選ぶ場合

#### 安定性重視型を選ぶ場合

- インク管理を簡素化したい
- Epson純正インクを使用
- 実用性を重視

**推奨**: まずは**階調重視型**で構築し、問題があれば安定性重視型に変更

## CISS構造の理解

### CISSの基本構造

```
外部インクタンク
    ↓ (ベント栓で大気開放)
    ↓
シリコンチューブ
    ↓
[オプション: ダンパー/フィルター]
    ↓
リフィルカートリッジ
    ↓
プリンター内部インク経路
    ↓
プリントヘッド
    ↓
ノズル
```

### 重要な物理原則

#### 負圧の重要性

**原理**:
```
ヘッド位置 > タンク位置
    ↓
負圧が発生
    ↓
インクが自然に落ちない
    ↓
吐出時のみインクが出る
```

**理想の高さ差**:
```
ヘッド - タンク = 5〜8cm
```

**高さが不適切な場合**:

タンクが高すぎる:
```
正圧
↓
インクが漏れる・滲む
```

タンクが低すぎる:
```
負圧が強すぎる
↓
インクが出ない・欠ける
```

#### ベント(大気開放)の役割

**ベント栓の機能**:
```
印刷中: 開ける
    ↓
外気導入
    ↓
タンク内圧が安定
    ↓
供給が安定

移動時: 閉める
    ↓
密閉
    ↓
漏れ防止
```

**ベントがないと**:
```
タンク内が真空に近づく
    ↓
インクが出なくなる
```

### 気泡管理

#### 気泡が問題になる理由

```
チューブ内に気泡
    ↓
気泡がヘッドに到達
    ↓
そのチャンネルが出ない(ノズル欠け)
```

#### 気泡の発生源

1. **インク充填時**
   - 初回セットアップ
   - インク交換時

2. **温度変化**
   - 室温上昇でインク内の空気が膨張
   - 気泡が発生

3. **インク溶存空気**
   - インクに元々含まれる空気
   - 時間と共に分離

4. **接続部の隙間**
   - チューブとカートリッジの接続
   - わずかな隙間から空気混入

#### 気泡対策

**設計段階**:
1. チューブの取り回しに気泡溜まりポイントを作らない
2. 下から上への配管を避ける
3. ダンパー/フィルターを使用

**運用段階**:
1. 定期的なチューブ確認
2. 気泡発見時はシリンジで除去
3. プライミング(インク充填)を適切に

### タンク配置の実務

#### 推奨配置

**プリンターとの位置関係**:
```
     [プリンター]
         ヘッド位置
           ↓
          5-8cm
           ↓
   [タンク9本を横一列]
```

**タンクの並び**:
```
[ULG][LG][MG][MG][DG][DG][VDG][PK][PK]
 1   2   3   4   5   6   7    8   9
```

**設置場所の条件**:
1. プリンターの横または後ろ
2. 平坦で安定した場所
3. 高さ調整可能な台
4. 温度変化が少ない場所

#### タンクホルダーの自作

**材料**:
- アクリル板またはMDF板
- タンク固定用クリップ
- 高さ調整用スペーサー

**設計**:
```
9本のタンクを等間隔で配置
各タンクを固定
高さを統一
```

## セットアップ手順

### 準備段階

#### 1. プリンターの準備

**純正インクの扱い**:
```
注意: 純正カートリッジは保管しておく
(改造を戻す場合に必要)
```

**手順**:
1. プリンター電源ON
2. 全ての純正インクカートリッジを取り外し
3. ヘッドクリーニング実行(1回)
4. 電源OFF

#### 2. リフィルカートリッジの準備

**ARCチップの確認**:
- チップが正しく装着されているか
- チップの接点が清潔か

**インク注入口の確認**:
- シリコン栓が正しく装着されているか
- 漏れがないか

#### 3. CISSタンクの組み立て

**タンクの準備**:
1. 9本のタンクを台に固定
2. 各タンクにベント栓を装着
3. チューブを各タンクに接続

**チューブの処理**:
1. 適切な長さにカット(60-100cm)
2. 両端をまっすぐカット
3. チューブ内を洗浄(純水で)
4. 完全に乾燥

### インク充填

#### 初回充填の手順

**重要**: 気泡を入れないことが最重要

**手順**:

1. **タンクへのインク注入**
   ```
   ベント栓を閉じた状態で
   各タンクにインクを注入
   容量: 50-100ml
   ```

2. **チューブへのインク充填**
   ```
   方法A: 重力で
   - タンクを高い位置に
   - チューブ下端からインクが出るまで待つ

   方法B: シリンジで
   - チューブ下端にシリンジを接続
   - ゆっくり吸引
   - インクがチューブ全体に満たされるまで
   ```

3. **カートリッジへの接続**
   ```
   - チューブ下端をカートリッジに接続
   - 接続部から漏れがないか確認
   - カートリッジ内にインクが入るのを確認
   ```

4. **プリンターへの装着**
   ```
   - 9本すべてのカートリッジを装着
   - プリンターが認識することを確認
   - エラーが出ないか確認
   ```

5. **ベント栓を開ける**
   ```
   - すべてのタンクのベント栓を開放
   - これで使用可能状態
   ```

### QTRのセットアップ

#### 1. QTRのインストール

**Mac環境**:
1. quadtonerip.comからダウンロード
2. インストーラー実行
3. Print-Toolインストール
4. プリンタードライバー設定

#### 2. プリンター設定

**システム環境設定**:
```
プリンタとスキャナ
↓
P800を追加
↓
ドライバー: QTR
```

#### 3. 初期カーブの設定

**とりあえずの設定**:
```
QTRフォルダ内の
Generic-linear.quad
を使用

(後で256ステップ測定後に自作カーブに置き換える)
```

### 初回テストプリント

#### ノズルチェック

**手順**:
1. Print-Toolを起動
2. Nozzle Check 選択
3. プリント

**確認項目**:
- 9チャンネルすべてが印刷されているか
- 欠けているラインはないか
- パターンが揃っているか

**問題がある場合**:
- ヘッドクリーニング実行
- 再度ノズルチェック
- 3回実行しても改善しない場合は気泡/詰まりを疑う

#### グラデーションテスト

**目的**: 各チャンネルが正常に機能しているか確認

**方法**:
1. 0-255のグラデーション画像を作成
2. Print-Toolで出力
3. 階調が出ているか確認

**確認項目**:
- バンディング(横縞)がないか
- 滑らかなグラデーションか
- ノイズが多くないか

## 安定運用のポイント

### 日常のメンテナンス

#### 毎日の確認事項

**プリント前**:
1. タンクのベント栓が開いているか
2. タンクのインク残量
3. チューブ内の気泡確認
4. ノズルチェック(週1回程度)

**プリント後**:
1. ヘッドクリーニング(必要な場合のみ)
2. タンクのベント栓(長期不使用時は閉じる)

#### 週次メンテナンス

1. **ノズルチェック**
   - 全チャンネル確認
   - 記録を取る(欠けの傾向を把握)

2. **気泡確認**
   - チューブ全体を目視
   - 気泡があれば除去

3. **タンク確認**
   - インク残量
   - 沈殿がないか(軽く振る)
   - ベント栓の詰まり確認

#### 月次メンテナンス

1. **詳細ノズルチェック**
   - パターン印刷
   - 粒状性確認

2. **256ステップテスト**
   - 濃度確認
   - カーブのドリフト確認

3. **接続部確認**
   - チューブとカートリッジ
   - 漏れがないか
   - 接続がゆるんでいないか

### 長期保管時の注意

#### 1週間以上使用しない場合

**手順**:
1. ヘッドクリーニング実行
2. ノズルチェック
3. タンクのベント栓を閉じる
4. プリンターの電源は入れたまま
   (自動メンテナンスが働く)

#### 1ヶ月以上使用しない場合

**推奨**: 純正カートリッジに戻す
- ヘッド保護のため
- 再開時にCISSを再セットアップ

### インク交換・補充

#### インクの補充

**タンクへの補充**:
1. ベント栓を閉じる
2. タンク上部から補充
3. ベント栓を開ける

**注意**:
- インクを混ぜない(Gray同士でも銘柄が違う場合)
- 補充後は軽く混ぜる(沈殿防止)

#### インクの完全交換

**理由**:
- 別のインクに変更
- 色が変わった
- 沈殿・劣化

**手順**:
1. 旧インクを完全に排出
2. チューブをクリーニング液で洗浄
3. 純水で洗浄
4. 完全乾燥
5. 新インクで初回充填手順を実施

## トラブルシューティング

### ノズル欠けの対処

#### 症状別の原因と対策

**毎回同じチャンネルが欠ける**

原因:
- そのチャンネルのヘッド詰まり
- そのチャンネルのカートリッジ問題

対策:
1. ヘッドクリーニング(3回まで)
2. カートリッジの取り外し・再装着
3. カートリッジ内のインク確認
4. チューブの確認

**ランダムに欠ける(日によって場所が変わる)**

原因:
- 気泡
- 負圧の変動
- ベントの問題

対策:
1. チューブ内の気泡を除去
2. タンクの高さを調整
3. ベント栓を清掃
4. 接続部を確認

**印刷開始直後は良いが、途中で欠ける**

原因:
- インク供給が追いつかない
- 負圧が強すぎる

対策:
1. タンク位置を5mm高くする
2. チューブの折れ・潰れを確認
3. インク粘度を確認(温度)

**しばらく置くと回復する**

原因:
- 乾燥
- キャッピング不良

対策:
1. 室内湿度を確認(40-60%推奨)
2. キャッピングステーションの清掃
3. 使用頻度を上げる(週1回は印刷)

### 濃度の問題

#### 濃度が日によって変わる

原因:
- 負圧の変動
- 温度変化
- インク沈殿

対策:
1. タンク位置を固定
2. 室温を安定させる(20-25℃)
3. 印刷前にタンクを軽く振る

#### 全体的に薄い

原因:
- インク吐出量不足
- QTRカーブの問題
- インクの劣化

対策:
1. ヘッドクリーニング
2. QTRカーブの見直し
3. インクの鮮度確認

#### ベタ面にムラ

原因:
- バンディング
- チャンネル間の不均一

対策:
1. ノズルチェック
2. ヘッドアライメント
3. 解像度設定の見直し

### 気泡の除去方法

#### チューブ内の気泡

**方法A: シリンジで吸引**
```
1. チューブをカートリッジから外す
2. シリンジをチューブ下端に接続
3. ゆっくり吸引
4. 気泡を含むインクを排出
5. 再接続
```

**方法B: 重力で**
```
1. タンクを高い位置に移動
2. チューブをまっすぐ垂らす
3. 気泡が上がるのを待つ(10-30分)
4. 元の位置に戻す
```

#### カートリッジ内の気泡

**方法**:
```
1. カートリッジを取り外す
2. 軽く振って気泡を集める
3. 上部のベントから気泡を逃がす
4. 再装着
```

### CISS固有の問題

#### インクが漏れる

原因:
- タンクが高すぎる
- 接続部のゆるみ
- カートリッジの破損

対策:
1. タンク位置を下げる
2. 接続部を締め直す
3. カートリッジを交換

#### インクが出ない

原因:
- タンクが低すぎる
- ベント栓が閉じている
- チューブの詰まり

対策:
1. タンク位置を上げる(ただし5-8cm範囲内)
2. ベント栓を開ける
3. チューブを確認・洗浄

### QTR関連の問題

#### Print-Toolで認識しない

対策:
1. QTRの再インストール
2. プリンタードライバーの再設定
3. Mac OS権限の確認

#### カーブが反映されない

対策:
1. カーブファイルの配置確認
2. Print-Toolでカーブを再選択
3. カラーマネジメントがOFFか確認

## まとめ

### 改造の要点

1. **ハードウェア**: CISS導入でインク連続供給
2. **インク**: Gray × 7, PK × 2 の構成
3. **ソフトウェア**: QTR + Print-Tool

### 成功のポイント

1. **慎重な初期セットアップ**
   - 気泡を入れない
   - 適切な高さ設定

2. **定期的なメンテナンス**
   - ノズルチェック
   - 気泡確認

3. **記録を取る**
   - 問題が起きた時の状況
   - メンテナンス履歴

4. **段階的な実験**
   - いきなり本番プリントしない
   - テストを重ねる

### この改造で得られるもの

- **デジタルフィルム**: 銀塩フィルムのような規格化された感材
- **高いDmax**: 2.2以上のUV濃度
- **滑らかな階調**: 特にハイライト域
- **安定性**: 作家の意思を保持できる再現性
