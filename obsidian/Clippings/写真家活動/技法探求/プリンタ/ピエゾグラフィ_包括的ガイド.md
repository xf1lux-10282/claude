## ピエゾグラフィとは

**ピエゾグラフィ（Piezography）**は、モノクロ写真を非常に高品質にプリントするために開発されたインクジェット印刷システムです。

### 開発者
- **Jon Cone** (Cone Editions Press)
- アメリカで開発
- ファインアート写真の世界で高く評価

### 商標について
**重要**: 「Piezography」はCone Editions Pressの登録商標です。同様の技術を使用する他のシステムも存在します。

### 基本コンセプト

**通常のインクジェット**
```
黒 + CMY（カラーインク）でグレーを作る
```

**ピエゾグラフィ**
```
6〜9種類のグレーインクだけで階調を作る
```

つまり、**複数のグレートーンインクのみを使用して写真を再現する方式**です。

## 通常のインクジェットとの違い

### 階調の作り方

#### 通常のモノクロインクジェット
```
グレー = 黒 + シアン + マゼンタ + イエロー
```

**問題点:**
- 微妙な色被り
- 階調の粗さ
- ハイライトが濁る
- 用紙による色転び

#### ピエゾグラフィ（例: 9インクシステム）
```
超薄グレー
  ↓
薄グレー
  ↓
ライトグレー
  ↓
ミドルグレー
  ↓
ダークグレー
  ↓
フォトブラック/マットブラック
```

**利点:**
- 非常に滑らかなトーン
- 色被りがほぼゼロ
- ハイライトが美しい
- 銀塩に近い階調表現

### システムのバリエーション

**K7 (7 shade system)**
- 7種類のグレーインク
- 標準システム

**Pro (9 shade system)**
- 9種類のグレーインク
- より滑らかな階調

**色調バリエーション**
- ニュートラル（純粋なグレー）
- ウォームトーン（温かみのある調子）
- クールトーン（冷たい調子）
- セピア調

## ピエゾプリントヘッドの技術

### ピエゾヘッドとは

**ピエゾプリントヘッド（Piezo Print Head）**は、**圧電素子（ピエゾ素子）**を使ってインクを押し出す仕組みのプリントヘッドです。

### 動作原理

```
① 電圧をかける
  ↓
② ピエゾ素子が変形する
  ↓
③ インク室の体積が変わる
  ↓
④ インクがノズルから押し出される
```

**特徴**: 機械的にインクを押し出している（熱を使わない）

### サーマルヘッド（Canon等）との違い

| 方式 | ピエゾ（Epson） | サーマル（Canon） |
|------|----------------|-------------------|
| 原理 | 圧電素子による機械的押出 | 熱でインクを沸騰させて噴射 |
| インク種類 | 顔料・特殊インク可 | 制限あり |
| ドット制御 | 非常に細かい | 比較的粗い |
| インク劣化 | なし（加熱しない） | あり（熱による劣化） |
| ファインアート | 適している | 限定的 |

### ファインアート印刷での優位性

#### 1. インクの自由度
ピエゾ方式では以下が使用可能:
- 顔料インク
- カーボンインク
- 特殊インク（UV遮光インクなど）

**これにより実現可能:**
- ピエゾグラフィ
- カーボンプリント用インク
- デジタルネガ用高濃度インク

#### 2. ドットサイズ制御の精密さ
- 非常に細かいインク量の制御が可能
- 結果として:
  - 滑らかなグラデーション
  - ハイライトの繊細さ
  - 微妙なトーン表現

#### 3. インクを加熱しない
- 顔料が劣化しない
- 粘度の高いインクも使用可能
- アーカイバル品質の維持

### 使用可能なプリンター

**Epson製プリンター（ピエゾヘッド搭載機）**
- P800
- P900
- P600
- P5000 / P7000
- SureColorシリーズ

**注意**: Canon製プリンターは使用不可（サーマルヘッドのため）

## プラチナプリントとの思想的共通点

### デジタル時代の「銀塩的モノクロ表現」

プラチナプリントを制作する作家にとって、ピエゾグラフィは**デジタル時代で最もプラチナに近い印刷方式**と言えます。

### 共通する思想

|要素|ピエゾグラフィ|プラチナプリント|
|---|------------|---------------|
|階調|非常に滑らか|非常に滑らか|
|物質性|紙に染み込む|紙に染み込む|
|色調|ニュートラル〜ウォーム|ニュートラル〜ウォーム|
|耐久性|高い（100年以上）|非常に高い（数百年）|
|表現|階調で写真を作る|階調で写真を作る|

### 違い

|要素|ピエゾグラフィ|プラチナプリント|
|---|------------|---------------|
|技法|インクジェット|化学写真|
|制作|デジタル|手作業|
|唯一性|複製可能|オリジナル|
|制作時間|短い|長い|

### 銀塩との比較

```
銀塩    → コントラストで表現
プラチナ → 階調で表現
ピエゾ   → 階調で表現
```

この思想の近さから、**海外のプラチナ作家の多くがピエゾグラフィを使用**しています。

---

## デジタルネガとの関係

### プラチナ作家がピエゾグラフィを使う最大の理由

実は、ピエゾグラフィの最も重要な用途は**展示プリント**ではなく、

**デジタルネガの作成**

です。

### なぜデジタルネガに最適なのか

#### 1. インク密度が高い
- より深い黒が得られる
- UV遮光性能が高い

#### 2. 階調が正確
- 複数のグレーインクによる精密な階調再現
- 補正カーブが安定

#### 3. UV遮光が安定
- カーボン顔料インクの特性
- プラチナプリントの露光に最適

### Piezography DN (Digital Negative) システム

**専用システム**:
- デジタルネガ作成に特化
- UV濃度最適化済み
- プラチナ/パラジウム/ヴァンダイク等に対応

**実際の使用**:
多くの海外プラチナ作家が以下を使用:
- Piezography DN system
- QuadTone RIP + カーボンインク
- カスタムインクカーブ

## UV濃度とDmaxの問題

### インクジェットの黒濃度の限界

写真の濃度は**Dmax（最大濃度）**で表します。

#### Dmax比較表

| 方式 | Dmax (可視光) | UV濃度 |
|------|--------------|--------|
| 銀塩プリント | 約2.1 | 高い |
| ピエゾグラフィ | 約2.0〜2.3 | 非常に高い |
| 通常インクジェット | 約1.6〜1.8 | 低い |
| プラチナプリント | 約1.4〜1.8 | - |

**補足**:
- ピエゾグラフィのDmax 2.3は理想値
- 実際は使用する紙により1.9〜2.3の範囲
- プラチナプリントのDmaxは技法・紙質で1.4〜1.8の幅がある

### なぜデジタルネガに高濃度が必要か

プラチナプリントは**UV露光**を使用します。

**問題**: 通常のインクジェットインクは可視光用に設計されているため、

```
可視光では黒く見える
  ↓
しかしUVは透過してしまう
```

**結果**:
- ハイライトが飛ぶ
- コントラスト不足
- 黒が締まらない

### UV濃度の実測比較

| インク種類 | UV濃度 |
|-----------|--------|
| Epson純正（フォトブラック） | 約1.6 |
| 染料インク | 約1.4 |
| カーボン顔料インク | 約2.0〜2.4 |

**この差は非常に大きい**です。

### カーボンインクの優位性

**カーボン顔料の特徴**:
- UV遮光が非常に強い
- 濃度が高い
- 色被りがない
- 耐久性が高い

**注意**: すべてのピエゾグラフィインクが純粋なカーボンとは限らず、一部は他の顔料も混合されています。

## システム構成

### ピエゾグラフィの3要素

完全なピエゾグラフィシステムは以下で構成されます:

```
① Epsonピエゾプリンター
  +
② 専用インクセット（6〜9色のグレーインク）
  +
③ RIPソフトウェア（QuadTone RIP等）
```

### QuadTone RIP (QTR) との関係

**重要な理解**:

- **QTR**: 無料のオープンソースRIPソフトウェア
- **Piezography**: QTR + 専用インク + カーブセットのパッケージ商品

つまり:
```
QTR = ソフトウェアのみ（無料）
Piezography = QTR + インク + カーブ（商品）
```

### RIPソフトウェアの役割

**RIP (Raster Image Processor)** とは:
- プリンタドライバを介さず、インクの吐出を直接制御
- カスタムインクカーブの適用
- 精密な階調コントロール

### 必要な投資

#### ハードウェア
- Epsonプリンター: すでに所有している場合は不要
- インクカートリッジ: 約3〜5万円（セット）

#### ソフトウェア
- QTR: 無料
- Piezographyシステム: インク + カーブで約5〜8万円

#### 学習コスト
- セットアップ: 中級〜上級
- キャリブレーション: 上級
- 日本語情報: 少ない

## 世界のプラチナ作家の実践

### 3つの主流アプローチ

#### ① Epson純正インク + 補正（最も多い）

**方法**:
- 通常のインクジェット
- LUT/グラデーションマップで補正
- デジタルネガ用に最適化

**利点**:
- 導入が容易
- カラープリントも可能
- メンテナンスが簡単

**限界**:
- UV濃度の物理的上限（約1.6〜1.8）
- 補正の複雑さ

**代表的研究者**:
- Dan Burkholder（グラデーションマップ法の開発）
- Mark Nelson（デジタルネガ研究）

#### ② Piezography DN（最も理想的）

**方法**:
- 専用インクシステム
- 事前最適化されたカーブ
- UV濃度最適化済み

**利点**:
- UV濃度が高い（2.0〜2.4）
- 階調が非常に安定
- プラチナに最適化

**制約**:
- プリンターがモノクロ専用になる
- インクコストが高い
- セットアップが複雑

**代表的システム**:
- Cone Editions Piezography DN
- Precision Digital Negatives (PDN) by Dana Sullivan

#### ③ カーボンインク自作（かなりマニア）

**方法**:
- QTR使用
- カスタムインク調合
- 独自カーブ作成

**利点**:
- 完全なカスタマイズ
- コスト削減の可能性

**制約**:
- 非常に高度な知識が必要
- 試行錯誤が膨大
- 再現性の確保が困難

### Dana Sullivanの研究

**Precision Digital Negatives (PDN)** システムの開発者

**研究の核心**:
- 単なるUV濃度最適化だけでなく
- **体系的なキャリブレーション手法**を確立
- ステップウェッジによる科学的測定
- 再現性の高いワークフロー

**主な貢献**:
- UV濃度測定の標準化
- キャリブレーションプロセスの体系化
- 各種オルタナティブプロセスへの対応

**参考**:
- ウェブサイト: precisiondigitalnegatives.com
- ワークショップ開催
- オンラインコース提供

## メリットとデメリット

### メリット

#### 1. プリント品質
- 銀塩に匹敵する階調の美しさ
- 色被りのないニュートラルトーン
- 深い黒と繊細なハイライト

#### 2. デジタルネガ品質
- 高いUV濃度（2.0〜2.4）
- 安定した階調再現
- プラチナプリントに最適

#### 3. アーカイバル品質
- 100年以上の耐久性
- 顔料インクの安定性
- 退色に強い

#### 4. 階調表現
- 9段階のグレーインク
- プラチナプリントと思想が近い
- 滑らかなトーン

### デメリット

#### 1. プリンター制約
- **完全にモノクロ専用機になる**
- カラープリントは不可
- インク交換の手間

#### 2. コスト
- 専用インクが高価
- カートリッジセット: 3〜5万円
- 定期的な交換が必要

#### 3. 技術的ハードル
- セットアップが複雑
- RIPソフトの理解が必要
- キャリブレーション技術

#### 4. 保証とサポート
- メーカー保証が失効
- 日本語サポートが限定的
- トラブル時の対処が困難

#### 5. リソース不足
- 日本語ドキュメントが少ない
- 国内コミュニティが小さい
- ワークショップが少ない

## 日本での状況

### 認知度

**現状**: 日本ではまだ少数派

**理由**:
1. セットアップの複雑さ
2. 専用インクの入手性
3. RIPソフトの学習コスト
4. **日本語ドキュメントの不足**（最大の障壁）
5. 国内コミュニティの小ささ

### 海外との比較

**海外（特に欧米）**:
- ファインアート写真家の定番
- ワークショップが豊富
- コミュニティが確立
- プリントラボでの標準技術

**日本**:
- 一部の先進的な作家のみ
- 情報が英語に限定
- 導入事例が少ない

### 世界の高品質モノクロ印刷の位置づけ

```
① 銀塩
② ピエゾグラフィ
③ プラチナプリント
```

この3つが「モノクロの最高品質」として認識されています。

特に:
- 版画的な深い黒
- 滑らかなハイライト
- 中間調の美しさ

はピエゾグラフィの特徴です。

### 現在の研究位置

木下さんの研究:

**これは実は**:

### 補正ポイント数の比較

| 作家/システム | 補正ポイント数 |
|-------------|--------------|
| 一般的な方法 | 9〜15ポイント |
| 木下大輔 | **32ポイント** |
| Piezography | システム組み込み |

**評価**: 32ポイント補正は世界的に見ても高度

### 現在のアプローチ

```
Epson純正インク
  +
32ポイントグラデーションマップ補正
  +
プラチナプリント
```

これは:
- 海外作家と同じ「通常インク + LUT補正」方式
- 非常に高度な補正レベル
- 実用的で柔軟性が高い

### 物理的限界

ただし存在する限界:
- **UV濃度のDmax上限（約1.6〜1.8）**
- これは補正では超えられない物理的制約

### ピエゾグラフィ移行を検討すべきタイミング

以下の場合に検討価値あり:

1. **ハイライトの飛びが改善できない場合**
   - 現在の補正で解決できない問題がある

2. **より大きな作品制作の場合**
   - 大判プリント（A2以上）
   - 階調の滑らかさが重要

3. **デジタルネガ専用機を確保できる場合**
   - カラープリントは別プリンターで対応可能

4. **より深い黒が必要な場合**
   - シャドウ部の表現を追求

### 現時点での評価

**率直に言うと**:

木下さんの現在の方法は
- かなり高度
- 実用的
- 柔軟性が高い

現在のプリント結果に**明確な不満がなければ**、急いでピエゾグラフィに移行する必要はないかもしれません。

ただし、将来的な選択肢として**研究する価値は十分にあります**。
