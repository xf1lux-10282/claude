## デジタルフィルムとは

### 定義

**デジタルフィルム**とは:
```
プリンター + インク + QTRカーブ + フィルム = 感材

銀塩フィルムのように特性が規格化され、
撮影者の意図を安定して記録する
デジタル時代の感光材料
```

### 従来のデジタルネガとの違い

#### デジタルネガ(従来)
```
アプローチ: 画像ごとの補正
画像 → カーブ調整 → プリント → ネガ

特徴:
・毎回トーンカーブやグラデーションマップで調整
・モニター環境に依存
・プリンター(刷る人)の感性に支配される
・再現性が不安定
```

#### デジタルフィルム(新しい概念)
```
アプローチ: 感材の規格化
画像 → 感材(固定特性) → ネガ

特徴:
・プリンター+インク+QTRカーブを「感材」として固定
・一度特性曲線を確立したら変更しない
・画像側は「入力規格」だけを守る
・銀塩フィルムのような再現性
```

### 思想の核心

**フィルム写真の思想**:
```
作家は「被写体の階調」を撮る
  ↓
感材(フィルム)の特性曲線が
その階調をどう写すかが決まっている
  ↓
追い込みは「現像(N-1, N, N+1)」と
「焼き込み・覆い焼き」で行う
  ↓
重要: 毎回特性曲線を作り直さない
```

**デジタルフィルムの思想**:
```
「プリンター+インク+QTRカーブ」を
一つの感材とみなす
  ↓
その感材の「特性曲線」を固定する
  ↓
作家の意図は、毎回の"好みのカーブ調整"ではなく
・撮影(露出・ライティング)
・現像(規則化された操作)
・定義されたゾーン運用
に乗せる
```

## 従来のデジタルネガの問題

### 不安定要素の正体

従来のデジタルネガ制作では、多くの不安定要素が混在しています:

```
撮影環境
  (光・露出・レンズ・センサー特性)
    ↓
現像
  (RAW現像のトーン、プロファイル、ガンマ、コントラスト)
    ↓
表示
  (モニター輝度・白色点・周囲光)
    ↓
補正
  (トーンカーブ・グラデーションマップ)
    ↓
出力
  (プリンター・インク・フィルム・RIP・解像・ディザ)
```

**問題**: これらが混ざったまま「最終トーンを画像で調整」すると:
```
→ 補正値は環境依存になる
→ モニターが変わると補正が変わる
→ 撮影環境で補正が変わる
→ 作家の意思が「工程の癖」に溶ける
```

### 責任の切り分けができていない

問題は単一の原因ではなく:
```
プリンター由来でも
モニター由来でも
単独の問題ではなく
「責任の切り分けがされていない」こと
```

### 作家の意思の喪失

最大の問題:
```
「プリンター(刷る人)の感性」に支配される

画像側で追い込む = "毎回、感性で合わせる"
  ↓
再現性は上がっても
作者の意思が「工程の癖」に溶ける危険
  ↓
特に複数人運用や将来の再制作でブレる
```

## 銀塩写真との対応関係

### 完全な構造的対応

| 要素 | 銀塩写真 | デジタルフィルム |
|------|---------|----------------|
| **感材** | フィルム乳剤 | プリンター+インク+QTRカーブ |
| **感度** | ISO (50, 100, 400...) | デジタルISO (露光時間) |
| **特性曲線** | H&D曲線 (トー/直線部/ショルダー) | QTRカーブ (逆Sカーブ) |
| **粒子** | 銀粒子 | ディザパターン |
| **粒状性** | 粒子サイズ | ディザ+チャンネル分散 |
| **コントラスト調整** | N現像 (N-1, N, N+1) | QTRカーブバリエーション |
| **ラチチュード** | フィルム特性 | カーブ設計 |
| **現像** | 化学処理 | (プラチナプリントが相当) |
| **焼き込み・覆い焼き** | 印画紙段階 | (プラチナプリント時に可能) |

### フィルムの多層構造との類似

#### 銀塩フィルムの乳剤層

```
表面: 小粒子層 (ハイライト担当)
  ↓
中間: 中粒子層 (中間調担当)
  ↓
深層: 大粒子層 (シャドウ担当)

粒子分布で階調を作る
```

#### デジタルフィルムのチャンネル構造

```
Ch1: ULG (Ultra Light Gray) - ハイライト専用
  ↓
Ch2: LG (Light Gray) - ハイライト〜中間調
  ↓
Ch3-4: MG (Mid Gray) × 2 - 中間調
  ↓
Ch5-6: DG (Dark Gray) × 2 - シャドウ手前
  ↓
Ch7: VDG (Very Dark Gray) - シャドウ
  ↓
Ch8-9: PK (Photo Black) × 2 - 最大濃度

ノズル分散で階調を作る
```

**原理的な類似**:
```
銀塩: 粒子の大きさと分布で階調
デジタル: インクのチャンネル分散で階調
```

### H&D曲線とQTRカーブ

#### H&D曲線 (Hurter-Driffield curve)

銀塩フィルムの特性曲線:
```
濃度
│        ┌─── ショルダー(飽和)
│       ／
│      /
│     /  直線部(ガンマ)
│    /
│   /
│  /
│ /
│/_____ トー(足)
└────────── log 露光量
```

**構造**:
- **トー(足)**: ハイライト域、緩やか
- **直線部**: 中間調、傾きがガンマ
- **ショルダー(肩)**: シャドウ域、飽和

#### QTRカーブ (プラチナプリント用)

デジタルフィルムの特性曲線:
```
ネガ濃度
│
│        ／
│      ／
│    ／   逆Sカーブ
│  ／
│／
└────────── 入力値(0-255)
```

**構造** (逆になる):
- **ハイライト**: 圧縮(0-20を小さな濃度差に)
- **中間調**: リニア(20-180)
- **シャドウ**: 軽い圧縮(180-255)

**なぜ逆Sか**:
```
ネガ濃度 ↑ → プラチナプリント濃度 ↓
(反転関係)

プラチナプリントでSカーブを得るため
ネガは逆Sカーブになる
```

## 感材としての規格化

### 規格化とは何か

**銀塩フィルムの規格化**:
```
例: Kodak Tri-X 400

・ISO感度: 400 (固定)
・特性曲線: 規定されている
・粒状性: 決まっている
・ラチチュード: 決まっている

作家は「この感材の特性」を前提に
撮影と現像を計画する
```

**デジタルフィルムの規格化**:
```
例: P800 + Gray×7 + PK×2 + QTRカーブN

・デジタルISO: 固定(露光時間で定義)
・特性曲線: QTRカーブ(固定)
・粒状性: ディザ+チャンネル構成で決定
・ラチチュード: カーブ設計で決定

作家は「この感材の特性」を前提に
撮影とネガ生成を計画する
```

### 固定すべき要素

#### ハードウェア層

1. **プリンター**
   - 機種: Epson SureColor P800
   - メンテナンス状態

2. **インク構成**
   - Gray × 7
   - PK × 2
   - 銘柄固定

3. **インク供給**
   - CISS設定
   - タンク高さ
   - 負圧管理

4. **ネガフィルム**
   - Pictorico OHP
   - 同一ロット(可能な限り)

#### ソフトウェア層

1. **QTRカーブ**
   - Nカーブ(標準)
   - 一度確立したら変更しない

2. **解像度**
   - 1440 × 1440 dpi (固定)

3. **ディザ**
   - Error Diffusion (固定)

4. **Print-Tool設定**
   - カラーマネジメントOFF
   - ガンマ補正OFF

#### プロセス層(プラチナプリント)

1. **UV露光**
   - 露光装置
   - UV光源
   - 距離

2. **感光液**
   - 調合比率
   - 塗布量
   - 乾燥条件

3. **現像**
   - 現像液
   - 温度
   - 時間

4. **紙**
   - 紙の種類
   - サイジング

### 入力規格

**画像側で守るべき規格**:

1. **ガンマ**
   - γ 1.0 (リニア)

2. **色空間**
   - グレースケール
   - 16bit TIFF

3. **プロファイル**
   - 埋め込みなし

4. **補正**
   - トーンカーブなし
   - グラデーションマップなし

**RAW現像の原則**:
```
Scene-referred (できるだけリニア)
↓
破綻しない範囲で
↓
ガンマ補正なし
```

**重要**: 見た目ではなく数値で管理

## デジタルISOの概念

### ISOとは何か(銀塩)

銀塩写真のISO:
```
定義: 一定の露光量に対して
      標準現像で得られる濃度

例:
・ISO 100: 低感度、高画質、微粒子
・ISO 400: 中感度、標準
・ISO 1600: 高感度、粗粒子
```

### デジタルISO (システム感度)

プラチナプリント用デジタルフィルムの場合:
```
デジタルISO =
「一定の露光条件(UV量・感光液・紙)に対して、
 標準露光で狙いのハイライト密度を得るために
 必要なネガ濃度スケール」

= システム感度 (System Speed)
```

### デジタルISOの決定方法

#### ゾーンシステムに基づく定義

**ゾーンの対応**:
```
Zone I  : やっと乗る濃度(シャドウの底)
Zone V  : 中間調
Zone VIII: ハイライトの基準(白に近いが質感が残る)
```

**標準露光の定義**:
```
1. ステップタブ(0-255)をQTRで出力
2. 同一条件でプラチナプリント
3. Zone VIII が「ちょうど良い」濃度になる露光量
   → これが標準露光
4. その標準露光で:
   ・Zone I が「やっと乗る」
   ・Zone VIII が「適正」
   になるようにQTRカーブを調整
```

**このとき**:
```
標準露光に対するネガ濃度スケール
= このデジタルフィルムのISO
```

### ISOの実務的な意味

#### 高ISO(速い)の意味

```
同じ露光時間でも「濃く写る」
↓
ネガ濃度が薄くても成立
↓
UV遮光が弱くても成立
```

**利点**:
- 露光時間が短い
- UV照射が少なくて済む

**欠点**:
- 粒状性が増す可能性

#### 低ISO(遅い)の意味

```
同じ露光時間でも「薄く写る」
↓
ネガ濃度を上げる必要がある
↓
露光時間を増やす必要がある
```

**利点**:
- ネガ濃度が高い(Dmax重視)
- 階調が豊か

**欠点**:
- 露光時間が長い

### N現像(コントラスト調整)

#### 銀塩のN現像

```
N現像  : 標準(基準)
N+現像 : 現像を延長 → コントラスト増
N-現像 : 現像を短縮 → コントラスト減
```

**使い分け**:
- ハイコントラストシーン → N-
- ローコントラストシーン → N+
- 標準シーン → N

#### デジタルフィルムのN運用

```
Nカーブ  : 標準(基準)
N+カーブ : シャドウ強調
N-カーブ : ハイライト保持
```

**運用原則**:
- 通常作品: **Nカーブ固定**
- 個別調整: 露光を微調整(±1/3〜2/3 stop相当)
- 特殊な場合のみ: N+/N- カーブ使用
- **2-3種類だけ用意**(これ以上増やすと感材の意味が失われる)

### ISO固定の重要性

**銀塩の運用**:
```
写真家は「Tri-X 400」を使うと決めたら
そのフィルムの特性に合わせて撮影する

特性曲線を毎回変えたりしない
```

**デジタルフィルムの運用**:
```
「P800 + Gray×7 + QTRカーブN」を
感材として固定する

QTRカーブを毎回変えたりしない

作家の意図は:
・撮影での露出決定
・N/N+/N-の選択(3種類のみ)
・プラチナプリント時の露光調整
に集約される
```

## 特性曲線の設計

### 256ステップ測定法

#### なぜγ1.0なのか

**補正なし・γ1.0で出力する理由**:
```
プリンター + フィルム + プラチナ紙
すべての特性をそのまま測定できる
```

**もし途中で補正すると**:
```
測定値が歪む
↓
正しい特性曲線が得られない
↓
逆カーブが作れない
```

#### 測定手順(詳細)

**1. γ1.0ネガの作成**
```
Photoshopまたは画像ソフト:
・0-255の256ステップ画像を作成
・各ステップ: 約5×5cm程度
・16bit グレースケール
・ガンマ: 1.0 (リニア)
・プロファイル埋め込みなし
```

**2. QTRで出力**
```
Print-Tool使用:
・カーブ: Linear (またはとりあえずのNカーブ)
・解像度: 1440 × 1440
・ディザ: Error Diffusion
・カラーマネジメント: OFF
・フィルム: Pictorico OHP
```

**3. プラチナプリント**
```
固定条件:
・UV露光時間: 固定(例: 5分)
・感光液: 調合を固定
・塗布量: 一定
・紙: 同一ロット
・現像: 時間・温度固定
・クリアリング: 固定
```

**4. 濃度測定**
```
方法A: スキャナー
・16bitグレースケールスキャン
・設定: すべての補正OFF
・各ステップのRGB値を取得
・平均値を計算

方法B: 濃度計
・各ステップを測定
・I濃度(ISO visual density)
```

**5. 濃度変換**
```
スキャナー値の場合:
Density = -log10(pixel_value / 255)

または:
Density = log10(255 / pixel_value)
```

**6. 特性曲線のプロット**
```
横軸: 入力値(0-255)
縦軸: プラチナプリント濃度

結果: S字カーブが得られる
```

**7. 逆カーブの生成**
```
理想: リニア(または弱いSカーブ)
現状: 測定された特性曲線

逆算:
理想濃度 → 必要なネガ入力値
```

**8. QTRカーブへの変換**
```
逆算で得た入力値を
各チャンネルの吐出率に変換

入力値 0-20   → ULG 主体
入力値 20-40  → LG 主体
入力値 40-90  → MG 主体
...
```

### Sカーブの必要性

#### なぜSカーブが自然に見えるか

**人間の視覚特性**:
```
Weber-Fechner の法則:
・暗部の変化に敏感
・明部では変化を感じにくい

対数的な知覚
```

**結果**:
```
リニア階調 → フラットに見える
Sカーブ   → 自然に見える
```

#### プラチナプリントの特性

**特徴**:
```
ハイライト: 非常に長い(繊細)
シャドウ: 比較的急(締まる)
```

**理想のプリントカーブ**:
```
プリント濃度
│    ┌────
│   /
│  /
│ /
│/
└──────── ネガ濃度

緩やかなSカーブ
```

**したがってネガカーブ**:
```
ネガ濃度
│
│      ／
│    ／
│  ／
│ ／
│／
└──────── 入力値

逆Sカーブ(ハイライト圧縮)
```

### カーブ設計の実例

#### ハイライトの処理(最重要)

**0-30の階調が命**

プラチナプリントでは:
```
ハイライトが非常に長い
↓
0-30の微小な差が
プリント階調に大きく影響
↓
ここが急だと「白飛び」に見える
```

**設計原則**:
```
入力 0-20: ULG のみ使用
・PKは一切使わない(粒状性)
・非常に緩やかなカーブ
・階調を圧縮

入力 20-40: LG 中心
・ULGと接続
・中間調へのスムーズな移行
```

#### 中間調の処理

**40-180: 直線的でOK**

```
MG × 2 + DG × 2
を使って滑らかに

基本的にはリニアに近い
```

#### シャドウの処理

**180-255: Dmax確保**

```
DG → VDG → PK × 2
の順に使用

目標: Dmax 2.0以上

ただし階調は保持
(完全に潰さない)
```

### 階調分担の設計表

```
入力値  | 担当インク | 吐出率  | 役割
--------|-----------|---------|------------------
0-10    | ULG       | 10-30%  | 最も明るい部分
10-20   | ULG       | 30-60%  | ハイライト
20-30   | LG        | 10-40%  | ハイライト〜中間調
30-40   | LG        | 40-70%  | 明るい中間調
40-70   | MG#1      | 20-60%  | 中間調前半
70-100  | MG#1,#2   | 40-80%  | 中間調中央
100-130 | MG#2, DG#1| 30-70%  | 中間調後半
130-160 | DG#1      | 50-90%  | 暗い中間調
160-190 | DG#1,#2   | 60-95%  | シャドウ手前
190-220 | DG#2, VDG | 70-100% | シャドウ
220-235 | VDG, PK#1 | 80-100% | 深いシャドウ
235-245 | PK#1      | 90-100% | 非常に深いシャドウ
245-255 | PK#1,#2   | 95-100% | 最大濃度(Dmax)
```

**ポイント**:
- チャンネル間で重複する領域がある(スムーズな移行)
- ハイライトはゆっくり立ち上がる
- シャドウは急だが階調は保持

## ゾーンシステムとの統合

### ゾーンシステムの基本

**Ansel Adamsのゾーンシステム**:
```
Zone 0  : 完全な黒
Zone I  : やっと質感がある黒
Zone II : 濃いシャドウ
Zone III: 平均的なシャドウ
Zone IV : 暗い中間調
Zone V  : 中間調(18%グレー)
Zone VI : 明るい中間調
Zone VII: 明るい部分
Zone VIII: ハイライト(質感あり)
Zone IX : ハイライト(ほぼ白)
Zone X  : 完全な白(紙の地)
```

### デジタルフィルムへのマッピング

#### 入力値とゾーンの対応

```
入力値   | ゾーン      | プラチナでの見え
---------|------------|------------------
0-25     | Zone IX-X  | ほぼ紙の地〜わずかな色
25-40    | Zone VIII  | ハイライト(質感あり)
40-70    | Zone VII   | 明るい部分
70-100   | Zone VI    | 明るい中間調
100-130  | Zone V     | 中間調
130-160  | Zone IV    | 暗い中間調
160-190  | Zone III   | 平均的なシャドウ
190-220  | Zone II    | 濃いシャドウ
220-245  | Zone I     | やっと質感がある黒
245-255  | Zone 0     | 完全な黒
```

### プレビジュアライゼーション

**銀塩の方法**:
```
1. シーンのゾーン配置を事前に想像
2. Zone VIII(ハイライト)を基準に露出決定
3. Zone I(シャドウ)が落ちすぎないか確認
4. 必要ならN+/-で調整
```

**デジタルフィルムの方法**:
```
1. 撮影時にヒストグラムでゾーン配置を確認
2. RAW現像でZone Vを中心に配置
3. Zone VIII, Zone I が範囲内に収まるか確認
4. 必要ならN+/-カーブで調整
```

「一番明るい、でも質感を残したい部分」

```

**デジタルの場合**:
```
その部分が
入力値 220-245

```

```

**ローコントラストシーン**:

## 実装の原則

### 作家の意思を保持する設計

#### 問題の構造

従来:
```
撮影 → RAW現像 → モニター確認 → カーブ調整 → 出力
            ↑           ↑           ↑
         感性依存    感性依存    感性依存

作家の意思が工程に溶ける
```

デジタルフィルム:
```
撮影 → RAW現像(規格化) → 感材(固定) → 出力
   ↑
作家の意思はここに集約

工程は機械的・再現的
```

#### 責任の分界

```
A: 感材(固定層)
・プリンター
・インク構成
・QTRカーブ
・解像度・ディザ
・フィルム

B: 入力(規格化層)
・γ1.0
・Scene-referred
・数値管理

C: 作家の意思(表現層)
・撮影での露出決定
・ライティング
・構図
・N/N+/N-の選択
・プラチナプリント時の露光微調整
```

**重要**: A層とB層は固定し、作家の意思はC層にのみ存在

### 測定で管理する原則

#### 感性ではなく数値で

**ダメな例**:
```
「モニターで見て暗いから明るくする」
「なんとなく良さそうだからこのカーブ」
```

**良い例**:
```
「256ステップ測定で Zone VIII が入力値35に対応」
「Dmaxは2.17を確保」
「ハイライトは入力0-30を濃度0.1-0.3に圧縮」
```

#### 定量的評価

**測定すべき項目**:
1. Dmax(最大濃度)
2. Dmin(最小濃度)
3. Zone VIII の濃度
4. Zone I の濃度
5. 各ステップの濃度差

**記録**:
- 日付
- 条件(インク、フィルム、露光時間等)
- 測定値
- 問題点

### 段階的な構築

#### フェーズ1: 基礎測定

```
1. P800 + CISS セットアップ
2. リニアカーブで256ステップ出力
3. プラチナプリント(条件固定)
4. 濃度測定
5. 特性曲線プロット
```

**目標**: システム全体の特性を理解する

#### フェーズ2: Nカーブ確立

```
1. 特性曲線から逆カーブ算出
2. QTRカーブ作成(初版)
3. テストプリント
4. 微調整(特にハイライト)
5. Nカーブ固定
```

**目標**: 標準カーブを確立する

#### フェーズ3: N+/N- カーブ

```
1. ハイコントラストシーン用: N-カーブ
2. ローコントラストシーン用: N+カーブ
3. 各カーブで256ステップ測定
4. 3本のカーブで固定
```

**目標**: 必要最小限のバリエーション

#### フェーズ4: 作品制作

```
1. Nカーブで通常作品制作
2. 必要に応じてN+/-選択
3. プラチナプリント時に露光微調整(±1/3 stop)
4. QTRカーブは変更しない
```

**目標**: 感材特性を前提とした創作

### 安定性のモニタリング

#### 定期チェック

**週次**:
- ノズルチェック
- グラデーションテスト

**月次**:
- 256ステップ出力
- 代表的な濃度ポイント測定
- カーブのドリフト確認

#### ドリフトの対処

**許容範囲内**:
```
±0.1濃度程度の変動
→ 許容(環境変動の範囲)
```

**要再調整**:
```
±0.2以上の変動
→ 原因調査
→ 必要なら再キャリブレーション
```

**再キャリブレーションのトリガー**:
- インク交換時
- フィルムロット変更時
- プリンターメンテナンス後
- 3ヶ月以上経過時

## 哲学的考察

### 技術と芸術の分離

#### 銀塩写真の知恵

```
技術: フィルムの特性、現像の化学
表現: 撮影での決断、ゾーンの配置

技術は規格化され、
表現に集中できる
```

#### デジタルの混乱

```
すべてが「調整可能」
↓
技術と表現の境界が曖昧
↓
無限の選択肢
↓
作家の意思が不明瞭に
```

#### デジタルフィルムによる再構築

```
技術: 感材(デジタルフィルム)として規格化
表現: 撮影とゾーン配置

技術を固定することで、
表現に集中できる
```

### 制約の創造性

**パラドックス**:
```
選択肢が無限 → 創造性が下がる
制約がある → 創造性が上がる
```

**理由**:
```
制約 → 決断を強制される
     → 意図が明確になる
     → 作品が強くなる
```

**デジタルフィルムの制約**:
```
・QTRカーブは3種類のみ(N, N+, N-)
・画像側では補正しない
・感材の特性に合わせて撮影する

この制約が創造性を生む
```

### 再現性と創造性

#### 再現性の重要性

**科学的側面**:
```
同じ条件 → 同じ結果
```

これは創造性を殺すのではなく、
**創造性を支える基盤**

**理由**:
```
再現性がある
↓
意図した結果が得られる
↓
試行錯誤が意味を持つ
↓
技術が積み重なる
```

#### 創造性の本質

**創造性は「変動」ではない**:
```
毎回結果が変わる ≠ 創造的
むしろ統制不能
```

**創造性は「意図的な選択」**:
```
確立された技術の上で
作家が意図的に選択する
= 真の創造性
```

### プロセスへの信頼

#### 銀塩写真家の態度

```
「Tri-X 400の特性を知り尽くしている」
↓
その特性を前提に撮影する
↓
現像は規則通り
↓
意図通りのネガが得られる

プロセスへの信頼
```

#### デジタルフィルムの目指すもの

```
「P800 + QTRカーブNの特性を知る」
↓
その特性を前提に撮影する
↓
ネガ生成は規則通り
↓
意図通りのネガが得られる

プロセスへの信頼の再構築
```

### 時間性と永続性

#### デジタルの脆弱性

```
ソフトウェアのバージョン変更
OSのアップデート
プリンタードライバーの変更
↓
再現性が失われる
```

#### デジタルフィルムの対抗

```
・QTRカーブをアーカイブ
・プリンター設定を文書化
・測定データを保存
・ワークフローを記録

将来の再現を可能にする
```

**意味**:
```
10年後、20年後に
同じネガを作れる

作品の永続性
```

### 作家性の本質

#### 問い

```
「毎回カーブを調整する」
vs
「固定されたカーブを使う」

どちらが作家的か？
```

#### 答え

```
毎回調整 = 工程に振り回されている
固定カーブ = 作家が工程を統制している

固定することで、
撮影時の判断に作家性が集約される
```

#### Ansel Adamsの例

```
彼は綿密な現像記録を残した
再現性を徹底的に追求した

それは作家性の放棄ではなく、
作家性の確立だった
```

工程は再現的・機械的に
