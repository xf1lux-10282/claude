# GT-X980 + Epson Scan でのQ-13キャリブレーション実践手順

## 環境
- **スキャナー**: EPSON GT-X980
- **ソフトウェア**: Epson Scan
- **ターゲット**: Kodak Q-13 Gray Scale
- **リファレンスデータ**: なし（標準値を使用）

## 重要な前提知識

### Epson Scan の制限
**Epson Scan には自動キャリブレーション機能がありません**

- IT8ターゲット読み込み機能なし
- プロファイル作成機能なし
- 手動でのトーン調整のみ可能

### 解決策
1. **VueScan を使用する（推奨）**
2. **手動キャリブレーション + プリント確認で調整**
3. **相対的な階調補正で対応**

---

## 方法1: VueScan を使用（最も推奨）

### なぜVueScanが必要か
```
Epson Scan → キャリブレーション機能なし
VueScan   → GT-X980対応 + IT8キャリブレーション機能あり
```

### VueScan の入手
- 公式サイト: https://www.hamrick.com/
- 価格: $99.95（プロフェッショナル版）
- 無料試用版でテスト可能（透かしが入る）

### VueScan での Q-13 キャリブレーション手順

#### 1. 初期設定
```
Input タブ:
- Source: Flatbed
- Mode: Grayscale
- Media: Reflective
- Resolution: 600 dpi
- Bits per pixel: 16 bit

Color タブ:
- Scanner: None (キャリブレーション前)
- Color balance: None
```

#### 2. Q-13 ターゲットの配置
```
1. GT-X980のガラス面を清掃
2. Q-13ターゲットを中央に配置
   - 印刷面を下向き（ガラスに密着）
   - 端から2-3cm内側に配置
3. 蓋を閉じる（外光を遮断）
```

#### 3. プレビュー
```
1. "Preview" ボタンをクリック
2. Q-13全体が表示されることを確認
3. 必要に応じてクロップ領域を調整
```

#### 4. キャリブレーション実行
```
1. Color タブ → "Scanner calibration" をクリック

2. キャリブレーションウィザードが開く:
   - "Gray scale target" を選択
   - ターゲットタイプ: "Custom"

3. 3点測定:
   ① ハイライト領域を選択（Step A または B）
      → Target L*: 95 を入力
      ※ VueScanのQ-13用デフォルト値

   ② 中間調領域を選択（Step J または K）
      → Target L*: 50 を入力

   ③ シャドウ領域を選択（Step S または T）
      → Target L*: 10 を入力

4. "Create profile" をクリック

5. プロファイル名を入力:
   例: "GT-X980_Q13_2026-04-13"

6. 保存
```

#### 5. 検証
```
1. Color タブ → Scanner: 作成したプロファイルを選択
2. Q-13を再度スキャン
3. Photoshop等で開く
4. スポイトツールで確認:
   - Step A: L*=96 付近
   - Step J: L*=50 付近
   - Step S: L*=10 付近
```

## 方法2: Epson Scan での手動調整（VueScan がない場合）

### 警告
この方法は**科学的に正確なキャリブレーションではありません**が、実用的な階調再現は得られます。

### 手順

#### 1. Epson Scan の設定
```
モード: プロフェッショナルモード

基本設定タブ:
- 原稿種: 反射原稿
- イメージタイプ: グレースケール
- 解像度: 600 dpi
- 出力設定: 16bit グレースケール

調整タブ:
- 色補正: なし
- アンシャープマスク: オフ
- 退色復元: オフ
- 逆光補正: オフ
```

#### 2. Q-13 基準スキャン
```
1. Q-13をスキャナーに配置
2. プレビュー
3. 調整なしでスキャン実行
4. 保存: "Q13_baseline.tif"
```

#### 3. Photoshop での確認と目標値設定
```
1. Q13_baseline.tif を Photoshop で開く
2. 情報パネルを表示（F8）
3. スポイトツールで各ステップを測定:

   Step    実測RGB    目標RGB    差分
   ----    -------    -------    ----
   A       ???        245        ?
   J       ???        128        ?
   S       ???        25         ?

※ L*=96 ≒ RGB=245
※ L*=50 ≒ RGB=128
※ L*=10 ≒ RGB=25
```

#### 4. Epson Scan のトーンカーブ調整
```
1. Epson Scan → 調整タブ → トーンカーブ

2. カーブ調整:
   - ハイライト側が暗い場合 → 右上を持ち上げる
   - シャドウ側が明るい場合 → 左下を下げる
   - 全体のガンマを調整

3. 調整の目安:

   実測が暗すぎる場合:
   カーブを上に持ち上げ（明るく）

   実測が明るすぎる場合:
   カーブを下に下げ（暗く）

4. 調整後、再度Q-13をスキャン
5. Photoshopで測定して確認
6. 繰り返し調整
```

#### 5. 設定の保存
```
Epson Scan:
1. すべての設定が完了したら
2. 「設定の保存」→「デジネガ用_Q13_calibrated」
3. 今後のスキャンでこの設定を使用
```

## 方法3: 相対的キャリブレーション（最もシンプル）

### 考え方
絶対的なL*値ではなく、**相対的な階調バランス**を重視

### 手順

#### 1. Q-13 をスキャン
```
Epson Scan の基本設定（調整なし）でスキャン
```

#### 2. Photoshop でレベル補正
```
1. イメージ → 色調補正 → レベル補正
2. スポイトツールで設定:

   白色点: Step A または B をクリック
   グレー点: Step J をクリック
   黒色点: Step S をクリック

3. この補正をアクションとして記録
4. 今後の全スキャン画像に適用
```

#### 3. LUT 作成（応用）
```
1. 補正後の画像を保存
2. Photoshop の 3D LUT として書き出し
3. DaVinci Resolve 等で LUT として適用
```

```
4. 目標値に近づくまで繰り返し
```

```
4. プリント結果を見て微調整

※ 絶対値より「再現性」が重要
```

## GT-X980 特有の注意点

### 1. ウォームアップ
```
電源投入後30分待ってからキャリブレーション
（光源が安定する）
```

### 2. ガラス面の清掃
```
無水エタノール + マイクロファイバークロス
指紋や埃を完全に除去
```

### 3. スキャン位置
```
GT-X980は中央が最も平面性が高い
Q-13は中央に配置
```

### 4. 環境光の遮断
```
蓋をしっかり閉じる
外光が入るとキャリブレーションが狂う
```

### 5. フィルムホルダーは外す
```
フラットベッドスキャン時は
フィルムホルダーを完全に取り外す
```

## トラブルシューティング

### Q: Epson Scan でトーンカーブが出てこない
A:
- プロフェッショナルモードに切り替え
- 「調整」タブ → 「トーンカーブ」をクリック
- 表示されない場合はドライバー再インストール

### Q: 何度調整してもRGB値が合わない
A:
- ±5の範囲内なら実用上問題なし
- 完全一致は困難（Epson Scanの制限）
- VueScan の使用を検討

### Q: キャリブレーション後もプリントが暗い/明るい
A:
- スキャナーキャリブレーションだけでは不十分
- デジネガのカーブ調整も必要
- QTRカーブのリニアライゼーションも実施

### Q: スキャンのたびに色が変わる
A:
- スキャナーのウォームアップ不足
- 室温の変化
- 毎回同じ時間帯にキャリブレーション実施
