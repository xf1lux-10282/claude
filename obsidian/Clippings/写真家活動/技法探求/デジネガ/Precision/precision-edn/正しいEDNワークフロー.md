# Precision EDN - 正しいワークフロー完全ガイド

## ⚠️ 重要な理解

**EDNは、デジネガでプリントした結果（プラチナ/銀塩プリント）をスキャンして補正します**

テストチャート画像そのものをスキャンするのではありません！

---

## 📋 完全なEDNワークフロー

### 【Phase 1: 1次補正（Absolute Beginner）】

```
ステップ1: EDNテストチャートのデジネガを出力
  - EDN 256 または EDN 101 のグレースケールチャート
  - プリンター: PX-1V
  - フィルム: TPS-100
  - 設定: カラーマネジメント「Photoshop」、プロファイル「一般RGB」

ステップ2: デジネガでプラチナプリント/銀塩プリント
  - UV露光機でプリント
  - 現像・定着
  - 乾燥

ステップ3: プリント結果をスキャン
  - 300dpi以上でスキャン
  - JPG/PNG で保存
  - **3回スキャン推奨**（自動平均化）

ステップ4: Precision EDNで処理
  - モード選択: 「1次補正（Absolute Beginner）」
  - スキャン画像をアップロード
  - 1次補正LUTをダウンロード
```

---

### 【Phase 2: 2次補正（Fine Tuning）】

```
ステップ5: 1次補正LUTを適用したデジネガを出力
  - Photoshopで1次補正LUTを適用
  - EDN 3D LUT（Color Blocker）も適用
  - デジネガをプリント（TPS-100）

ステップ6: そのデジネガでプラチナプリント/銀塩プリント
  - UV露光機でプリント
  - 現像・定着
  - 乾燥

ステップ7: プリント結果をスキャン
  - 300dpi以上でスキャン
  - JPG/PNG で保存
  - **3回スキャン推奨**

ステップ8: Precision EDNで処理
  - モード選択: 「2次補正（Fine Tuning）」
  - スキャン画像をアップロード
  - 2次補正LUTをダウンロード
```

---

### 【Phase 3: 最終LUT生成】

```
ステップ9: 1次+2次補正LUTを結合
  - モード選択: 「LUT結合（1次+2次）」
  - 1次補正LUT (.cube) をアップロード
  - 2次補正LUT (.cube) をアップロード
  - 結合LUTをダウンロード

最終的な成果物:
  - 結合LUT（1次+2次の統合版）
  - これをPhotoshopで適用すれば完璧なデジネガが出力される
```

---

## 🎯 各モードの使い分け

### 1次補正（Absolute Beginner）
**目的**: 初回の補正
**入力**: 無補正デジネガのプリント結果のスキャン
**出力**: 1次補正LUT

### 2次補正（Fine Tuning）
**目的**: 微調整
**入力**: 1次補正適用後デジネガのプリント結果のスキャン
**出力**: 2次補正LUT

### LUT結合
**目的**: 最終LUT作成
**入力**: 1次補正LUT + 2次補正LUT
**出力**: 統合LUT（これを使えば1回の適用で完璧な補正）

## 📸 具体例（PX-1V + TPS-100 + プラチナプリント）

### 1次補正の流れ

```
1. EDN_256.tif を開く（~/Downloads/EDN2_2/TestCharts/）

2. Photoshopでデジネガ出力
   - 16bit/チャンネル、AdobeRGB
   - EDN 3D LUT（Color Blocker）適用 ← あなたは既にこれを持っている
   - プリント設定:
     • カラー処理: Photoshopによるカラー管理
     • プロファイル: 一般 RGB プロファイル
   - PX-1V設定:
     • 1440×1440 dpi
     • 色補正なし、Black Enhance OFF

3. TPS-100にデジネガをプリント

4. そのデジネガをUV露光機にセット

5. プラチナ印画紙に露光
   - 現像・定着
   - 乾燥

6. プリント結果をスキャン
   - 300dpi、3回スキャン

7. Precision EDN
   - モード: 「1次補正」
   - 3枚のスキャン画像をアップロード
   - 1次補正LUTをダウンロード
```

### 2次補正の流れ

```
8. 実際の写真で作業
   - RAW現像 → 16bit TIFF
   - Photoshopで編集
   - EDN 3D LUT 適用
   - 1次補正LUT 適用 ← ★ここで使う
   - デジネガ出力（同じプリント設定）

9. そのデジネガでプラチナプリント
   - 実際の作品としてプリント

10. プリント結果をスキャン
   - EDNテストチャートではなく、実際の作品をスキャン
   - 300dpi、3回スキャン

11. Precision EDN
   - モード: 「2次補正」
   - スキャン画像をアップロード
   - 2次補正LUTをダウンロード

12. LUT結合
   - モード: 「LUT結合」
   - 1次補正LUT + 2次補正LUT
   - 統合LUTをダウンロード

13. 以後、統合LUTを使用
   - EDN 3D LUT → 統合LUT の順で適用
   - 完璧なデジネガが出力される
```

## ❓ よくある誤解

### ❌ 間違い
「EDNテストチャートの画像をスキャンする」

### ✅ 正しい
「EDNテストチャートのデジネガでプリントした**プラチナ/銀塩プリント**をスキャンする」

---

## 🔧 なぜこの流れが必要なのか

### 問題
プリンター（PX-1V）の出力特性と、最終プリント（プラチナ/銀塩）の特性は異なる

### デジネガ → プラチナプリントの連鎖
```
デジタルデータ
  ↓ (PX-1Vの非線形特性)
デジネガ（TPS-100）
  ↓ (UV露光の非線形特性)
プラチナ印画紙
  ↓ (化学反応の非線形特性)
最終プリント
```

### EDNの役割
この**3段階の非線形性**を測定して補正する

1. **1次補正**: デジネガ → プラチナプリント の大まかな補正
2. **2次補正**: 1次補正後の残存誤差を微調整
3. **LUT結合**: 2段階を統合して1回の適用で完璧に

---

## 💡 Tips

### 測定精度を上げるために
1. **同じ条件で測定**: 露光時間、現像時間を統一
2. **3回スキャン**: 自動平均化でノイズ除去
3. **品質スコア90点以上**: これを目指す

### 反復補正も可能
```
1次補正 → 2次補正 → 3次補正 → ...
```
品質スコアが90点以上になるまで繰り返せます
