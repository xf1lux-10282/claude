# Precision EDN - クイックスタートガイド

## 🚀 すぐに始める（3ステップ）

### ステップ1: ブラウザでツールを開く
```
precision-edn/index.html をダブルクリック
```
→ ブラウザで自動的に開きます

---

### ステップ2: プリント結果のスキャン画像をアップロード
1. **EDNテストチャートのデジネガでプリントした最終プリント**（プラチナ/銀塩）をスキャン
2. スキャン画像をドラッグ&ドロップ
3. 複数枚アップロード可能（自動で平均化されます）

**対応画像:**
- **EDN 256 (1507×1507px)** ← **推奨！110mm×110mm @ 300DPI、高精度補正**
- EDN 101 (1003×1087px) も対応可能（精度は劣る）

**重要**: デジネガ自体ではなく、デジネガで**プリントした結果**をスキャンします

---

### ステップ3: ダウンロード
1. 品質スコアを確認（90点以上が理想）
2. 「📦 全ファイル一括ダウンロード (ZIP)」をクリック
3. ZIPを解凍 → `.cube` ファイルをPhotoshopで使用

## 🎯 推奨設定

### 測定精度を最大化するために
1. **テストチャート: EDN 256**
   → 256階調測定、シャドウ部も高精度

2. **プリントを3枚制作**（プラチナ/銀塩の場合）
   → 感光液の手塗布によるばらつきを平均化
   → 各プリントを1回ずつスキャンして、全3枚をアップロード
   → 自動で中央値フィルタが適用されます

3. **サンプリングポイント: 1024**
   → 10bit相当の精度（処理速度と精度のバランス）

4. **補間方法: PCHIP**
   → トーンジャンプを完全に防ぐ

## ❓ よくある質問

### Q: 画像をアップロードしたが何も起きない
A: 以下を確認してください:
- 画像サイズが正しいか（**EDN 256推奨**: 1507×1507px）
- プリント結果をスキャンしているか（テストチャート画像そのものではない）
- ブラウザのコンソールにエラーが表示されていないか
- JPG/PNG形式か

### Q: 品質スコアが低い
A: 以下を試してください:
1. プリント設定を見直す（特にカラーマネジメント）
2. スキャン品質を確認（焦点、照明）
3. **複数枚プリントして平均化**（特にプラチナ/銀塩の手作業プリント）

### Q: ダークモードにしたい
A: 右上の「🌓 ダークモード」ボタンをクリック
