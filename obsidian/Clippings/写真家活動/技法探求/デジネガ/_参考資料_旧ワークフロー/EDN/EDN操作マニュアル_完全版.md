## EDNとは

### Easy Digital Negatives (EDN) の概要

EDNは、デジタルネガティブ作成のためのオンラインプログラムです。ブラウザベースで動作し、macOS、Windows、Linux、モバイルデバイスなど、あらゆる環境で使用できます。

**公式サイト**: http://www.easydigitalnegatives.com/

### EDNの特徴

- **無料**: すべてのツールがブラウザで無料使用可能
- **クロスプラットフォーム**: JavaScript で動作、OS非依存
- **プリンター非依存**: あらゆるインクジェットプリンターに対応
- **科学的アプローチ**: UV波長特性測定に基づく最適化

### EDNワークフロー全体図

```
【ステップ1】Color Blocker でUV最適化インク配合を測定
  ↓ (.cube LUT生成)
【ステップ2】Absolute Beginner で階調補正カーブを測定
  ↓ (.cube LUT生成)
【ステップ3】本番画像に両LUTを適用してプリント
  ↓
【ステップ4】UV露光・銀塩現像
  ↓
【ステップ5】Fine Tuning で微調整（必要に応じて）
```

## 必要な機材

### ハードウェア

- **コンピュータ**: デスクトップ、ノートPC、タブレット等（JavaScript対応ブラウザが動けばOK）
- **Webブラウザ**: Chrome、Firefox、Safari、Edge等（JavaScript有効）
- **インクジェットプリンター**: 任意のメーカー・機種（EPSON、Canon、HP等）
- **OHPフィルム**: Pictorico TPS-100 推奨（その他のOHPフィルムも可）
- **スキャナー**: フラットベッドスキャナー（またはデジタルカメラでの撮影でも可）
- **UV露光機**: 銀塩印画紙用のUV露光装置
- **暗室機材**: 銀塩印画紙、現像液、停止液、定着液、バット、タイマー等

### ソフトウェア

- **Adobe Photoshop**: LUT適用と画像編集用（推奨: 16bit対応版）
- **代替**: GIMP（無料、GIMPフォーマットのLUTも生成可能）

## EDN Absolute Beginner 操作手順

### Absolute Beginnerとは

**目的**: プリンターとUV露光機の組み合わせにおける階調特性を測定し、補正LUTを生成する

**仕組み**:
```
理論的な階調（0-100%）
  ↓
実際のプリント
  ↓
UV露光・現像
  ↓
スキャンで実測
  ↓
理論値と実測値の差を補正するLUT生成
```

---

### 全体ワークフロー概要

```
【ステップ1】テストチャート準備・プリント
  ↓
【ステップ2】UV露光・現像
  ↓
【ステップ3】スキャン測定
  ↓
【ステップ4】補正LUT作成
  ↓
【ステップ5】本番画像に適用
```

### ステップ1: テストチャート（Step Wedge）の準備とプリント

#### 1-1. Step Wedgeファイルのダウンロード

1. **EDN公式サイトにアクセス**
   - http://www.easydigitalnegatives.com/absolute-beginner/

2. **Step Wedgeファイルをダウンロード**
   - 21段階のグレースケールステップウェッジ
   - TIFF形式（16bit推奨）

#### 1-2. Photoshopでの準備

1. **ファイルを開く**
   - ダウンロードしたStep WedgeをPhotoshopで開く

2. **左右反転（ミラー）**
   - イメージ > 画像の回転 > カンバスを左右に反転
   - （理由: ネガと印画紙を密着させる際の向き）

3. **階調反転（ネガティブ化）**
   - イメージ > 色調補正 > 階調の反転
   - （白黒が逆になる）

#### 1-3. 正しい設定でプリント ★最重要

```
【Photoshop プリント設定】
□ カラー処理: "Photoshopによるカラー管理"
             （プリンターによる管理ではない！）
□ プロファイル: "一般 RGB プロファイル"
             （またはsRGB、AdobeRGB等の汎用プロファイル）

【プリンタードライバー設定】
□ 用紙種類: OHPシート（またはフィルム、透明フィルム等）
□ 印刷品質: 1440×1440 dpi（推奨）
□ カラーマネジメント: OFF（またはColorSync等、色補正なし）
□ 双方向印刷: OFF
□ 自動補正系: すべてOFF（Black Enhance、スムージング等）
```

**注意**: この設定を間違えると、測定が無意味になります。

4. **OHPフィルムにプリント**

---

### ステップ2: UV露光と銀塩現像

#### 2-1. UV露光

1. **銀塩印画紙を準備**
   - 暗室またはセーフライト下で作業

2. **密着露光**
   - プリントしたStep Wedgeフィルムを印画紙に密着
   - UV露光機で露光（露光時間は機材による）

3. **現像**
   - 通常の銀塩プリント手順で現像・停止・定着・水洗

4. **乾燥**
   - 完全に乾燥させる

#### 2-2. ポジ画像の作成

露光・現像した印画紙は、元のStep Wedgeの「ポジ画像」になります。

```
【元のデジタル画像】
白 → グレー → 黒（21段階）
  ↓ 反転してネガプリント
【OHPフィルム】
黒（濃い）→ グレー → 透明（21段階）
  ↓ UV露光
【銀塩印画紙】
白 → グレー → 黒（21段階）= ポジ画像
```

### ステップ3: スキャン測定

#### 3-1. 印画紙のスキャン

1. **スキャナー設定**
   - カラーモード: **RGB** （重要）
   - ビット深度: 8bit
   - 解像度: 300-600 dpi
   - カラーマネジメント: OFF（または埋め込みプロファイルなし）
   - **自動補正: すべてOFF**
     - 自動露出: OFF
     - 色補正: OFF
     - シャープネス: OFF

2. **スキャン実行**
   - 乾燥した印画紙をスキャン
   - TIFF形式で保存

#### 3-2. スキャンデータの確認

- 21段階のパッチが明確に分離しているか確認
- 極端に暗すぎる/明るすぎる場合は露光時間を調整して再試行

---

### ステップ4: EDN Absolute Beginnerで補正LUT作成

#### 4-1. プログラムへアクセス

1. **ブラウザでEDN Absolute Beginnerを開く**
   - http://www.easydigitalnegatives.com/absolute-beginner/
   - （またはローカルにダウンロードしたHTML版）

#### 4-2. スキャン画像のアップロード

1. **「ファイルを選択」ボタンをクリック**
2. **スキャンしたTIFF画像を選択**
3. プログラムが自動的に21パッチを検出・測定

#### 4-3. 補正LUT生成

1. **測定完了後、自動的に計算**
   - プログラムが理論値と実測値の差を計算
   - 補正カーブを生成

2. **ダウンロード**
   ```
   【推奨】
   - Maximum LUT 1D (.cube) ← これをクリック
     （21ポイント、52ポイント、または256ポイント）

   【その他のオプション】
   - Optimum LUT 1D (.cube)
   - Adobe Gradient Map (.grd)
   - GIMP Gradient Map (.ggr)
   ```

3. **ファイル名を確認**
   - ダウンロードされたファイル名（例: `EDN_correction.cube`）
   - わかりやすい名前にリネーム推奨
   - 例: `AbsBeginner_TPS100_PX1V_1440dpi_20260307.cube`

#### 4-4. Maximum版のポイント数確認

1. **.cubeファイルをテキストエディタで開く**
2. `LUT_1D_SIZE` の値を確認
   - 21ポイント = 標準
   - 52ポイント = 高精度
   - 256ポイント = 最高精度

---

### ステップ5: 本番画像への適用

#### 5-1. Photoshopでの画像準備

1. **RAW現像**
   - Adobe Camera Raw / Lightroomで基本補正
   - 16bit TIFFで出力

2. **Photoshopで開く**
   - イメージ > モード > RGBカラー（AdobeRGB推奨）
   - 16bit/チャンネル を確認

#### 5-2. Color Blocker適用（UV最適化）

1. **レイヤー > 新規調整レイヤー > カラールックアップ**
2. **「3D LUTファイルを読み込み」をクリック**
3. **Color Blocker LUTを選択** （例: `ColorBlocker_TPS100_PX1V.cube`）

#### 5-3. Absolute Beginner補正適用

1. **レイヤー > 新規調整レイヤー > カラールックアップ**
2. **「3D LUTファイルを読み込み」をクリック**
3. **Absolute Beginner LUTを選択** （例: `AbsBeginner_TPS100_PX1V.cube`）

**重要**: 適用順序
```
元画像（16bit RGB）
  ↓
1. Color Blocker LUT（UV最適化インク配合）
  ↓
2. Absolute Beginner LUT（階調補正）
```

#### 5-4. 最終調整

1. **左右反転（ミラー）**
   - イメージ > 画像の回転 > カンバスを左右に反転

2. **階調反転（ネガティブ化）**
   - イメージ > 色調補正 > 階調の反転

3. **必要に応じて**
   - アンシャープマスク（シャープネス調整）
   - サイズ調整

#### 5-5. プリント

```
【Photoshop プリント設定】★最重要
□ カラー処理: "Photoshopによるカラー管理"
□ プロファイル: "一般 RGB プロファイル"

【プリンタードライバー設定】
□ 用紙種類: OHPシート
□ 印刷品質: 1440×1440 dpi（測定時と同じ）
□ カラーマネジメント: OFF（色補正なし）
□ 双方向印刷: OFF
□ 自動補正: すべてOFF
```

**重要**: 測定時とまったく同じプリント設定を使用すること

## EDN Color Blocker 操作手順（詳細版）

### Color Blockerとは

Color Blockerは、**UV波長特性に基づいて最適なインク配合を見つける**ためのプログラムです。

**目的**:
- UV露光によって銀塩印画紙を感光させる際、各インクのUV透過率が異なる
- CMYKインクを最適なバランスで配合することで、ニュートラルなグレーを実現
- 色かぶりのないデジタルネガティブを作成

**技術的背景**:
```
【問題】
通常のグレースケール画像をCMYKインクで印刷すると...
  ↓
各インクのUV吸収特性が異なる
  ↓
UV露光時に特定の色が強く遮断される
  ↓
銀塩プリントに色かぶり発生

【解決】
Color Blockerで最適なインク配合（RGB値）を見つける
  ↓
各階調レベルでUV透過率が均等になる組み合わせを計算
  ↓
3ポイントの1D LUTとして保存
  ↓
ニュートラルなグレーネガティブを実現
```

6. **ダウンロード**
   - 対応する "Download" ボタンをクリック
   - ファイルを保存

### Color Blockerの技術的仕組み

#### UV波長特性とインク配合

```
【UV露光機の波長】
UVA: 315-400 nm（主に使用）
UVB: 280-315 nm

【各インクのUV吸収特性】
Cyan (C):    UVを強く吸収
Magenta (M): UVを中程度吸収
Yellow (Y):  UVをほとんど通す
Black (K):   UVを完全に遮断

【問題】
通常のグレー印刷: C=M=Y=均等配合
  ↓
UV露光時: Cyanが強すぎてUV遮断過剰
  ↓
結果: 青みがかった銀塩プリント

【Color Blockerの解決】
最適インク配合を計算:
  R=60%, G=0%, B=10%（中間調の例）
  ↓
実際のインク: C少なめ、M中程度、Y多め
  ↓
UV透過率が均等になる
  ↓
ニュートラルなグレー銀塩プリント
```

#### 3ポイント1D LUTの意味

```
【3ポイント】
ポイント1: 黒（RGB 0,0,0）    → インク配合1
ポイント2: 中間調（RGB値可変） → インク配合2
ポイント3: 白（RGB 1,1,1）    → インク配合3

【なぜ3ポイントで十分か】
- UV最適化は主に「インク配合比率」の問題
- 階調の細かい補正はAbsolute Beginnerが担当
- Color Blockerは「色かぶり防止」に特化
```

### よくある質問

**Q1: Optimum と Maximum の違いは？**

A: 両方とも3ポイント1D LUTですが、RGB値の計算アルゴリズムが異なります。
- Optimum: バランス重視
- Maximum: より積極的な補正

まずOptimumを試し、色かぶりが残る場合Maximumを試してください。

**Q2: Color Blockerのポイント数は増やせますか？**

A: いいえ。Color Blockerは設計上3ポイントです。これはUV最適化インク配合の「基準点」を定義するためで、細かい階調補正はAbsolute Beginnerが行います。

**Q3: Color BlockerとAbsolute Beginnerの適用順序は？**

A:
1. Color Blocker（UV最適化インク配合）
2. Absolute Beginner（階調補正）

この順序が正しいワークフローです。

**Q4: 再測定が必要なタイミングは？**

A:
- フィルムのロット変更
- プリンターのメンテナンス後
- UV露光機のランプ交換後
- 明らかな色かぶりが発生した場合

## EDN Fine Tuning（微調整）操作手順

### Fine Tuningとは

**目的**: Absolute BeginnerとColor Blockerで基本的な補正を行った後、さらに細かい調整を行う

**使用タイミング**:
- Absolute Beginnerだけでは階調が完璧でない場合
- 特定の濃度域（ハイライト・シャドウ）を微調整したい場合
- より高精度なデジタルネガティブを目指す場合

---

### Fine Tuning の3つの方法

#### 方法1: Absolute Beginner を高解像度で再測定

**手順**:
1. **Step Wedge をより細かいポイント数で再生成**
   - 21ポイント → 52ポイント または 256ポイント
   - EDN Absolute Beginner で Maximum 版を選択

2. **同じ手順で測定**
   - プリント → UV露光 → スキャン → LUT生成

3. **効果**:
   - より滑らかな階調補正カーブ
   - トーンジャンプの軽減

**メリット**: 最も体系的で再現性が高い
**デメリット**: UV露光・現像が必要（時間と手間）

---

#### 方法2: Photoshop カーブ調整レイヤーで手動微調整

**手順**:

1. **現在のワークフローで1枚テストプリント**
   ```
   元画像（16bit）
     ↓
   Color Blocker LUT
     ↓
   Absolute Beginner LUT
     ↓
   階調反転
   ```

2. **UV露光・現像して銀塩プリントを作成**

3. **問題のある濃度域を特定**
   - ハイライトが飛んでいる → カーブでハイライトを下げる
   - シャドウが潰れている → カーブでシャドウを持ち上げる
   - 中間調が平坦 → カーブで中間調のコントラストを調整

4. **Photoshopでカーブ調整レイヤーを追加**
   ```
   【レイヤー構成】
   元画像（16bit RGB）
     ↓
   カラールックアップ: Color Blocker LUT
     ↓
   カラールックアップ: Absolute Beginner LUT
     ↓
   カーブ調整レイヤー ← ★Fine Tuning用
     ↓
   階調反転（ネガティブ化）
   ```

5. **微調整の実例**
   ```
   【ハイライトが飛ぶ場合】
   カーブの右上（ハイライト）を下げる
   入力255 → 出力240など

   【シャドウが潰れる場合】
   カーブの左下（シャドウ）を持ち上げる
   入力0 → 出力15など

   【中間調のコントラストを上げる】
   中間部分（入力128付近）でS字カーブ
   ```

6. **再度テストプリント → UV露光 → 評価**

7. **調整を繰り返す**

**メリット**:
- UV露光不要（Photoshop上で完結）
- 特定の濃度域をピンポイントで調整可能

**デメリット**:
- 最終的な銀塩プリントでの確認が必須
- 試行錯誤が必要

---

#### 方法3: カスタムStep Wedge による部分再測定

**手順**:

1. **問題のある濃度域だけのStep Wedgeを作成**
   ```
   例: ハイライト域（RGB 200-255）のみ
   20%, 22%, 24%, ..., 100%の細かいステップ
   ```

2. **その範囲だけ再測定**
   - プリント → UV露光 → スキャン
   - EDNツールまたは手動でLUT微調整

3. **既存のLUTと統合**
   - テキストエディタで.cubeファイルを編集
   - 特定範囲のRGB値だけ置き換え

**メリット**: 問題のある部分だけ高精度化
**デメリット**: .cubeファイルの手動編集が必要（上級者向け）

---

### Fine Tuning 推奨ワークフロー

```
【ステップ1】まずAbsolute Beginner（21ポイント）で基本補正
  ↓
【ステップ2】テストプリント → UV露光 → 評価
  ↓
【ステップ3-A】大きな問題がある場合
              → Absolute Beginner Maximum版（52/256ポイント）で再測定
  ↓
【ステップ3-B】部分的な微調整が必要な場合
              → Photoshopカーブ調整レイヤーで手動調整
  ↓
【ステップ4】満足できるまで繰り返し
```

---

### Fine Tuning で調整すべきポイント

#### 1. ハイライト域（明るい部分）

**確認方法**:
- 白い雲、肌のハイライト、雪など
- 銀塩プリントで階調が残っているか

**調整**:
```
問題: 白飛びしている
  ↓
カーブでハイライトを下げる（入力255 → 出力230など）
  ↓
ネガの濃度が増加 → UV遮断増加 → 銀塩プリントが暗くなる → 階調復活
```

#### 2. シャドウ域（暗い部分）

**確認方法**:
- 影、黒い服、暗い森など
- 銀塩プリントで真っ黒に潰れていないか

**調整**:
```
問題: 黒潰れしている
  ↓
カーブでシャドウを持ち上げる（入力0 → 出力20など）
  ↓
ネガの濃度が減少 → UV透過増加 → 銀塩プリントが明るくなる → 階調復活
```

#### 3. 中間調（グレー）

**確認方法**:
- 18%グレーカード、コンクリート、曇り空など
- 滑らかなグラデーションになっているか

**調整**:
```
問題: トーンジャンプ、階調が粗い
  ↓
Absolute Beginner を高解像度版（256ポイント）で再測定
  ↓
または、カーブで中間調を滑らかに調整
```

---

### 測定精度を上げるためのヒント

#### 1. スキャン時の注意

```
【重要】
□ スキャナーのガラス面を清掃
□ 印画紙を完全に乾燥させる
□ 自動補正をすべてOFF
□ 同じスキャナー設定を一貫して使用
```

#### 2. UV露光の一貫性

```
【重要】
□ 露光時間を正確に計測（タイマー使用）
□ UV露光機のウォームアップ時間を一定に
□ フィルムと印画紙の密着を均一に
□ 室温を一定に保つ
```

#### 3. 現像処理の一貫性

```
【重要】
□ 現像液の温度管理（±0.5℃以内）
□ 現像時間の厳守
□ 新鮮な現像液を使用
□ 撹拌方法を統一
```

### よくある問題

#### 問題1: グラデーションが粗い

**原因**:
- プリンター設定が間違っている
- Absolute Beginner LUTのポイント数が少ない（21ポイント）

**解決策**:
1. プリント設定を確認・修正
2. Maximum版LUTで再生成（52または256ポイント）

#### 問題2: 色かぶりがある

**原因**:
- Color Blockerが正しく機能していない
- カラーマネジメント設定の誤り

**解決策**:
1. Photoshopによるカラー管理に変更
2. Color Blocker LUTを再生成

#### 問題3: 濃度が合わない

**原因**:
- UV露光時間が不適切
- LUT測定時の露光と本番の露光が異なる

**解決策**:
1. 露光時間を調整
2. 同じ条件でLUT再測定

#### 問題4: 調整しても改善しない（Fine Tuning編）

**原因**:
- プリント設定が間違っている（最も多い）
- Absolute BeginnerのLUTが古い

**解決策**:
1. プリント設定を再確認（Photoshopによるカラー管理）
2. Absolute Beginnerを最初から再測定

#### 問題5: ハイライトとシャドウを同時に改善できない

**原因**:
- UV露光時間が不適切
- ネガの濃度範囲がプリンターの出力範囲を超えている

**解決策**:
1. UV露光時間を調整
2. 元画像のダイナミックレンジを圧縮（トーンカーブで事前調整）

## EDN使用上の重要なポイント

### 1. 一貫性の維持

```
【測定時と本番で同じ条件】
- プリンター設定（最重要）
- フィルムの種類
- UV露光時間
- 現像処理
- スキャナー設定
```

### 2. プリント設定の厳守

**最重要**: 「Photoshopによるカラー管理」+ 「一般 RGB プロファイル」（または汎用プロファイル）

これを守らないと、EDNの測定・補正が全て無効化されます。

### 3. LUTファイルの管理

```
【推奨ファイル命名規則】
ColorBlocker_[フィルム]_[プリンター]_[日付].cube
AbsoluteBeginner_[フィルム]_[プリンター]_[解像度]_[ポイント数]_[日付].cube

【例】
ColorBlocker_TPS100_PX1V_20260307.cube
AbsoluteBeginner_TPS100_PX1V_1440dpi_21pt_20260307.cube

【記録すべき情報】
- 作成日
- フィルム種類（TPS-100等）
- プリンター機種
- 解像度（1440×1440 dpi等）
- ポイント数（21/52/256）
- UV露光時間
- 現像条件
```

### 4. 定期的な再測定

以下の変更があった場合は再測定推奨：
- フィルムのロット変更
- プリンターのヘッドクリーニング実施
- インクカートリッジ交換
- 露光機のUVランプ交換
- 現像液の交換

### 5. 16bitワークフローの推奨

```
【8bit vs 16bit】
8bit（256階調）
  ↓ LUT変換
実質100-150段階に減少（トーンジャンプ）

16bit（65536階調）
  ↓ LUT変換
数千段階を維持（滑らか）
```

可能な限り16bit/チャンネルで作業してください。
