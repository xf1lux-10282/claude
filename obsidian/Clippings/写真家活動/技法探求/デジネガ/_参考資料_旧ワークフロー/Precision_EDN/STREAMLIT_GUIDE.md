## 🔄 ワークフロー

### ステップ1: データ入力

#### オプションA: テンプレート作成

1. **「データ入力」タブ** を開く
2. **「テンプレートCSVを作成」** ボタンをクリック
3. **「テンプレートをダウンロード」** ボタンでCSVを保存
4. エクセル/テキストエディタで開く
5. 測定データを入力
6. 保存

#### オプションB: 既存CSVアップロード

1. **「データ入力」タブ** を開く
2. **「CSVアップロード」** エリアにファイルをドラッグ
3. または **「Browse files」** をクリックして選択

### ステップ2: データプレビュー

アップロード後、自動的に:
- ✓ データ表形式で表示
- ✓ 統計情報表示（データ数、測定完了数）
- ✓ データ検証（エラーチェック）

**確認ポイント**:
```
✓ データは正常です。LUT生成を実行できます
```

このメッセージが出ればOK。

### ステップ3: LUT生成

1. **「LUT生成」タブ** を開く
2. サイドバーで **出力名** を確認/変更
3. **「🚀 LUT生成を実行」** ボタンをクリック

**処理中**:
```
解析中...
📊 データを読み込み中...      [10%]
📈 カーブを解析中...          [30%]
🔄 逆カーブを生成中...        [50%]
💾 LUTをエクスポート中...     [70%]
📊 グラフを生成中...          [90%]
✓ 完了！                     [100%]
```

### ステップ4: 結果確認・ダウンロード

**解析グラフ表示**:
- 左: 入力 → ネガ濃度
- 中央: ネガ濃度 → プリント濃度
- 右: 入力 → プリント濃度

**ダウンロードボタン**:
- 📥 **CUBE LUT** - Photoshop/Lightroomで使用
- 📥 **カーブCSV** - 33点カーブデータ
- 📥 **レポート** - 解析レポート（テキスト）
- 📥 **解析グラフ (PDF)** - 高解像度グラフ

## 💡 使用例

### 初回キャリブレーション

1. **テンプレート作成**
   ```
   データ入力タブ → テンプレート作成 → ダウンロード
   ```

2. **測定①実施**
   - 33ステップチャート印刷
   - ネガ濃度測定
   - CSVの `negative_density` 列に入力

3. **測定②実施**
   - プラチナプリント作成
   - プリント濃度測定
   - CSVの `print_density` 列に入力

4. **CSVアップロード**
   ```
   データ入力タブ → CSVアップロード
   ```

5. **LUT生成**
   ```
   LUT生成タブ → 出力名入力 → LUT生成実行
   ```

6. **ダウンロード**
   ```
   CUBE LUT → [作品名]_v1.cube として保存
   ```

### 再キャリブレーション（条件変更時）

用紙/FO濃度/プリンターを変更した場合:

1. 同じテンプレートで再測定
2. 新しいCSVアップロード
3. 出力名を変更（例: `precision_edn_v2_washi_kozo`）
4. LUT生成
5. 新しいCUBE LUTをダウンロード

## ⚙️ サイドバー設定

### 出力設定

**出力ファイル名**:
- デフォルト: `precision_edn_v2`
- 推奨命名: `precision_edn_v2_[用紙]_[FO濃度]_[日付]`
- 例: `precision_edn_v2_washi_kozo_high_FO_20260309`

### 設計パラメータ（表示のみ）

`config.py` で設定された値:
- **γ**: 1.8
- **Dmin**: 0.10
- **Dmax**: 2.10
- **中間調入力**: 128
- **中間調濃度**: 0.70

変更する場合は `config.py` を編集。

## 📊 グラフの見方

### 左グラフ: 入力 → ネガ濃度

- **青線**: 実測カーブ
- **緑背景**: 線形領域（プリンター特性が安定）
- **傾き**: 急 = 低γ、緩 = 高γ

**確認ポイント**:
- Dmin ≈ 0.10
- Dmax ≈ 2.10
- 入力128 → 濃度 ≈ 0.70

### 中央グラフ: ネガ濃度 → プリント濃度

- **紫線**: プラチナプリント特性
- **飽和検出**: プラチナが反応できる限界

**確認ポイント**:
- プリントDmax ≈ 2.1以上
- カーブが飽和していないか

### 右グラフ: 入力 → プリント濃度

- **オレンジ線**: 合成カーブ
- **緑破線**: 理想（リニア）
- **赤線**: 逆カーブ（補正）

**確認ポイント**:
- 合成カーブが理想に近いか
- 逆カーブが滑らか

## 🛠️ トラブルシューティング

### アプリが起動しない

**症状**: `streamlit: command not found`

**解決策**:
```bash
pip3 install streamlit
```

### ブラウザが開かない

**解決策**:
```
手動でブラウザを開く
→ http://localhost:8501
```

### CSVアップロードエラー

**症状**: 「必須列が不足」

**解決策**:
```csv
ヘッダーを確認:
input,negative_density,print_density
```

### データ検証エラー

**症状**: 「ネガ濃度が未入力」

**解決策**:
- `negative_density` 列にデータを入力
- 33行すべて入力必須

**症状**: 「プリント濃度が未入力」

**解決策**:
- 測定②を実施
- `print_density` 列にデータを入力

### LUT生成エラー

**症状**: 「解析エラー」

**解決策**:
1. データが33ステップか確認
2. 濃度値が妥当か確認（Dmin < Dmax）
3. 欠損値がないか確認

## 🎨 本番での使用

### Photoshopでの適用

1. **作品画像を開く**（16bit RGB）
2. **レイヤー > 新規調整レイヤー > カラールックアップ**
3. **3DLUT ファイル** をクリック
4. **ダウンロードした .cube ファイル** を選択
5. **読み込み** → 適用
6. **階調の反転**
7. **Color Blocker適用**
8. **16bit RGB TIFF保存**
9. **カラーモード印刷**

### Lightroomでの適用

1. **現像モジュール**を開く
2. **カラーグレーディング** → **LUT**
3. **カスタムLUT** → .cubeファイル選択
4. 書き出し（16bit TIFF）
5. Photoshopで反転・Color Blocker
