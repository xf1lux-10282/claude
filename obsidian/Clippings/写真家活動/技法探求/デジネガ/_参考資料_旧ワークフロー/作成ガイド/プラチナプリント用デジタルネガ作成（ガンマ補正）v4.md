## 第1部：理論的基礎とデータ多様性

### 1.1 ガンマ補正の真の理解

**従来の理解：**
「逆ガンマ補正」= γ2.2をγ1.0に戻す単純な処理

**実際の内容：**
```
第1層：基本リニア化（γ2.2→1.0）
　　　　↓
第2層：プラチナプリント材料特性補正
　　　　↓  
第3層：視覚的・美的最適化
```

### 1.2 データソース多様性の実態

**重要な発見：**
実際の運用で6人のデータ送信者中3人（50%）で「逆ガンマ補正が不要」

**原因分析：**
- リニア出力設定の普及（Capture One等）
- プラチナプリントコミュニティの高技術レベル
- ProPhoto RGB環境での特殊処理

**判定基準：**
```
18%グレー相当のRGB値測定：
RGB 40-50   → リニアデータ（逆ガンマ不要）
RGB 110-120 → γ2.2データ（逆ガンマ必要）
RGB 90-100  → γ1.8データ（要調査）
```

## 付録：技術仕様詳細

### A.1 数学的計算式

```
基本ガンマ補正式：
出力 = 255 × (入力/255)^0.45

明度計算式：
明度(%) = (RGB値/255)^0.45 × 100

変化率計算式：
変化率 = (次の明度% - 現在の明度%) / (次の位置% - 現在の位置%)

材料補正差分式：
差分 = 実際RGB - 理論RGB
```

### A.2 品質管理基準

```
RGB値精度：±1RGB（設計値との差）
明度精度：±1.5%（目標明度との差）
変化率許容範囲：±0.2（設計変化率との差）
処理時間：5分以内（2レイヤー処理）
成功率：95%以上（初回適用）
```

### A.3 ファイル管理

```
グラデーションマップファイル：
- GammaCorrection_Pure.grd（Module A）
- PlatinumMaterial_Specific.grd（Module B）
```
