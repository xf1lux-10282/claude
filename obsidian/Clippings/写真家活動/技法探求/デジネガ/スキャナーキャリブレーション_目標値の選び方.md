# スキャナーキャリブレーション目標値の選び方

## 基本方針

**最優先: ターゲット付属のリファレンスファイルを使用**
- Kodak IT8ターゲットには .it8 または .txt ファイルが付属
- そこに記載されている実測L*a*b*値が最も正確

## よく見る数値とその用途

### L*=96/50/10
- **用途**: フィルムスキャン、プリントスキャン
- **適用**: デジタルネガティブ制作、写真アーカイブ
- **ダイナミックレンジ**: 86 (広い)
- **シャドウディテール**: 良好

### L*=95/50/5
- **用途**: オフセット印刷、CMYK変換
- **適用**: 商業印刷向けスキャン
- **ダイナミックレンジ**: 90 (最も広い)
- **シャドウディテール**: 非常に良好

### L*=96/52/3
- **用途**: 高精度アーカイブ
- **適用**: 美術館・博物館のデジタル化
- **ダイナミックレンジ**: 93 (極めて広い)
- **特徴**: 中間調をわずかに明るめに設定

## デジネガ制作での推奨設定

```
目標値: L*=96/50/10
理由:
- プラチナ/パラジウムプリントは階調豊かなプロセス
- シャドウディテールの保持が重要
- L*=10 で十分な黒の深さとディテールを両立
```

## 実際の設定手順

### 1. リファレンスファイルの確認
```bash
# Kodak Q-60 ターゲットの場合
# 付属の .it8 ファイルを確認
# ファイル内のグレーパッチ値を確認
```

### 2. スキャナーソフトでの設定
- VueScan の場合:
  - Color タブ → IT8 calibration
  - リファレンスファイルを読み込み
  - 自動的に最適な目標値が設定される

- SilverFast の場合:
  - IT8 Calibration ダイアログ
  - ターゲットタイプを選択
  - 目標値は自動設定

### 3. カスタム設定する場合
**デジネガ用推奨:**
- White point: L*=96
- Mid gray: L*=50
- Black point: L*=10

**確認方法:**
キャリブレーション後、同じターゲットを再スキャンして
設定した目標値に近い値が得られているか確認

## よくある質問

### Q: 数値が少し違っても大丈夫？
A: L*値で±2以内なら実用上問題ない
  - L*=96 と L*=95 → ほぼ同じ
  - L*=50 と L*=52 → わずかに明るい中間調
  - L*=10 と L*=3 → シャドウ再現に違いあり

### Q: どの数値が「正しい」？
A: ターゲットの実測値が正解
  - 標準値はあくまで目安
  - 実際のターゲットは個体差がある
  - リファレンスファイル使用が最も正確

### Q: 数値を変えると何が変わる？
A: トーンカーブの形状が変わる
  - ハイライト値 → 明部の再現
  - 中間調値 → 全体のコントラスト
  - シャドウ値 → 暗部のディテール

## 結論

**迷ったら:**
1. ターゲット付属のリファレンスファイルを使う
2. それがない場合は L*=96/50/10 を使う
3. キャリブレーション後に実際のプリント結果で調整

**デジネガ制作では:**
- 数値の厳密性よりも「一貫性」が重要
- 一度決めた設定を継続して使用
- プリント結果を見ながら微調整
