## 発見した問題

### 1. カーブファイルの問題
- `/Library/Printers/QTR/quadtone/QuadP9500/PX1V-PtPd-xf1-3ink-v2-P9500.quad` が壊れたシンボリックリンク（自己参照）になっていた
- **解決策**: デスクトップから実ファイル（8.4KB）をコピー

### 2. 解像度の問題  
- 72dpi画像（595x842px）が拡大されて印刷される
- PPDの全解像度オプション（1440dpi、2880dpiなど）がすべて `HWResolution[720 720]` に設定されている
- これはQuadToneRIPの標準設計（解像度名はプリント品質モードを示す）

### 3. 給紙設定の問題
- PPDの `ripFeed` 設定がプリンターにロール紙を要求させる
- `<</MediaClass(Sheet)>>setpagedevice` 形式の設定がEPSONプリンターに正しく伝わらない
- **試した解決策**:
  - ripFeed設定に値を追加 → 失敗
  - ripFeedセクションを完全削除 → まだ失敗
