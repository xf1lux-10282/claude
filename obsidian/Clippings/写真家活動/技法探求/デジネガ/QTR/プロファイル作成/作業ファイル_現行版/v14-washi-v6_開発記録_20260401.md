## 目的

和紙にプラチナプリントする際、Input 240-255が全て紙白になってしまう問題を解決する。

## 問題の本質

### 観察された現象
- 和紙でInput 240-255が全て紙白になる
- 露光時間を延ばしても改善しない（6.75分 → 10分 → 11分）
- 中間調も同時に濃くなりすぎる問題が発生

### 原因分析
**和紙の物理特性**:
- 薄く、透過性が高い
- 光が通過しやすい

**v14の240-255領域の問題**:
- ネガのK値範囲が和紙には不十分
- 和紙を透過する際、光の差が識別できない

## 重大な誤解とその修正

### ❌ 誤った理解（v1～v3）: ハイライト圧縮

**誤った考え方**:
```
「240-255が紙白になる」
→ ネガ濃度差が大きすぎる
→ K値範囲を狭くする（圧縮）
→ ネガ濃度差を小さくする
```

**実装例（v3）**:
- K[240-255]: 28000-29400（1400 K範囲）
- v14の4400 K範囲の約1/3に圧縮

**結果**:
- さらに識別不能になる（逆効果）

### ✓ 正しい理解（v5～v6）: ハイライト拡張

**正しい考え方**:
```
「240-255が紙白になる」
→ ネガ濃度差が小さすぎる
→ K値範囲を広くする（拡張）
→ ネガ濃度差を大きくする
→ 和紙透過時の光の差も大きくなる
```

**デジタルネガの基本原理**:
```
K値が高い → インク量多い → ネガが濃い → 光を遮る → プリントが明るい
K値が低い → インク量少ない → ネガが薄い → 光が通る → プリントが暗い
```

**ネガの濃度差と階調表現**:
```
ネガのK値範囲が広い
  → ネガ濃度差が大きい
  → 透過光の差が大きい
  → プリントで階調が識別可能

ネガのK値範囲が狭い
  → ネガ濃度差が小さい
  → 透過光の差が小さい
  → プリントで階調が圧縮・消失
```

**問題**:

**解決**:
- Zone 2を24-240（240を含む）とする
- Zone 3を241-255とする
- これにより連続性を確保: K[239]=26894 → K[240]=27000（差分+106）

**実装**:
```python
# Zone 2: 24-240
zone2_steps = 240 - 24  # 216 steps
for i in range(1, zone2_steps + 1):
    k = round(zone2_k_start + (zone2_k_end - zone2_k_start) * i / zone2_steps)
    zone2_k_values.append(k)

# Zone 3: 241-255

for i in range(1, zone3_steps + 1):

```

## 学んだ教訓

### 1. プリントとネガの逆転関係

**最も重要な教訓**: プリントとネガでは全てが逆になる

#### プリント（デジタル画像）での考え方
```
明るくしたい → 高いピクセル値（255に近い）
暗くしたい   → 低いピクセル値（0に近い）

階調を豊かにしたい → ヒストグラムを広げる
階調を圧縮したい   → ヒストグラムを狭める
```

#### ネガでの考え方（完全に逆）
```
明るいプリントにしたい → 濃いネガ（高いK値）→ 光を遮る
暗いプリントにしたい   → 薄いネガ（低いK値）→ 光が通る

プリントで階調を豊かにしたい → ネガの濃度差を大きく（K値範囲を広く）
プリントで階調を圧縮したい   → ネガの濃度差を小さく（K値範囲を狭く）
```

#### 今回の誤りの本質

**問題**: 「和紙で240-255が全て紙白になる（階調が失われる）」

**❌ プリント思考（誤り）**:
```
プリントで階調が失われている
→ プリントで「圧縮」されている
→ ネガでも「圧縮」すればいい
→ K値範囲を狭くする（28000-29400）
結果: さらに悪化
```

**✓ ネガ思考（正しい）**:
```
プリントで階調が失われている
→ プリントで「圧縮」されている
→ ネガでは「拡張」する必要がある（逆！）
→ K値範囲を広くする（27000-32000）
結果: 階調が表現可能に
```

#### デジタルネガの基本原理

**K値とネガ濃度の関係**:
- K値 = インク量 = ネガ濃度
- 高いK値 = インク多い = 濃いネガ = 光を遮る = 明るいプリント
- 低いK値 = インク少ない = 薄いネガ = 光が通る = 暗いプリント

**K値範囲と階調の関係**:
- 広いK値範囲 = 大きな濃度差 = 大きな透過光の差 = プリントで豊かな階調
- 狭いK値範囲 = 小さな濃度差 = 小さな透過光の差 = プリントで階調圧縮

#### なぜ逆になるのか

**ネガ = 中間媒体**:
```
デジタル画像 → ネガ（反転） → 光 → プリント（反転）
              ↑               ↑
              ここで逆転      ここで逆転
```

1. デジタル画像でInput 255（白）
2. → ネガで高いK値（濃い、黒に近い）
3. → 露光で光を遮る
4. → プリントで紙白（反転して白）

**2段階の反転**: デジタル→ネガ（1回目の反転）、光→プリント（2回目の反転）

#### 実践的な思考法

**常にネガの立場で考える**:
- 「プリントを明るくしたい」→「ネガを濃くする」
- 「プリントを暗くしたい」→「ネガを薄くする」
- 「プリントの階調を豊かにしたい」→「ネガの濃度差を大きくする」
- 「プリントの階調を圧縮したい」→「ネガの濃度差を小さくする」

**今回のケース**:
```
プリントで240-255の階調が失われている
→ ネガ思考: ネガの240-255の濃度差を大きくする
→ 実装: K値範囲を拡張（27000-32000）
→ 結果: プリントで階調表現が可能に
```

### 2. 階調表現の原理

**ネガの濃度差と階調**:
- 広いK値範囲 = 大きな濃度差 = 豊かな階調
- 狭いK値範囲 = 小さな濃度差 = 階調圧縮

**和紙での問題**:
- 和紙は透過性が高い
- 小さな濃度差は識別できない
- したがってネガの濃度差を大きくする必要がある
