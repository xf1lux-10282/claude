## 問題の本質

### 観察された現象
- 和紙にプリントする際、Input 240-255が全て紙白になってしまう
- 露光時間を6.75min → 10min → 11minと延ばしても改善しない
- 中間調（ミッドトーン）が同時に濃くなりすぎる問題も発生

### 原因分析
1. **和紙の物理特性**
   - 薄く、透過性が高い
   - 光が通過しやすいため、ネガの濃度差が大きいと全て紙白になる

2. **v14の240-255領域の問題**
   - K値範囲: 29400-33800（4400 Kの幅）
   - この範囲が和紙には広すぎる
   - ネガの濃度差が大きすぎて、全て光が透過してしまう

3. **誤ったアプローチ（v14-washi, v14-washi-v2）**
   - K_maxを下げて全体を薄くしようとした
   - これは問題を悪化させた（ネガが薄い = 光が多く通る = より紙白に）

## v14-washi-v3の解決策: ハイライト圧縮

### 設計コンセプト
**240-255を狭いK値範囲に圧縮し、ネガの濃度差を最小化する**

## ハイライト圧縮の原理

### デジタルネガの基本
```
K値が高い → インク量多い → ネガが濃い → 光を遮る → プリントが明るい
K値が低い → インク量少ない → ネガが薄い → 光が通る → プリントが暗い
```

### 和紙での240-255問題
```
v14の場合:
Input 240: K=29400 → ネガ濃度 D1
Input 255: K=33800 → ネガ濃度 D2
濃度差 = D2 - D1 = 大きい

和紙は薄く透過性が高い
→ D1でもD2でも光が十分透過
→ 両方とも紙白に見える（階調が消失）
```

### v14-washi-v3の解決
```
v14-washi-v3の場合:
Input 240: K=28000 → ネガ濃度 d1
Input 255: K=29400 → ネガ濃度 d2
濃度差 = d2 - d1 = 小さい（v14の約1/3）

濃度差が小さい
→ 透過光の差も小さいが存在する
→ 和紙でも240-255の階調表現が可能
```
