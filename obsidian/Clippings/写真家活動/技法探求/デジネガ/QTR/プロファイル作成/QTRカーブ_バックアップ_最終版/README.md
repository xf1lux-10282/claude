## インストール方法

### 1. QTRカーブファイルのコピー

```bash
# v18をインストールする場合
sudo cp PX1V-PtPd-Linear-v18.quad /Library/Printers/QTR/quadtone/QuadP700/

# v22をインストールする場合
sudo cp PX1V-PtPd-v22.quad /Library/Printers/QTR/quadtone/QuadP700/
```

### 2. QTRカーブデータベースの更新

```bash
/Library/Printers/QTR/bin/quadcurves QuadP700
```

### 3. QTR Print-Toolでカーブを選択

- v18: `PX1V-PtPd-Linear-v18`
- v22: `PX1V-PtPd-v22`

## 使い分けの推奨

### v18を使用する場合
- ハイライト階調をより多く出したい場合
- 240-255で0.07Dの範囲が必要な場合
- 実績のある安定版を使いたい場合

### v22を使用する場合（推奨）
- 紙白1.25D制約を確実に守りたい場合
- より新しい最適化技術を使いたい場合
- v21で255が紙白を超過した問題を解決したい場合

**通常はv22の使用を推奨**

## 技術的背景

### v22の革新的な手法

**実測Quad値の直接再利用**:
1. v18のInput 240で実測1.16Dを出したQuad値（36407）
2. v21のInput 240で実測1.25Dを出したQuad値（38496）
3. これらを直接v22のInput 232とInput 255に使用

**驚異的な精度で予測通りの結果を達成**

## 重要な注意事項

### ⚠️ 絶対に実行してはいけないコマンド

```bash
defaults delete com.quadtonerip.QTR-Print-Tool
```

**理由**: QTR Print-Toolの全設定（ログイン情報含む）が消去される

### ✓ 正しいインストール手順

1. `sudo cp` でカーブファイルをコピー
2. `quadcurves` でデータベース更新
3. QTR Print-Toolでカーブを選択
