## 🌟 現行運用カーブ（4種類）

### **xf1-v14 (洋紙用、芸術的表現)** ⭐
- **K_max**: 33800（紙白保証）
- **Zone 2 Gradient**: 115.0 K/step（最も豊かな階調）
- **露光時間**: 6分45秒
- **特徴**: 最も緩やかなミッドトーン勾配で最大の階調表現
- **実用域**: Input 0-255（全範囲）
- **運用方針**: 豊かな階調を確保し、コントラスト等の調整はPhotoshopで行う

**設計思想:**
- 目標: 豊かな階調表現、黒を潰さず、白も消さない
- 方針: 最も豊かな階調（115.0 K/step）を提供し、後処理で柔軟に調整
- 理由: Photoshopでコントラスト調整可能なため、最大の階調情報を保持することが最適

---

### **xf1-v14-washi-v6 (和紙用、ハイライト拡張)** ⭐
- **K_max**: 32000
- **ハイライト範囲**: Input 240-255で5000 K（v14の4400 Kより1.14倍拡張）
- **露光時間**: 6分45秒
- **特徴**: 和紙のハイライト表現不足に対応、ネガ思考で設計
- **実用域**: Input 0-255（特にハイライト域）
- **運用方針**: 和紙でのハイライト階調を確保、洋紙でもハイライト重視の作品に有効

**設計思想:**
- 目標: 和紙でInput 240-255の階調を見せる
- 方針: ネガの濃度差を拡張（プリント思考の逆）
- v14との違い: Input 240-255のK値範囲を1.14倍に拡張

**開発背景:**
- v14で和紙プリント時、Input 240-255が紙白になる問題
- v1-v3の失敗: プリント思考（階調圧縮 → ネガも圧縮）→ 悪化
- v5-v6の成功: ネガ思考（階調豊か → ネガの濃度差拡張）→ 解決

---

### **xf1-v28 (紙白確実到達、K単色)** ⭐
- **K_max**: 40000（v26の33500から+19.4%）
- **Input 0-254**: v26と完全に同一（良好なシャドウバランス保持）
- **Input 255**: 40000（紙白確実到達）
- **露光時間**: **6分05秒**（v26の6分45秒から-40秒、-10%）
- **特徴**: v27失敗の教訓からシンプル戦略を採用
- **実用域**: Input 0-255（全範囲）
- **運用方針**: 紙白到達が重要な作品、K単色での純粋なモノトーン表現

**設計思想:**
- 目標: 紙白（濃度0.09以下）の確実な到達
- 方針: v26のシャドウバランスを保持 + Input 255のみ増加
- 理由: v27の複合的調整が裏目、シンプルな単一調整に集中

**開発背景:**
- v27失敗: Input 8下げ（2520→2080）→ シャドウ悪化（K値↓ → プリント濃↑）
- v28: シンプル戦略 = v26の0-254保持 + K[255]のみ40000
- 測定誤差の理解: ±0.01-0.02はハンドコーティング誤差範囲

---

### **xf1-v28-multiink (ディザリング解消、滑らかな階調)** ⭐
- **K値**: v28と完全に同一
- **GY (LKチャンネル)**: K値の3%
- **LGY (LLKチャンネル)**: K値の3%
- **露光時間**: **6分05秒**（v28と同じ）
- **特徴**: マルチインクでディザリングパターン解消
- **実用域**: Input 0-255（全範囲）
- **運用方針**: ディザリングパターンが気になる作品、より滑らかな階調表現

**設計思想:**
- 目標: プリンターのディザリングパターン解消
- 方針: K + GY 3% + LGY 3% = 3種類のインクでドット配置をランダム化
- 3%の根拠: シャドウ域（Input 8: K=2520 → GY=76）で十分なディザリング効果

**期待効果:**
1. v28と同じ濃度バランス（紙白到達）
2. ディザリングパターン解消（GY+LGY 各3%）
3. わずかな暖色傾向（グレーインク混合による）
4. より滑らかな階調表現

## Photoshop/Lightroom調整ガイド

### カーブ別の画像調整
```
v14は最も豊かな階調を提供するため、後処理で柔軟に調整可能

【基本方針】
- ハイライト: 積極的に持ち上げ可能
- ミッドトーン: 必要に応じてコントラスト調整
- シャドウ: 慎重に落とす
- ヒストグラム: 0-255の全域を使える

【コントラスト調整が必要な場合】
- Photoshopのカーブツールで中間調を調整
- v12相当のコントラスト: ミッドトーンを少し下げる
- v9相当のコントラスト: ミッドトーンをさらに下げる
- 調整の自由度が最も高い = v14の最大の利点
```

## v14単独運用の実践ワークフロー

### 1. Photoshopでの画像準備
```
1. RAW現像（Lightroom/Photoshop）
2. 階調調整（カーブ、レベル補正）
3. コントラスト調整（必要に応じて）
4. デジタルネガ出力（グレースケール、16bit TIFF）
```

### 2. QTRプリント
```
1. Print-Toolでxf1-v14選択
2. 用紙: Pictorico OHP（または使用する用紙）
3. プリント実行
```

### 3. プラチナ/パラジウムプリント
```
1. 感光液塗布
2. 乾燥
3. 露光: 6分45秒
4. 現像・定着
```

### 4. 評価とフィードバック
```
【プリント評価】
├─ 満足 → 完成
├─ コントラスト高めたい → Photoshopで中間調下げて再プリント
├─ コントラスト低めたい → Photoshopで中間調上げて再プリント
└─ 露光調整必要 → 露光時間微調整（±15秒程度）

重要: カーブは変えず、Photoshop側で調整
```
