# SP1v21 露光時間補正記録

**作成日**: 2026年3月19日
**カーブ**: PX1V-PtPd-SP1v21
**問題**: 露光オーバーによるハイライト階調の喪失
**解決策**: 8分15秒 → 7分への露光時間短縮

---

## 📋 問題の発見

### 症状

SP1v21ネガを使用して8分15秒で露光したプリントで、**Input 0-32のハイライト部分がほぼ同じ濃度**になり、階調が失われた。

**実測データ（反射濃度）:**
```
Input 0:  1.44D
Input 8:  1.46D
Input 16: 1.44D
Input 24: 1.44D
Input 32: 1.42D
```

→ Input 0-24で濃度差がわずか0.02D = **ハイライトが潰れている**

### 正常な状態との比較

適正露光の場合、Input 0-32で**0.1D以上の濃度差**が期待される:
```
期待値（適正露光）:
Input 0:  1.45D
Input 8:  1.40D  ← 0.05D減
Input 16: 1.35D  ← さらに0.05D減
Input 24: 1.28D  ← さらに0.07D減
Input 32: 1.20D  ← さらに0.08D減
```

## 🔧 露光時間の補正計算

### 方法1: 絞り値からの計算

**前提:**
- 現在の露光: 8分15秒 (495秒)
- 露光オーバー: 約1/3絞り分

**計算:**
```
1/3絞り補正係数 = 2^(-1/3) = 0.7937

適正露光時間 = 495秒 × 0.7937
             = 393秒
             ≈ 6分33秒
```

**実用的な値:**
- 計算値: 6分33秒
- 実際の設定: **7分 (420秒)** ← 切りの良い値

### 方法2: 比率からの逆算

**前提:**
- Input 0-32の濃度差: 0.02D（実測）
- 目標濃度差: 0.10D以上

**推定:**
```
必要な露光減少率 = 現在の濃度差 / 目標濃度差
                 = 0.02D / 0.10D
                 = 0.20 (20%程度の露光減が必要)

適正時間 = 495秒 × 0.80
         = 396秒
         ≈ 7分
```

両方法とも**7分 (420秒)** が適正値と推定される。

## 🔍 教訓

### なぜ露光オーバーが発生したか

**推測される原因:**
1. **露光時間の設定ミス**: 8分15秒は過去の設定値を流用した可能性
2. **ネガ濃度の変化**: SP1v21はハイライト拡張版で、Input 224-255の濃度が高い
   - しかし、Input 0-32は変更していないため、露光時間に影響しないはず
3. **環境変化**: 室温、湿度、薬品の劣化など
4. **光源の変化**: UV光源の経年劣化または電圧変動

**最も可能性が高い原因:**
- 過去のテスト（v7/v18/v19）で適正だった8分15秒を使い続けていた
- しかし、薬品の活性や光源の強度が変化していた

### 今後の対策

1. **露光テストの定期実施**
   - 新しいネガを使う前に、必ず露光テストを実施
   - 6分、7分、8分、9分の4パターンでテストプリント

2. **ハイライト階調の確認**
   - Input 0-32の測定を必須にする
   - 濃度差が0.2D以上あることを確認

3. **露光時間の記録**
   - すべてのプリントに露光時間を記録
   - 環境条件（室温、湿度）も記録

4. **基準プリントの保存**
   - 適正露光のプリントを保存
   - 次回からこれを基準に露光時間を調整
