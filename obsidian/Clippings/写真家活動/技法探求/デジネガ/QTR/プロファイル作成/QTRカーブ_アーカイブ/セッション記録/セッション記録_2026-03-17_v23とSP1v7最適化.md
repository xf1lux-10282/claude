## 🧠 重要な学び

### 1. 計算式の限界

```
SLOPE = 0.007387
INTERCEPT = 0.0935

この計算式はv22/v23から導出したものだが、完璧ではない。
実測値を信頼し、必要に応じて逆算でQuad値を決定すべき。
```

**例:**
- v23 Input 248: 計算1.20D vs 実測1.23D（差+0.03D）
- v21 Input 255: 計算1.29D vs 実測1.34D（差+0.05D）

### 2. 化学処方の重要性

```
QTRカーブ最適化だけでは不十分。
化学的なバランス（FO/金属モル比、遊離シュウ酸）が
最終的なプリント品質を決定する。
```

**v23の失敗から:**
- カーブは正しく生成された
- しかし化学処方の問題でハイライトが飽和
- → 化学処方の最適化が先決

### 3. 段階的最適化の有効性

```
SP-1v4 → SP-1v5 → SP-1v7
各段階で実測結果に基づき微調整を重ねることで、
最終的に高精度なカーブを達成。
```

**プロセス:**
1. 目標値の設定（BK+GYの測定データ）
2. 初期カーブ生成（v21/v22/v23から選択）
3. 実測と比較（measurement_QTR-SP-4.csv）
4. 誤差の大きい箇所を特定
5. Quad値を調整（SP-1v5）
6. ユーザー要望に応じて拡張（SP-1v7）

### 4. 境界線リスクの定量化

```
Quad差分 > 200: 境界線リスクあり
Quad差分 > 500: 高リスク
Quad差分 > 2000: 極めて高リスク

各ステップの濃度差:
  < 0.05D: 許容範囲
  0.05-0.08D: 境界線が見える可能性
  > 0.10D: 明確な境界線
```

**SP-1シリーズの比較:**
- SP-1v6: Quad差分3,863、濃度差0.11D → 不採用
- SP-1v7: Quad差分2,485、濃度差0.071D → 採用

## 🎓 技術的詳細

### SP-1v7のQuad値生成コード

```python
# SP-1v5をベースに使用
sp1v5_quads = read_quad_file('PX1V-PtPd-SP1v5.quad')

# Input 255を1.7Dに設定
target_density_255 = 1.7
quad_255_new = density_to_quad(target_density_255)  # = 55891

# SP-1v7を初期化
sp1v7_quads = np.copy(sp1v5_quads)
sp1v7_quads[255] = quad_255_new

# Input 248-254を線形補間
anchor_inputs = [248, 255]
anchor_quads = [sp1v7_quads[248], quad_255_new]

for i in range(249, 255):
    ratio = (i - 248) / (255 - 248)
    sp1v7_quads[i] = int(round(
        anchor_quads[0] + ratio * (anchor_quads[1] - anchor_quads[0])
    ))

# 単調性を保証
for i in range(1, 256):
    if sp1v7_quads[i] < sp1v7_quads[i-1]:
        sp1v7_quads[i] = sp1v7_quads[i-1]
```

### 濃度逆算関数

```python
def density_to_quad(density):
    """濃度からQuad値を逆算"""
    baseline = (density - INTERCEPT) / SLOPE
    quad = int(round(baseline * 257.0))
    return quad

# 例: 1.7D → Quad 55891
```

### Quad値上限の確認

```python
if quad_255_new > 65535:
    print(f"⚠️ 警告: Quad値が上限を超過")
    quad_255_new = 65535  # 上限に制限
    actual_density = quad_to_density(quad_255_new)
    print(f"実際の濃度: {actual_density:.2f}D")
```
