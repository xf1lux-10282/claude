## コントラスト問題の診断

### コントラストが高すぎる症状

**視覚的特徴**:
- ハイライトが白飛び（ディテール消失）
- シャドウが黒潰れ（ディテール消失）
- 中間調が急激に変化（滑らかさがない）

**測定による確認**:
```python
#!/usr/bin/env python3
"""
現在のカーブのコントラスト診断
"""

import pandas as pd
import numpy as np

# 実測データ読み込み
data = pd.read_csv('measurement_QTR-current.csv')

# 濃度範囲を確認
d_min = data['negative_density'].min()
d_max = data['negative_density'].max()
density_range = d_max - d_min

print(f"=== コントラスト診断 ===")
print(f"最小濃度 (D-min): {d_min:.4f}")
print(f"最大濃度 (D-max): {d_max:.4f}")
print(f"濃度範囲: {density_range:.4f}")

# 参考値との比較
print(f"\n=== 参考値 ===")
print(f"Pt/Pd標準: 0.09 - 1.22 (範囲: 1.13)")
print(f"低コントラスト: 0.09 - 1.00 (範囲: 0.91)")
print(f"高コントラスト: 0.09 - 1.50 (範囲: 1.41)")

if density_range > 1.20:
    print("\n⚠️ コントラストが高い可能性があります")
    recommended_max = d_min + 1.05
    print(f"推奨: D-maxを{recommended_max:.2f}に調整")
elif density_range < 0.95:
    print("\n⚠️ コントラストが低い可能性があります")
    recommended_max = d_min + 1.15
    print(f"推奨: D-maxを{recommended_max:.2f}に調整")
else:
    print("\n✓ 適切な濃度範囲です")
```

### 原因の特定

**チェックリスト**:
1. **露光時間**: 適切か？（Pt/Pd: 5-15分が一般的）
2. **現像時間**: 標準通りか？
3. **温度・湿度**: 極端な環境でないか？
4. **ネガ濃度**: Input 255が目標値（1.22）を大幅に超えていないか？
5. **印刷設定**: Ink Limitsが100%か？

## 調整方法の選択

### 決定フローチャート

```
コントラストが高い
    ↓
全体的に高い？
    ├─ Yes → 方法1: 濃度範囲の圧縮
    └─ No
        ↓
    ハイライトのみ問題？
        ├─ Yes → 方法3: 部分的調整（ハイライト）
        └─ No
            ↓
        シャドウのみ問題？
            ├─ Yes → 方法3: 部分的調整（シャドウ）
            └─ No
                ↓
            中間調が急激？
                ├─ Yes → 方法2: ガンマ補正
                └─ No → 方法4: カスタムトーンカーブ
```

## 方法1: 濃度範囲の圧縮

### 原理
Input 255（最大濃度）を下げることで、全体的なコントラストを下げる。

### 適用場面
- 全体的にコントラストが高すぎる
- ハイライトもシャドウも飛んでいる
- シンプルな調整で済ませたい

### 実装

```python
#!/usr/bin/env python3
"""
濃度範囲圧縮によるコントラスト調整
v18をベースに、Input 255の濃度を下げる
"""

import pandas as pd
import numpy as np
from scipy.interpolate import CubicSpline
import os

# スクリプトのディレクトリを取得
script_dir = os.path.dirname(os.path.abspath(__file__))

# v18の実測データ（v12データ）
v12_data = pd.read_csv(os.path.join(script_dir, 'measurement_QTR2-12.csv'))

# ========== 調整パラメータ ==========
# 元の濃度範囲: 0.0935 - 1.22 (範囲: 1.1265)
# 新しい濃度範囲を設定
TARGET_DENSITY_MIN = 0.0935  # 通常は変更しない
TARGET_DENSITY_MAX = 1.10     # 元: 1.22（約10%コントラスト低下）

# 圧縮比を計算
ORIGINAL_DMAX = 1.22
compression_ratio = (TARGET_DENSITY_MAX - TARGET_DENSITY_MIN) / (ORIGINAL_DMAX - TARGET_DENSITY_MIN)

print(f"=== コントラスト調整設定 ===")
print(f"元の濃度範囲: {TARGET_DENSITY_MIN:.4f} - {ORIGINAL_DMAX:.4f}")
print(f"新しい濃度範囲: {TARGET_DENSITY_MIN:.4f} - {TARGET_DENSITY_MAX:.4f}")
print(f"圧縮比: {compression_ratio:.3f} ({(1-compression_ratio)*100:.1f}%低下)\n")

# v14のスプライン補間（v18と同じ）
v14_control_points_input = np.array([40, 48, 104, 112])
v14_control_points_density = []
for cp in v14_control_points_input:
    original_density = v12_data[v12_data['input'] == cp]['negative_density'].values[0]
    # 圧縮適用
    compressed = TARGET_DENSITY_MIN + (original_density - TARGET_DENSITY_MIN) * compression_ratio
    v14_control_points_density.append(compressed)
v14_spline = CubicSpline(v14_control_points_input, np.array(v14_control_points_density))

# v17のスプライン補間（v18と同じ）
v17_control_points_input = np.array([104, 112, 144, 192, 200])
v17_control_points_density = []
for cp in v17_control_points_input:
    original_density = v12_data[v12_data['input'] == cp]['negative_density'].values[0]
    # 圧縮適用
    compressed = TARGET_DENSITY_MIN + (original_density - TARGET_DENSITY_MIN) * compression_ratio
    v17_control_points_density.append(compressed)
v17_spline = CubicSpline(v17_control_points_input, np.array(v17_control_points_density))

# Baseline関係式（v18と同じ）
slope = 0.007387
intercept = 0.0935

# カーブデータ生成
remapped_curve_8bit = []
remapped_curve_16bit = []

for input_val in range(256):
    # v18と同じロジック
    if 48 <= input_val <= 104:
        if input_val % 8 == 0:
            target_density = v14_spline(input_val)
        else:
            lower = (input_val // 8) * 8
            upper = lower + 8
            lower_density = v14_spline(lower)
            upper_density = v14_spline(upper)
            ratio = (input_val - lower) / 8
            target_density = lower_density + ratio * (upper_density - lower_density)

    elif 112 <= input_val <= 192:
        if input_val in [112, 120, 128, 136, 144, 152, 160, 168, 176, 184, 192]:
            target_density = v17_spline(input_val)
        else:
            lower = (input_val // 8) * 8
            upper = lower + 8
            lower_density = v17_spline(lower)
            upper_density = v17_spline(upper)
            ratio = (input_val - lower) / 8
            target_density = lower_density + ratio * (upper_density - lower_density)

    elif input_val % 8 == 0:
        # v12実測値を圧縮
        original_density = v12_data[v12_data['input'] == input_val]['negative_density'].values[0]
        target_density = TARGET_DENSITY_MIN + (original_density - TARGET_DENSITY_MIN) * compression_ratio

    else:
        # 線形補間
        lower = (input_val // 8) * 8
        upper = lower + 8
        upper = min(upper, 255)

        if 48 <= lower <= 104:
            lower_density = v14_spline(lower)
        elif 112 <= lower <= 192:
            lower_density = v17_spline(lower)
        else:
            original = v12_data[v12_data['input'] == lower]['negative_density'].values[0]
            lower_density = TARGET_DENSITY_MIN + (original - TARGET_DENSITY_MIN) * compression_ratio

        if 48 <= upper <= 104:
            upper_density = v14_spline(upper)
        elif 112 <= upper <= 192:
            upper_density = v17_spline(upper)
        else:
            original = v12_data[v12_data['input'] == upper]['negative_density'].values[0]
            upper_density = TARGET_DENSITY_MIN + (original - TARGET_DENSITY_MIN) * compression_ratio

        ratio = (input_val - lower) / (upper - lower)
        target_density = lower_density + ratio * (upper_density - lower_density)

    # Quad値計算
    baseline = (target_density - intercept) / slope
    baseline = np.clip(baseline, 0, 255)
    remapped_curve_8bit.append(baseline)
    remapped_curve_16bit.append(int(round(baseline * 257)))

# 検証
print("=== 主要ポイント検証 ===")
print(f"{'Input':<6} {'濃度':<10} {'元のv18':<10} {'差分':<10}")
print("-" * 40)
for inp in [0, 128, 200, 255]:
    current_density = remapped_curve_8bit[inp] * slope + intercept
    original_density = v12_data[v12_data['input'] == inp]['negative_density'].values[0]
    diff = current_density - (TARGET_DENSITY_MIN + (original_density - TARGET_DENSITY_MIN) * compression_ratio)
    print(f"{inp:<6} {current_density:.4f}   {original_density:.4f}   {diff:+.6f}")

# .quadファイル生成
quad_content = f"""## QuadToneRIP K,C,M,Y,LC,LM,LK,LLK,V,MK
# QuadProfile Version 2.8.0
# PX1V-PtPd-Linear v18-LowContrast - Compressed density range
# Original: 0.0935 - 1.22, New: {TARGET_DENSITY_MIN:.4f} - {TARGET_DENSITY_MAX:.4f}
# Compression: {compression_ratio:.3f} ({(1-compression_ratio)*100:.1f}% lower contrast)
# BOOST_K=0 - NO BOOST
# K curve
"""

for val in remapped_curve_16bit:
    quad_content += f"{val}\n"

# 他のインクカーブ
for ink_name in ['C', 'M', 'Y', 'LC', 'LM', 'LK', 'LLK', 'V', 'MK']:
    quad_content += f"# {ink_name} curve\n"
    for _ in range(256):
        quad_content += "0\n"

quad_path = os.path.join(script_dir, 'PX1V-PtPd-Linear-v18-LowContrast.quad')
with open(quad_path, 'w') as f:
    f.write(quad_content)

print(f"\n✓ .quadファイル生成完了")
print(f"  保存先: {quad_path}")
print(f"\n次のステップ:")
print(f"  1. sudo cp {quad_path} /Library/Printers/QTR/quadtone/QuadP700/")
print(f"  2. /Library/Printers/QTR/bin/quadcurves QuadP700")
print(f"  3. テストプリントで効果を確認")
```

### 推奨値

| コントラスト感 | Input 255濃度 | 圧縮比 | 用途 |
|--------------|--------------|--------|------|
| 非常に高い | 1.22 | 1.00 | v18標準 |
| やや高い | 1.15 | 0.94 | 微調整 |
| 中間 | 1.10 | 0.90 | バランス重視 |
| やや低い | 1.05 | 0.85 | 柔らかい表現 |
| 非常に低い | 1.00 | 0.81 | ハイキー向け |

## 方法2: ガンマ補正

### 原理
中間調を明るく/暗くすることで、コントラストを調整。

### 適用場面
- 中間調が急激に変化している
- ハイライトとシャドウは良いが、中間だけ問題
- より自然なトーンカーブが欲しい

### ガンマ値の意味

```
gamma < 1.0: 中間調を明るく → コントラスト低下
gamma = 1.0: 変化なし（リニア）
gamma > 1.0: 中間調を暗く → コントラスト上昇
```

### 実装

```python
#!/usr/bin/env python3
"""
ガンマ補正によるコントラスト調整
v18をベースに、中間調を調整
"""

import pandas as pd
import numpy as np
import os

script_dir = os.path.dirname(os.path.abspath(__file__))
v18_quad_path = os.path.join(script_dir, 'PX1V-PtPd-Linear-v18.quad')

# ========== 調整パラメータ ==========
GAMMA = 0.85  # < 1.0でコントラスト低下、> 1.0でコントラスト上昇

print(f"=== ガンマ補正設定 ===")
print(f"ガンマ値: {GAMMA:.2f}")
if GAMMA < 1.0:
    print(f"効果: 中間調を明るく、コントラスト低下")
elif GAMMA > 1.0:
    print(f"効果: 中間調を暗く、コントラスト上昇")
else:
    print(f"効果: 変化なし")

# v18のQuad値を読み込み
def read_quad_k_curve(quad_path):
    with open(quad_path, 'r') as f:
        lines = f.readlines()

    k_start = None
    for i, line in enumerate(lines):
        if '# K curve' in line:
            k_start = i + 1
            break

    k_values = []
    for i in range(k_start, k_start + 256):
        k_values.append(int(lines[i].strip()))

    return k_values

v18_quad_values = read_quad_k_curve(v18_quad_path)

# ガンマ補正適用
gamma_corrected_quad = []

for input_val in range(256):
    # Input値を0-1に正規化
    normalized = input_val / 255.0

    # ガンマ補正
    corrected = pow(normalized, GAMMA)

    # 補正後のInput値
    corrected_input_float = corrected * 255.0

    # v18カーブから補間して対応するQuad値を取得
    # 線形補間
    lower_input = int(np.floor(corrected_input_float))
    upper_input = int(np.ceil(corrected_input_float))

    if lower_input == upper_input:
        quad_value = v18_quad_values[lower_input]
    else:
        ratio = corrected_input_float - lower_input
        quad_value = int(round(
            v18_quad_values[lower_input] * (1 - ratio) +
            v18_quad_values[upper_input] * ratio
        ))

    gamma_corrected_quad.append(quad_value)

# 検証
print("\n=== 主要ポイント検証 ===")
print(f"{'Input':<6} {'v18 Quad':<10} {'Gamma Quad':<12} {'差分':<10}")
print("-" * 50)
for inp in [0, 64, 128, 192, 255]:
    v18_val = v18_quad_values[inp]
    gamma_val = gamma_corrected_quad[inp]
    diff = gamma_val - v18_val
    print(f"{inp:<6} {v18_val:<10} {gamma_val:<12} {diff:+10}")

# .quadファイル生成
quad_content = f"""## QuadToneRIP K,C,M,Y,LC,LM,LK,LLK,V,MK
# QuadProfile Version 2.8.0
# PX1V-PtPd-Linear v18-Gamma{GAMMA:.2f} - Gamma corrected
# Base: v18, Gamma: {GAMMA:.2f}
# BOOST_K=0 - NO BOOST
# K curve
"""

for val in gamma_corrected_quad:
    quad_content += f"{val}\n"

for ink_name in ['C', 'M', 'Y', 'LC', 'LM', 'LK', 'LLK', 'V', 'MK']:
    quad_content += f"# {ink_name} curve\n"
    for _ in range(256):
        quad_content += "0\n"

quad_path = os.path.join(script_dir, f'PX1V-PtPd-Linear-v18-Gamma{GAMMA:.2f}.quad')
with open(quad_path, 'w') as f:
    f.write(quad_content)

print(f"\n✓ .quadファイル生成完了")
print(f"  保存先: {quad_path}")
```

### 推奨ガンマ値

| ガンマ値 | 効果 | 用途 |
|---------|------|------|
| 0.70 | 大幅に明るく | ハイキー専用 |
| 0.85 | 適度に明るく | コントラスト低下 |
| 1.00 | 変化なし | v18標準 |
| 1.15 | 適度に暗く | コントラスト上昇 |
| 1.30 | 大幅に暗く | ローキー専用 |

## 方法3: 部分的調整

### 原理
特定の範囲（ハイライト、シャドウ、中間調）のみを調整。

### 適用場面
- ハイライトのみ飛んでいる
- シャドウのみ潰れている
- 一部の範囲だけ問題がある

### ハイライト圧縮の実装

```python
#!/usr/bin/env python3
"""
ハイライト範囲のみ圧縮
Input 200-255のコントラストを下げる
"""

import pandas as pd
import numpy as np
from scipy.interpolate import CubicSpline
import os

script_dir = os.path.dirname(os.path.abspath(__file__))
v12_data = pd.read_csv(os.path.join(script_dir, 'measurement_QTR2-12.csv'))

# ========== 調整パラメータ ==========
HIGHLIGHT_START = 200  # ハイライト開始Input値
COMPRESSION_RATIO = 0.80  # ハイライトの圧縮率（80% = 20%低下）

print(f"=== ハイライト圧縮設定 ===")
print(f"対象範囲: Input {HIGHLIGHT_START}-255")
print(f"圧縮率: {COMPRESSION_RATIO:.0%} ({(1-COMPRESSION_RATIO)*100:.0f}%低下)\n")

# v18のスプライン（v14, v17）は変更なし
# （省略: v18生成スクリプトと同じ）

# ハイライト範囲のスプライン補間
highlight_control_points = np.array([192, 210, 230, 255])
highlight_densities = []

slope = 0.007387
intercept = 0.0935
TARGET_DENSITY_MIN = 0.0935

for cp in highlight_control_points:
    original_density = v12_data[v12_data['input'] == cp]['negative_density'].values[0]

    # ハイライトのみ圧縮
    # 基準点（D-min）からの距離を圧縮
    compressed = TARGET_DENSITY_MIN + (original_density - TARGET_DENSITY_MIN) * COMPRESSION_RATIO
    highlight_densities.append(compressed)

highlight_spline = CubicSpline(highlight_control_points, np.array(highlight_densities))

# カーブ生成
remapped_curve_16bit = []

for input_val in range(256):
    if input_val >= HIGHLIGHT_START:
        # ハイライト範囲: 圧縮されたスプライン
        target_density = highlight_spline(input_val)
    elif 48 <= input_val <= 104:
        # v14範囲: 変更なし
        # （v18と同じロジック）
        pass
    elif 112 <= input_val <= 192:
        # v17範囲: 変更なし
        # （v18と同じロジック）
        pass
    else:
        # その他: v12実測値
        target_density = v12_data[v12_data['input'] == input_val]['negative_density'].values[0]

    baseline = (target_density - intercept) / slope
    baseline = np.clip(baseline, 0, 255)
    quad_value = int(round(baseline * 257))
    remapped_curve_16bit.append(quad_value)

# .quadファイル生成
quad_content = f"""## QuadToneRIP K,C,M,Y,LC,LM,LK,LLK,V,MK
# QuadProfile Version 2.8.0
# PX1V-PtPd-Linear v18-HighlightComp - Highlight compression
# Compressed range: Input {HIGHLIGHT_START}-255, Ratio: {COMPRESSION_RATIO:.0%}
# BOOST_K=0 - NO BOOST
# K curve
"""

for val in remapped_curve_16bit:
    quad_content += f"{val}\n"

for ink_name in ['C', 'M', 'Y', 'LC', 'LM', 'LK', 'LLK', 'V', 'MK']:
    quad_content += f"# {ink_name} curve\n"
    for _ in range(256):
        quad_content += "0\n"

quad_path = os.path.join(script_dir, 'PX1V-PtPd-Linear-v18-HighlightComp.quad')
with open(quad_path, 'w') as f:
    f.write(quad_content)

print(f"✓ .quadファイル生成完了: {quad_path}")
```

### 推奨設定

| 調整対象 | Input範囲 | 圧縮率 | 用途 |
|---------|----------|--------|------|
| ハイライト | 200-255 | 80-90% | ハイライト飛び防止 |
| 中間調 | 100-180 | 90-95% | 中間調を柔らかく |
| シャドウ | 0-80 | 90-95% | シャドウ潰れ防止 |

## 方法4: カスタムトーンカーブ

### 原理
Photoshopのトーンカーブのように、任意のポイントで濃度を調整。

### 実装

```python
#!/usr/bin/env python3
"""
カスタムトーンカーブ
複数のポイントでコントラストをコントロール
"""

import numpy as np
from scipy.interpolate import CubicSpline
import os

# ========== カスタムトーンカーブ定義 ==========
# Input値と目標濃度のペア
custom_curve = {
    0:    0.0935,   # シャドウ（変更なし）
    64:   0.35,     # ダークトーン（少し明るく）
    128:  0.70,     # 中間調（変更なし）
    192:  1.00,     # ハイライト（圧縮）
    255:  1.15      # 最大（圧縮）
}

print("=== カスタムトーンカーブ ===")
for input_val, density in custom_curve.items():
    print(f"  Input {input_val:>3}: {density:.4f}")

# スプライン補間
control_inputs = list(custom_curve.keys())
control_densities = list(custom_curve.values())

tone_curve_spline = CubicSpline(control_inputs, control_densities)

# カーブ生成
slope = 0.007387
intercept = 0.0935
remapped_curve_16bit = []

for input_val in range(256):
    target_density = tone_curve_spline(input_val)
    baseline = (target_density - intercept) / slope
    baseline = np.clip(baseline, 0, 255)
    quad_value = int(round(baseline * 257))
    remapped_curve_16bit.append(quad_value)

# .quadファイル生成
quad_content = """## QuadToneRIP K,C,M,Y,LC,LM,LK,LLK,V,MK
```

## トラブルシューティング

### 問題1: 調整後も改善しない

**原因**:
- 露光時間が不適切
- 現像プロセスの問題
- ネガの問題ではなく、印刷プロセスの問題

**対策**:
1. 露光時間を変えてテスト（±20%）
2. 現像時間・温度を確認
3. 新しい薬品で再度テスト

### 問題2: 調整しすぎて平坦になった

**原因**:
- 圧縮率が高すぎる（< 0.75）
- ガンマ値が極端（< 0.7 or > 1.3）

**対策**:
1. 圧縮率を0.85-0.95に設定
2. ガンマ値を0.8-1.2に設定
3. 段階的に調整（一度に大きく変えない）

### 問題3: 境界線が発生した

**原因**:
- カスタムトーンカーブの制御点が不適切
- スプライン補間の問題

**対策**:
1. 制御点を増やす（最低5点）
2. 制御点間隔を均等に
3. v18のスプライン補間を維持し、圧縮のみ適用

### 問題4: Input 255の濃度がずれる

**原因**:
- Baseline関係式の誤差
- 圧縮計算のミス

**対策**:
```python
# Input 255を固定値として扱う
if input_val == 255:
    target_density = TARGET_DENSITY_MAX  # 直接指定
else:
    # 通常の計算
```

## まとめ

### 方法の選択基準

| 症状 | 推奨方法 | 調整範囲 | 難易度 |
|------|---------|----------|--------|
| 全体的に高い | 方法1 | 全域 | 低 |
| 中間調が急 | 方法2 | 全域 | 中 |
| ハイライトのみ | 方法3 | 部分 | 中 |
| 複雑な調整 | 方法4 | 任意 | 高 |

### ベストプラクティス

1. **段階的調整**: 一度に大きく変えず、10%刻みで調整
2. **テストプリント**: 各調整後に必ずテストプリント
3. **記録**: 各バージョンのパラメータを記録
4. **バックアップ**: 元のv18は必ず保持

### 調整の記録

各調整版のファイル名例:
- `PX1V-PtPd-Linear-v18-LC10.quad` - 10%コントラスト低下
- `PX1V-PtPd-Linear-v18-G0.85.quad` - ガンマ0.85
- `PX1V-PtPd-Linear-v18-HL80.quad` - ハイライト80%圧縮
