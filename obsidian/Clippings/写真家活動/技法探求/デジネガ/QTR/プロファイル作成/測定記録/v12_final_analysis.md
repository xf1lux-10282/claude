## 📈 露光時間とK_maxの新しい理解

### 発見1: K_max増加と露光時間短縮の関係

**v7-NAからv12への変更**:
```
K_max: 33051 → 33800 (+2.3%)
露光時間: 7分 → 6.5分 (-7.1%)
```

**結果**: 紙白到達を維持しつつ、露光時間短縮に成功

**新しい原則**:
```
K_max +2.3% → 露光時間 -7.1% 可能
（従来の理解「K_max +10% → 露光時間 -15%」よりも緩やか）
```

### 発見2: 露光延長の非線形効果

**+0.5分（+7.7%）の効果**:
- Shadow (Input 0-24): +0.008D（濃度わずかに増加）
- Midtone (Input 24-240): -0.014D（濃度減少）
- Highlight (Input 240-255): -0.020D（濃度大幅減少）

**重要な発見**:
> 露光延長の効果はハイライト域ほど大きく、Shadow域では逆効果の場合もある

これは**階調曲線の非線形性**を反映している。

### 教訓1: K_maxと露光時間の最適バランス

**v12の実績**:
- K_max +2.3% → 露光時間 -7.1%で紙白到達維持

**新しい原則**:
```
K_max増加率の約3倍の露光時間短縮が可能
（例: K_max +3% → 露光時間 -10% 程度）
```

### 教訓2: 露光延長の部分的効果

**発見**:
- 露光延長はHighlight域で最も効果的（-0.020D/+0.5分）
- Midtone域では中程度の効果（-0.014D/+0.5分）
- Shadow域では逆効果の場合も（+0.008D/+0.5分）

**実用的指針**:
- Highlight調整には露光時間変更が有効
- Shadow/Midtone調整にはK-value設計変更が必要
