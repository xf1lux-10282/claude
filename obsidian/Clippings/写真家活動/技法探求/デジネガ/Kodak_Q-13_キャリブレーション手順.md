# Kodak Q-13 スキャナーキャリブレーション手順

## Q-13 ターゲットの特徴

### 基本情報
- **タイプ**: グレースケール専用キャリブレーションターゲット
- **ステップ数**: 20段階 (Step A〜T)
- **用途**: モノクロスキャン、グレースケール再現のキャリブレーション
- **デジネガ制作への適性**: 非常に高い（カラー情報不要）

### Q-13 vs Q-60
| 特徴 | Q-13 | Q-60 |
|------|------|------|
| パッチ数 | 20 (グレーのみ) | 240 (カラー+グレー) |
| 用途 | モノクロ専用 | カラー+モノクロ |
| IT8準拠 | 非準拠 | IT8.7準拠 |
| デジネガ向き | ◎ | ○ |

## キャリブレーション方法

### Option 1: スキャナーソフトの自動機能を使う（推奨）

#### VueScan の場合
```
1. Q-13ターゲットをスキャナーに配置
2. Color タブを開く
3. Scanner → "None" (キャリブレーション前)
4. Preview でターゲットをスキャン
5. Color タブ → "Scanner calibration" ボタンをクリック
6. グレーパッチ領域を手動で指定（Step A, J, T など）
7. VueScanが自動的にトーンカーブを作成
8. プロファイルを保存
```

#### SilverFast の場合
```
1. ターゲットを配置
2. Expert mode → Densitometer
3. グレーパッチを測定
4. Auto-adjust → Gray balance
5. 測定結果をプロファイルとして保存
```

### Option 2: マニュアルで3点キャリブレーション

Q-13の20ステップから**3つの代表点**を選んで設定:

```
ハイライト: Step A または B (最も明るい)
中間調:     Step J または K (中央付近)
シャドウ:   Step S または T (最も暗い)
```

**設定する目標値:**
```
ハイライト → L*=96
中間調     → L*=50
シャドウ   → L*=10
```

## 実測値がない場合の対処法

### 方法1: 分光測色計で実測（最も正確）
```bash
# i1Pro2 などの分光測色計がある場合
1. Q-13の各ステップを測定
2. L*a*b*値を記録
3. CSVファイルとして保存
4. スキャナーソフトに読み込み
```

### 方法2: 理論値を使用（実用的）
上記の表の標準値を使用してキャリブレーション

### 方法3: 相対的キャリブレーション
```
1. Q-13をスキャン
2. ヒストグラムを確認
3. Step A が RGB=(245,245,245) 程度になるよう調整
4. Step T が RGB=(10,10,10) 程度になるよう調整
5. 全体のトーンカーブが滑らかになるよう調整
```

## デジネガ制作での推奨ワークフロー

### 初回キャリブレーション
```
1. Q-13をスキャナーに配置
2. フラットベッドスキャナーの場合:
   - 解像度: 300-600 dpi
   - ビット深度: 16-bit グレースケール
   - 色補正: オフ

3. 3点測定方式:
   - Step A または B → 目標 L*=96
   - Step J または K → 目標 L*=50
   - Step S または T → 目標 L*=10

4. キャリブレーションプロファイルを保存
   例: "Scanner_Q13_2026-04-13.icc"
```

### 定期的な検証
```
- 月1回: Q-13を再スキャンして値を確認
- 補正が必要か判断
- 必要に応じて再キャリブレーション
```

## トラブルシューティング

### Q: Q-13のリファレンスファイルが見つからない
A:
1. 上記の標準値表を使用
2. または相対的キャリブレーションで対応
3. 厳密な色管理が必要な場合は分光測色計で実測

### Q: スキャナーソフトがQ-13に対応していない
A:
1. 手動で3点を測定してトーンカーブ調整
2. またはIT8形式のリファレンスファイルを自作:

```csv
# 簡易IT8形式（例）
# Q-13 グレースケール
SAMPLE_ID,LAB_L,LAB_A,LAB_B
A,96,0,0
J,50,0,0
T,10,0,0
```

### Q: キャリブレーション後も色が合わない
A:
1. スキャナーのウォームアップ（30分程度点灯）
2. ターゲットの配置位置を確認（スキャン領域中央）
3. ガラス面の清掃
4. 環境光の影響を排除（ターゲットを黒い紙で覆う）

## Q-13 を使うメリット（デジネガ制作の場合）

✓ **グレースケール専用なので精度が高い**
- カラー情報の干渉がない
- モノクロ階調再現に特化

✓ **20段階で細かいトーン確認が可能**
- Q-60のグレーパッチは20段階
- Q-13も20段階なので同等以上

✓ **シンプルで使いやすい**
- パッチ数が少ないので測定が楽
- キャリブレーションが早い

✓ **デジタルネガティブに最適**
- 階調再現が最重要
- カラー精度は不要

## まとめ

**Q-13でのキャリブレーション推奨設定:**
```
ハイライト: Step A → L*=96
中間調:     Step J → L*=50
シャドウ:   Step S → L*=10
```

**重要なポイント:**
- リファレンス値より「一貫性」が重要
- 同じ設定で継続的にキャリブレーション
- プリント結果を見ながら微調整
