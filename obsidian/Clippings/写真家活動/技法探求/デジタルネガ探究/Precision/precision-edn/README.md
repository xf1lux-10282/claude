## 特徴

### 🎯 精密補正
- **16bit完全対応**: 65536階調の滑らかなグラデーション
- **PCHIP補間**: トーンジャンプを完全に防ぐ単調性保持補間
- **4096ポイント対応**: 超高密度サンプリング

### 📊 詳細分析
- **トーンジャンプ検出**: 自動検出＆警告
- **品質スコア**: 総合評価（90点以上で後補正不要）
- **問題領域特定**: ピンポイントで改善箇所を提示

### 💾 豊富な出力形式
- LUT 1D (.cube) - 1024/256/52ポイント
- Screen Proof 3D LUT (.cube)
- Adobe Photoshop Curve (.acv)
- 測定データ (CSV)
- 設定メモ (TXT)
- **ZIP一括ダウンロード対応**

### 🌐 オフライン動作
- **完全にブラウザ内で処理**
- インターネット接続不要（初回読み込み後）
- データは一切外部送信されません

## 使い方

### 1. テストチャートの準備

#### EDNテストチャートを入手
- [Easy Digital Negatives公式サイト](http://www.easydigitalnegatives.com/)からダウンロード
- **EDN 256** (1507×1507px) **← 推奨！高精度補正に最適**
- EDN 101 (1003×1087px) も対応可能（精度は劣る）

#### プリント
```
Photoshop設定:
- カラー処理: Photoshopによるカラー管理
- プロファイル: 一般 RGB プロファイル
- マッチング方法: 相対的な色域を維持
- 黒点の補正: OFF

PX-1V設定:
- 解像度: 1440×1440 dpi（推奨）
- カラーマッチング: ColorSync
- カラー処理: 色補正なし
- Black Enhance: OFF
- 双方向印刷: OFF
- スムージング: OFF
```

#### プリントとスキャン

**最高精度を得るために（推奨）**:
1. 同じデジネガで**プリントを3枚制作**
   - プラチナプリント/銀塩プリントの場合、感光液の手塗布によるばらつきがあるため
   - 各プリントで露光・現像条件は統一
2. 各プリントを300dpi以上でスキャン（1回ずつ）
3. JPG または PNG で保存
4. **3枚全てをアップロード** → 自動で中央値フィルタ適用、ばらつきを平均化

**標準精度（時短）**:
- プリント1枚をスキャン（1回）でも使用可能

**重要**: デジネガでプリントした**最終プリント**（プラチナ/銀塩プリント）をスキャンします。テストチャート画像そのものではありません。

---

### 2. Precision EDNで処理

#### ステップ1: ファイルアップロード
1. `index.html` をブラウザで開く
2. スキャン画像をドラッグ&ドロップ
   - または、クリックしてファイル選択
3. 複数ファイル対応（自動平均化で精度向上）

#### ステップ2: 設定
```
テストチャート: EDN 256（推奨）
サンプリングポイント数: 1024（推奨）
補間方法: PCHIP（推奨）
```

**推奨設定の理由:**
- EDN 256 = 256階調測定、シャドウ部も高精度
- 1024ポイント = 10bit相当の精度
- PCHIP = トーンジャンプなし、単調性保持
- この組み合わせで後補正不要レベルを実現

#### ステップ3: 分析結果を確認
- **総合評価 90点以上**: 後補正不要！
- **80-90点**: 良好（そのまま使用可）
- **70-80点**: 可（必要に応じて微調整）
- **70点未満**: 再測定を推奨

#### ステップ4: ダウンロード
- 「全ファイル一括ダウンロード (ZIP)」をクリック
- LUT、Photoshop Curve、設定メモなど全てまとめてダウンロード

---

### 3. Photoshopでデジネガ作成

#### ワークフロー
```
RAW現像 (16bit TIFF)
  ↓
Photoshopで編集（16bit/チャンネル、AdobeRGB）
  ↓
EDN 3D LUT適用（Color Blocker）
  ↓
Precision EDN 補正LUT適用 ← ★ここで使用
  ↓
プリント（上記の推奨設定）
```

#### LUTの適用方法
1. `ファイル > カラー設定` → AdobeRGB
2. `レイヤー > 新規調整レイヤー > カラールックアップ`
3. 「カスタム」から生成した `.cube` ファイルを選択

## 用紙別プリセット管理

### 用紙を登録
1. 「➕ 新規用紙を登録」をクリック
2. 用紙名を入力（例: TPS-100、Pictorico Premium）
3. プリンター名を入力（例: PX-1V）
4. 備考（オプション）

### 用紙を切り替え
- ドロップダウンメニューから選択
- 各用紙ごとに最適化された補正が適用されます

### 設定のエクスポート/インポート
- LocalStorageに自動保存
- ブラウザを閉じても設定は保持されます

## 技術仕様

### 対応画像形式
- JPG, PNG, TIF
- EDN 256: 1507×1507px (±10px)
- EDN 101: 1003×1087px (±10px)

### 色空間変換
- RGB → XYZ → Lab (D50白色点)
- ITU-R BT.709 輝度係数
- 16bit精度を維持

### 補間アルゴリズム
1. **PCHIP** (推奨)
   - 単調性保持
   - トーンジャンプなし
   - オーバーシュートなし

2. **Catmull-Rom**
   - 最も滑らか
   - わずかにオーバーシュート可能性

3. **線形補間**
   - シンプル
   - 高速

### 出力形式
- **LUT 1D (.cube)**: テキスト形式、可変ポイント数
- **LUT 3D (.cube)**: スクリーンプルーフ用、21×21×21
- **Adobe Curve (.acv)**: バイナリ形式、15ポイント
- **CSV**: 測定データ（理想値/実測値/誤差）

## トラブルシューティング

### Q: 「サポートされていない画像サイズ」エラーが出る
A: **EDN 256 (1507×1507px)** を使用してください（推奨）。EDN 101 (1003×1087px) も対応可能です。±10pxの誤差は許容されます。

### Q: 品質スコアが低い（70点未満）
A: 以下を確認してください:
- スキャン品質（焦点、照明）
- プリント設定（カラーマネジメント、解像度）
- テストチャートの印刷品質
- **複数枚プリントして平均化**（プラチナ/銀塩の手作業による感光液塗布のばらつきを除去）

### Q: トーンジャンプが検出される
A: 以下を試してください:
1. 補間方法を「Catmull-Rom」に変更
2. サンプリングポイント数を増やす（2048, 4096）
3. 複数回スキャンして平均化

### Q: ダウンロードボタンが無効
A: まず画像をアップロードして処理を完了してください。
