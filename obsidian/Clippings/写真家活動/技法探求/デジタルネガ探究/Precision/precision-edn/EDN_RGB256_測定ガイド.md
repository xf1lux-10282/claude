## ⚠️ 重要：測定順序

EDN RGB256チャートは **右から左** の順序で配置されています。

```
        ← 測定方向（右から左）
┌─────────────────────────────────┐
│ 240 224 208 ... 32  16  0  │ 1行目
│ 241 225 209 ... 33  17  1  │ 2行目
│ 242 226 210 ... 34  18  2  │ 3行目
│  :   :   :       :   :  :  │
│ 255 239 223 ... 47  31  15 │ 16行目
└─────────────────────────────────┘
  左                      右
```

**チャート右端が値0（最暗）、左端が値240（最明）**

---

## 📏 正しい測定手順

### ステップ1: チャート配置確認

1. チャートを目の前に置く
2. **右側に「0」のラベル**があることを確認
3. **左側に「240」付近**の明るいパッチがあることを確認

### ステップ2: 測定開始位置

**1行目の右端（値0）から開始**

```
開始位置 →  [■] ← ここ（最も暗いパッチ）
           右端
```

### ステップ3: 測定順序（各行）

**各行を 右 → 左 に測定**

```
1行目（上）:
  右端(0) → (16) → (32) → (48) → ... → (224) → 左端(240)

2行目:
  右端(1) → (17) → (33) → (49) → ... → (225) → 左端(241)

3行目:
  右端(2) → (18) → (34) → (50) → ... → (226) → 左端(242)

...

16行目（下）:
  右端(15) → (31) → (47) → (63) → ... → (239) → 左端(255)
```

## 🎯 測定のポイント

### 濃度計の設定

- **透過濃度モード**: ネガ測定時
- **反射濃度モード**: プリント測定時
- **ISO Status T**: 透過濃度（推奨）
- **ISO Status M**: 反射濃度（推奨）

### 測定位置

```
┌─────────┐
│         │
│    ●    │ ← パッチ中央を測定
│         │
└─────────┘
```

- 各パッチの **中央** を測定
- エッジや境界線を避ける
- 各パッチ **3回測定** して平均（推奨）

### 測定環境

- 温度: 20〜25℃
- 湿度: 40〜60%
- 一定の環境で測定
- 直射日光を避ける

## 📊 データ入力フォーマット

### Precision EDN (densitometer.html) の場合

#### オプション1: 正しい順序で測定（推奨）

**右→左、上→下** の順序で測定し、濃度値のみを1行に1つずつ入力：

```
0.05   ← 値0（右端・最暗）
0.08   ← 値16
0.12   ← 値32
...
0.15   ← 値240（左端・最明）
0.06   ← 値1（2行目右端）
...
2.15   ← 値255（16行目左端）
```

チャートタイプ: **連番順（デフォルト）** を選択

#### オプション2: 左→右で測定してしまった場合

**左→右、上→下** で測定してしまった場合も大丈夫！

```
0.15   ← 左端から測定（値240のはずだが...）
...
0.05   ← 右端まで測定（値0のはずだが...）
...
```

チャートタイプ: **右→左（EDN RGB256）** を選択

→ プログラムが自動的に正しい順序に並び替えます ✓

### ステップ番号付きの場合

```
step 1  0.05
step 2  0.08
step 3  0.12
...
step 256  2.15
```

または CSV形式：

```
1,0.05
2,0.08
3,0.12
...
256,2.15
```

## ✅ 確認方法

### データ入力後の確認

グラフが **滑らかな曲線** になっていれば正しい：

```
濃度
 ↑
 │        ／
 │      ／
 │    ／
 │  ／
 │／
 └─────────→ ステップ

正しい：滑らかな曲線 ✓
```

### 間違った例

グラフが **階段状** になっている場合は順序が逆：

```
濃度
 ↑  ┐ ┐ ┐
 │  └─┘ └─┘
 │  ┌─┐ ┌─┐
 │  └─┘ └─
 └─────────→ ステップ

間違い：階段状 ✗
```

この場合は、測定順序を確認して再入力。

## 🔄 測定ワークフロー

### ネガ測定（透過濃度）

1. 濃度計を透過モードに設定
2. チャート右端（値0）から測定開始
3. 各行を右→左に測定
4. 濃度値を記録（測定順序のまま）
5. Precision EDNに入力
6. グラフ確認（滑らか？）

### プリント測定（反射濃度）

1. 同じネガでプラチナプリント作成
2. 完全乾燥（24時間推奨）
3. 濃度計を反射モードに設定
4. **ネガ測定と同じ順序**で測定
5. 濃度値を記録
6. Precision EDNに入力
7. グラフ確認

## 🚨 よくある間違い

### 間違い1: 左から右に測定

❌ **左端（240）から開始**
```
240 → 224 → 208 → ... → 16 → 0
```

これだと値が逆になり、グラフが階段状になります。

✅ **右端（0）から開始**
```
0 → 16 → 32 → ... → 224 → 240
```

### 間違い2: データを値順に並び替え

❌ **入力時に0, 1, 2, 3...の順に並び替える**

測定順序とステップ番号がズレます。

✅ **測定した順序でそのまま入力**

### 間違い3: 行の途中で折り返し

❌ **1行目の途中で改行して2行目へ**

各行は完全に右→左まで測定してから次の行へ。

✅ **1行を完全に測定してから次の行**

## 📖 参考情報

### EDN RGB256 チャート仕様

- サイズ: 1507 x 1507 ピクセル
- レイアウト: 16列 × 16行 = 256ステップ
- 配置: 右（暗）→ 左（明）
- ステップ値: 0-255
- 作成者: Peter Mrhar (www.easydigitalnegatives.com)

### 値の配置ルール

```
行番号(0-15) + 列番号(0-15) × 16 = ステップ値(0-255)

例:
1行目・右端(列0): 0 + 0×16 = 0
1行目・左端(列15): 0 + 15×16 = 240
2行目・右端(列0): 1 + 0×16 = 1
2行目・左端(列15): 1 + 15×16 = 241
...
16行目・左端(列15): 15 + 15×16 = 255
```
