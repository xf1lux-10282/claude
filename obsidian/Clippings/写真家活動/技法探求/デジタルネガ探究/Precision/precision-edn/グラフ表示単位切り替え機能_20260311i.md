### 4. ui.js - 設定の保存・復元

```javascript
    ...this.edn.settings,
    linearityDisplayMode: this.linearityDisplayMode,
    curveDisplayMode: this.curveDisplayMode
  };
  localStorage.setItem('precisionEDN_settings', JSON.stringify(settings));
}
```

**復元**:
```javascript
loadSettings() {
  const saved = localStorage.getItem('precisionEDN_settings');
  if (saved) {
    const settings = JSON.parse(saved);
    // ... 既存の設定復元 ...

    // グラフ表示モードを復元

      this.linearityDisplayMode = settings.linearityDisplayMode;

}
```

```javascript
updateLinearityChart(curve, measured, measurementsList = [], fileNames = []) {
  // 表示モードに応じてスケールを決定
  const displayMode = this.linearityDisplayMode || 'rgb';
  const yScale = displayMode === 'rgb' ? 255 : 1;

  // Y軸ラベルを更新
  const yAxisLabel = displayMode === 'rgb' ? 'RGB値 (0-255)' : '相対値 (0-1)';
  const labelElement = document.getElementById('linearityYAxisLabel');
  if (labelElement) {
    labelElement.textContent = yAxisLabel;
  }

  // データ準備（横軸=入力、縦軸=出力）
  // 理想線: y = x
  const idealData = [];
  for (let i = 0; i <= 255; i++) {
    idealData.push({ x: i, y: i * yScale / 255 });
  }

  // 測定値データ
  const measuredData = rgbData ? rgbData.map((measuredValue, i) => ({
    x: i,
    y: measuredValue * yScale  // 表示モードに応じたスケール
  })) : [];

  // 個別ファイルデータも同様にスケール
  const individualDatasets = measurementsList.map((fileMeasured, fileIndex) => {
    return {
      // ...
      data: fileMeasured.map((measuredValue, i) => ({
        x: i,
        y: measuredValue * yScale  // 表示モードに応じたスケール
      })),
      // ...
    };
  });

  // Y軸設定
  scales: {
    y: {
      type: 'linear',
      title: {
        display: true,
        text: displayMode === 'rgb' ? '出力（RGB値 0-255）' : '出力（相対値 0-1）'
      },
      min: 0,
      max: yScale,
      ticks: {
        stepSize: displayMode === 'rgb' ? 32 : 0.125,
        callback: function(value) {
          return displayMode === 'rgb' ? Math.round(value) : value.toFixed(2);
        }
      }
    }
  }
}
```

## 📊 グラフの意味

### リニアリティ検証グラフ

**目的**: 測定値が線形か確認

| 項目 | 説明 |
|------|------|
| **X軸** | ステップ番号 (0-255) |
| **Y軸** | 反射光量（RGB値） |
| **理想線** | y = x の直線 |
| **測定値** | 実際のRGB値（`curve.rawRgb`を使用、反転なし） |

**表示モード**:
- **RGB値 (0-255)**: 実際のRGB値で表示（デフォルト）
- **相対値 (0-1)**: 正規化された相対値で表示

### 補正カーブ

**目的**: 濃度を線形化するLUT生成

| 項目 | 説明 |
|------|------|
| **X軸** | 入力デジタル値 |
| **Y軸** | 出力デジタル値（LUT変換後） |
| **理想線** | y = x の直線 |
| **測定値平均（補正前）** | RGB値を濃度に変換（`curve.raw`、反転済み） |
| **補正カーブ（LUT）** | 濃度を線形化するための補正曲線 |

**表示モード**:
- **正規化 (0-1)**: 0-1の正規化値で表示（デフォルト）
- **デジタル値 (0-65535)**: 16bitデジタル値で表示

## 🔍 データフローの整理

```
測定RGB値（スキャン）
  ↓
  ├→ normalizeRgbForGraph()
  │   └→ RGB値をそのまま正規化（反転なし）
  │       └→ リニアリティ検証グラフで表示（curve.rawRgb）
  │
  └→ normalizeMeasurements()
      └→ RGB値を濃度に変換（反転あり）
          └→ 補正カーブ生成に使用（curve.raw）
```

### 具体例

測定結果:
- ステップ0: RGB=5（最暗）
- ステップ128: RGB=128（中間）
- ステップ255: RGB=250（最明）

**`normalizeRgbForGraph()`（反転なし）**:
```
正規化[0]   = (5 - 5) / (250 - 5) = 0 / 245 = 0
正規化[128] = (128 - 5) / (250 - 5) = 123 / 245 ≈ 0.502
正規化[255] = (250 - 5) / (250 - 5) = 245 / 245 = 1
```
→ リニアリティ検証グラフ:
- RGB値モード: 0 → 128 → 250（右上がり）
- 相対値モード: 0 → 0.502 → 1（右上がり）

**`normalizeMeasurements()`（反転あり）**:
```
濃度[0]   = (250 - 5) / (250 - 5) = 245 / 245 = 1（高濃度）
濃度[128] = (250 - 128) / (250 - 5) = 122 / 245 ≈ 0.498
濃度[255] = (250 - 250) / (250 - 5) = 0 / 245 = 0（低濃度）
```
→ 補正カーブ:
- 正規化モード: 1 → 0.498 → 0
- デジタル値モード: 65535 → 32634 → 0

## 🎯 使い方

### リニアリティ検証グラフ

1. **RGB値 (0-255)** を選択（デフォルト）
   - 実際のRGB値で表示
   - わかりやすい絶対値

2. **相対値 (0-1)** を選択
   - 正規化された相対値で表示
   - 他のグラフと単位を統一したい場合

### 補正カーブ

1. **正規化 (0-1)** を選択（デフォルト）
   - 0-1の正規化値で表示
   - 理論的な理解に適している

2. **デジタル値 (0-65535)** を選択
   - 16bitデジタル値で表示
   - 実際のLUT値を確認したい場合
