## プロジェクトの背景

### 根本的な問題: デジタルネガの非線形性

プラチナプリントを作る際、以下の3つの非線形が存在する:

```
デジタル画像（入力値0-255）
  ↓ 【非線形①】
プリンター（インクジェット）の特性
  ↓
ネガ濃度（透過濃度）
  ↓ 【非線形②】
UV露光（光の透過）
  ↓ 【非線形③】
プラチナプリントの濃度（反射濃度）
```

**結果**: そのまま印刷すると、最終的なプリントが意図した階調にならない。

### 従来の対処法の限界

#### 従来法1: 経験と勘

- 何度もテストプリントを繰り返す
- 時間とコストがかかる
- 再現性が低い（条件が変わると最初から）
- 科学的根拠がない

#### 従来法2: 既存のEDN（Easy Digital Negative）

- Webツールで簡易的な補正
- カーブを手動で調整
- 測定データに基づかない
- 自分の環境に最適化されていない

### あなたが直面した課題

1. **プラチナプリントの再現性を高めたい**
   - 和紙、FO濃度、露光時間などの条件を固定しても、ネガ作成が不安定

2. **科学的なアプローチをしたい**
   - 測定データに基づいた補正
   - 再現可能な方法

3. **自分の環境に最適化したい**
   - EPSON PX-1V
   - ピクトリコOHP
   - 特定の和紙
   - 特定のFO配合

4. **深い黒（Dmax 2.1）を実現したい**
   - グレースケール印刷では限界（Dmax 1.5-1.7）
   - カラーモード印刷でDmax 2.1を達成していた
   - しかしこれを科学的に管理する方法がなかった

## なぜVersion 2が必要だったのか

### Version 1の構想（仮想）

もし従来の考え方でシステムを作るなら:
- 単純なガンマ補正
- プリンターの線形化のみ
- プラチナプリントの特性は考慮しない

**問題点**:
- プラチナプリントの非線形を無視
- 中間調が崩れる
- シャドウ/ハイライトが不自然

### Version 2の革新性

#### 1. 2段階測定方式

```
測定①: プリンター特性
  入力値 → ネガ濃度

測定②: プラチナプリント特性
  ネガ濃度 → プリント濃度
```

**これにより**:
- プリンター単体の特性を分離
- プラチナプリント単体の特性を分離
- それぞれの非線形を独立して理解
- 合成して全体の補正カーブを生成

#### 2. 逆カーブ生成

```
理想: 入力値 → プリント濃度 がリニア（直線）

実際: 入力値 → プリント濃度 が非線形

解決策: 逆カーブを適用
  入力値 → [補正LUT] → 実際の入力値

結果: 最終的なプリントがリニアに
```

#### 3. 中間調中心設計（γ=1.8）

従来の単純なγ=2.2ではなく、プラチナプリントに最適な:
- **γ=1.8**: 和紙の内部散乱特性に適合
- **中間調基準点**: 入力128 → ネガ濃度0.70
- **弱逆Sカーブ**: 自然なハイライト/シャドウの保護

#### 4. カラーモード対応（Dmax 2.1）

既存EDNはグレースケール前提だったが:
- RGBカラーモード印刷に対応
- Color Blockerとの統合
- Dmax 2.1の実現
- ランダムドット効果の獲得
- ノズル分散による安定性向上

## 技術的アプローチ

### 数学的基礎

#### ネガ濃度の理想カーブ（γ=1.8）

```python
D(x) = Dmax - (Dmax - Dmin) * (x / 255)^γ
```

ここで:
- `D(x)`: 入力値xに対する理想ネガ濃度
- `Dmax`: 最大濃度 (2.10)
- `Dmin`: 最小濃度 (0.10)
- `γ`: ガンマ値 (1.8)
- `x`: 入力値 (0-255)

#### 逆カーブの生成

```python
# 1. 実測カーブを取得
actual_curve: 入力値 → 実測ネガ濃度

# 2. 理想カーブを計算
ideal_curve: 入力値 → 理想ネガ濃度

# 3. 逆カーブを生成
for each input_value:
    ideal_density = ideal_curve(input_value)
    # この濃度を実際に出すために必要な入力値を逆算
    corrected_value = inverse(actual_curve, ideal_density)
    LUT[input_value] = corrected_value
```

#### プラチナプリント特性の統合

```python
# 合成カーブ
combined: 入力値 → プリント濃度
combined(x) = platinum_curve(printer_curve(x))

# 理想: リニア
ideal(x) = linear_function(x)

# 補正カーブ
correction(x) = inverse(combined, ideal(x))
```

## プログラムが行うこと

### 入力

**測定データCSV**:
```csv
input,negative_density,print_density
0,2.10,0.05
8,2.05,0.08
...
128,0.70,0.85
...
255,0.10,2.10
```

33ステップの:
- 入力値（0-255）
- ネガ濃度（透過濃度）
- プリント濃度（反射濃度）

### 処理

#### ステップ1: データ読み込み

```python
data = MeasurementData()
data.load_from_csv("measurement.csv")
# → 検証: 必須列、欠損値、濃度範囲
```

#### ステップ2: カーブ解析

```python
analyzer = CurveAnalyzer(data)

# プリンター特性カーブ
printer_curve = analyzer.printer_curve
# 入力値 → ネガ濃度

# プラチナ特性カーブ
platinum_curve = analyzer.platinum_curve
# ネガ濃度 → プリント濃度

# 合成カーブ
combined_curve = analyzer.combined_curve
# 入力値 → プリント濃度
```

#### ステップ3: 逆カーブ生成

```python
inverse_gen = InverseCurveGenerator(analyzer)

# 理想ネガ濃度を計算
ideal_density = gamma_curve(input, gamma=1.8)

# 実際にその濃度を出すための入力値を逆算
corrected_input = inverse_function(printer_curve, ideal_density)

# 弱逆Sカーブ適用
final_curve = apply_weak_inverse_s(corrected_input)
```

#### ステップ4: LUT書き出し

```python
exporter = LUTExporter(inverse_gen, "output_name")

# CUBE LUT（Photoshop/Lightroom用）
exporter.export_cube()

# カーブCSV（33点）
exporter.export_curve_csv()

# レポート（テキスト）
exporter.export_report()
```

#### ステップ5: グラフ生成

```python
fig = analyzer.plot_analysis()
# 3パネルグラフ:
#   左: 入力 → ネガ濃度
#   中: ネガ濃度 → プリント濃度
#   右: 入力 → プリント濃度（合成+補正）
```

プラチナプリントにする際、大きな問題があります:

```
デジタル画像 → プリンター → ネガ → UV露光 → プラチナプリント
```

この各段階で非線形（歪み）が発生し、
最終的なプリントが意図した階調にならないのです。

### 従来の対処法

1. **経験と勘**
   - 何度もテストプリントを繰り返す
   - 時間とコストがかかる
   - 条件が変わると最初から

2. **既存ツール**
   - 簡易的な補正
   - 自分の環境に最適化されていない
   - 測定データに基づかない

### Precision EDN v2の革新

このシステムは、**測定データに基づいて**
あなたの環境に完全に最適化された補正カーブを
**自動生成**します。

#### 手順

1. **測定用チャート作成**
   - Photoshopスクリプトで33ステップチャート生成
   - RGB変換 → 反転 → Color Blocker → 印刷

2. **測定① プリンター特性**
   - 濃度計でネガ濃度を測定
   - プリンターの非線形を把握

3. **測定② プラチナプリント特性**
   - 同じネガでプリント作成
   - プリント濃度を測定
   - プラチナの非線形を把握

4. **LUT生成**
   - 測定データをCSVに入力
   - プログラムが自動解析
   - 数秒でLUT完成

5. **本番制作**
   - PhotoshopでLUTを適用
   - 反転 → Color Blocker → 印刷
   - プラチナプリント完成

#### 技術的特徴

- **2段階測定**: プリンターとプラチナを分離して理解
- **逆カーブ生成**: 非線形を数学的に補正
- **γ=1.8設計**: 和紙の特性に最適化
- **カラーモード対応**: Dmax 2.1の実現
- **Streamlit GUI**: ビジュアルなインターフェース

#### 得られる価値

- ✓ **再現性**: 同じ条件なら何度でも同じ結果
- ✓ **効率**: テストの繰り返しが不要
- ✓ **品質**: 深い黒（Dmax 2.1）、豊かな階調
- ✓ **科学性**: 測定データに基づく客観的手法
- ✓ **柔軟性**: 環境が変わっても再測定で対応

### まとめ

Precision EDN v2は、
プラチナプリント制作を「芸術と勘」から
「芸術と科学」に変えるツールです。

技術の部分を科学的に管理することで、
あなたは創作に集中できます。
