## Precision EDN Version 1

### 開発の背景と目的

Precision EDN Version 1は、プラチナプリント用デジタルネガの科学的アプローチの**第一歩**として開発されました。

**主な目的**:
- 従来の「経験と勘」から脱却
- **測定データに基づく再現可能な方法**の確立
- プラチナプリントの階調を科学的に管理

### Version 1の設計思想

**核心哲学**: 「完全リニア」

```
入力値 → プリント濃度 が完全な直線になるように補正
```

**設計理由**:
1. **シンプルさ**: 理解しやすい、実装しやすい
2. **数学的明快さ**: 直線=理想という明確な目標
3. **測定の容易さ**: リニアからのズレを測定すればOK
4. **検証の容易さ**: 結果が直線か確認するだけ

### Version 1の技術仕様

| 項目 | 仕様 | 根拠 |
|------|------|------|
| **ネガγ** | 2.0 | 従来のγカーブ理論 |
| **カーブ形状** | 完全γカーブ | シンプルな数学モデル |
| **ネガDmin** | 0.10 | 和紙での光漏れ防止 |
| **ネガDmax** | 1.60 | グレースケール印刷の限界 |
| **濃度レンジ** | 1.50 | Dmax - Dmin |
| **測定方法** | 1段階 | 入力→プリント濃度の直接測定 |
| **印刷モード** | グレースケール | Kインクのみ使用 |
| **bit深度** | 16bit | 階調破壊防止 |
| **制御点数** | 32点（推定） | 十分な階調制御 |

### Version 1のアプローチ

#### 測定方法: 1段階測定

```
入力値(0-255)
  ↓
【ブラックボックス】
  - プリンター印刷
  - ネガ濃度
  - UV露光
  - プラチナ反応
  ↓
プリント濃度
```

**考え方**:
- プロセス全体を一括で測定
- 結果（プリント濃度）だけを見る
- 逆カーブで補正

**利点**:
- ✓ シンプル
- ✓ 測定が1回で済む
- ✓ 理解しやすい
- ✓ 実装が容易

**欠点**:
- △ 内部が見えない（ブラックボックス）
- △ 問題の原因特定が困難
- △ 微調整が難しい

#### カーブ設計: 完全γカーブ

**数式**:
```
ネガ濃度 = Dmax - (Dmax - Dmin) × (入力値/255)^γ

Version 1パラメータ:
  Dmax = 1.60
  Dmin = 0.10
  γ = 2.0

D(x) = 1.60 - 1.50 × (x/255)^2.0
```

**特徴**:
- 完全な数学的γカーブ
- 補正なし、追加調整なし
- シンプルで美しい

#### 印刷方式: グレースケール

```
使用インク: K（黒）のみ
Dmax: 1.5〜1.7（実測1.6達成）
ドットパターン: 規則的
ノズル使用: Kノズルのみ
```

### Version 1の成果

**達成できたこと**:
- ✓ **ネガDmin 0.1の実現**: 和紙での光漏れ防止成功
- ✓ **ネガDmax 1.6の実現**: グレースケールで十分な濃度
- ✓ **プラチナプリントDmax 2.1の達成**: FO高濃度で深い黒
- ✓ **プリント露光レンジ 6.6EVの確認**: 測定による実証
- ✓ **測定に基づく科学的アプローチの実証**: 再現可能な方法の確立
- ✓ **16bit処理の有効性**: 十分な階調が得られることを確認

**重要な発見**:
1. グレースケール印刷でもDmax 1.6は達成可能
2. プラチナプリントでDmax 2.1は実現可能（FO高濃度）
3. 測定データからカーブ設計が実際に機能する
4. 16bit処理で階調破壊を防げる

**Version 1の意義**:
- プラチナプリント×デジタルネガの科学的アプローチの**実証**
- 測定→補正→検証のサイクルの確立
- Version 2への基盤構築

### Version 1の課題

実際に使用して判明した改善点:

#### 1. γ=2.0は強すぎる可能性

**問題**:
- 技術的総括の推奨値: 1.7〜1.9
- Version 1の2.0はこの範囲を超える
- 結果: コントラストが強すぎる傾向

**原因**:
- 従来のγ理論を踏襲
- 和紙+蛍光UV環境の特性を未考慮

#### 2. 完全γカーブは視覚的に眠く見える

**問題**:
- 数学的にはリニアでも、人間の視覚には最適でない
- 特に中間調が平坦に感じる
- シャドウ・ハイライトに抑揚がない

**原因**:
- 完全リニアを目指した結果
- 人間の視覚特性を考慮していない

#### 3. 環境特性が未考慮

**考慮されていない要素**:
- **蛍光UV散乱**: 365nm付近の光の挙動
- **和紙内部散乱**: 長繊維による光拡散
- **Optical dot gain**: インク粒子による光散乱

**影響**:
- 中間調の実際の挙動が理論と異なる
- ハイライトが予想より持ち上がる

#### 4. 測定の分離不足

**問題**:
- プリンターとプラチナプリントを一括測定
- どちらの非線形が支配的か不明確
- 問題の切り分けが困難

**実例**:
```
中間調が崩れた場合:
  - プリンターの問題？
  - プラチナの問題？
  - 両方？
  → 判別不可能
```

#### 5. グレースケール印刷の限界

**制約**:
- Kインクのみ → Dmax 1.6が限界
- より深い黒（Dmax 2.0+）は物理的に不可能
- インク量を増やすと滲み、詰まりのリスク

## Precision EDN Version 2

### 開発の動機

Version 1の成果を認めつつ、以下の**進化**を目指す:

1. **視覚的な自然さ**: 数学的リニアより、人間の視覚に最適化
2. **環境への完全適合**: 和紙+蛍光UV+カラーモードの特性を反映
3. **中間調の豊かさ**: プラチナプリントの本質（階調の写真）を最大化
4. **測定の精密化**: プリンターとプラチナを分離して理解
5. **深い黒の実現**: Dmax 2.1を科学的に管理

### Version 2の設計思想

**核心哲学**: 「中間調基準 + 弱逆Sカーブ」

```
完全リニアではなく、
人間の視覚と和紙の特性に最適化された階調
```

**設計理由**:
1. **プラチナプリントの本質**: 「階調の写真」
2. **中間調が最重要**: Zone V-VI が作品の核心
3. **和紙の特性活用**: 内部散乱が中間調を豊かにする
4. **蛍光UVの特性活用**: 散乱がハイライトを自然に持ち上げる
5. **視覚の非線形**: 人間の目はリニアでない

### Version 2の技術仕様

| 項目 | 仕様 | 根拠 |
|------|------|------|
| **ネガγ** | 1.8 | 和紙+蛍光UV推奨値(1.7〜1.9)の中央 |
| **カーブ形状** | 弱い逆Sカーブ | 視覚最適化 + optical dot gain補正 |
| **ネガDmin** | 0.10 | Version 1と同じ（実証済み） |
| **ネガDmax** | 2.10 | **カラーモード対応** |
| **濃度レンジ** | 2.00 | Dmax - Dmin |
| **測定方法** | **2段階分離** | プリンター/プリント特性を分離 |
| **印刷モード** | **RGB + Color Blocker** | 8色インク、Dmax 2.1実現 |
| **中間調基準** | **入力128 → 濃度0.70** | 厳密な基準点設定 |
| **bit深度** | 16bit | Version 1と同じ |
| **制御点数** | 33点（最適配分） | 中間調に重点配分 |

### Version 2のアプローチ

#### 測定方法: 2段階分離測定

```
【測定①: プリンター特性】
入力値(0-255)
  ↓
プリンター印刷
  ↓
ネガ濃度（測定）

【測定②: プラチナ特性】
ネガ濃度
  ↓
UV露光
  ↓
プラチナ反応
  ↓
プリント濃度（測定）
```

**考え方**:
- プロセスを2段階に分離
- 各段階を独立して理解
- 両方を統合して補正カーブを設計

**利点**:
- ✓ プリンターの特性が見える
- ✓ プラチナの特性が見える
- ✓ 問題の原因特定が容易
- ✓ 微調整が可能

**欠点**:
- △ 測定が2回必要
- △ やや複雑

#### カーブ設計: γ + 弱逆Sカーブ

**基本γカーブ**:
```
D_base(x) = Dmax - (Dmax - Dmin) × (x/255)^γ

Version 2パラメータ:
  Dmax = 2.10
  Dmin = 0.10
  γ = 1.8

D_base(x) = 2.10 - 2.00 × (x/255)^1.8
```

**弱逆S補正**:
```
S(x) = 逆S補正関数

D_final(x) = D_base(x) + S(x)

S(x)の定義:
  - シャドウ(0-64): +0.02〜+0.05 (持ち上げ)
  - 中間調(64-192): ±0 (直線維持)
  - ハイライト(192-255): -0.02〜-0.05 (圧縮)
```

**特徴**:
- **シャドウ**: わずかに持ち上げてディテール保持
- **中間調**: 直線を維持して階調豊か
- **ハイライト**: 軽く圧縮して自然に

**効果**:
- 視覚的に自然
- Optical dot gain補正
- 中間調の豊かさ維持

#### 印刷方式: RGB + Color Blocker

```
使用インク: C, M, Y, K + LC, LM, LK, LLK（8色）
Dmax: 2.1
ドットパターン: ランダム（ディザリング）
ノズル使用: 全ノズル（分散）
Color Blocker: 余白を黒で覆って光漏れ防止
```

**RGB + Color Blockerの革新**:

これはVersion 2開発中の**大発見**でした。

**発見の経緯**:
1. 当初はVersion 1と同じグレースケールを想定
2. ユーザーが「カラーモードでDmax 2.1達成」と報告
3. 検証した結果、カラーモードに多数の利点が判明
4. RGB + Color BlockerワークフローをVersion 2に統合

**カラーモードの利点**:

**1. Dmax 2.1の実現**
```
グレースケール: Kのみ → Dmax 1.5-1.7
カラーモード: 8色重ね → Dmax 2.1
```

**2. ランダムドット効果**
```
グレースケール: 規則的ドット → 粒状感
カラーモード: ランダムドット → 滑らか
```

**3. ノズル分散**
```
グレースケール: Kノズルのみ → 詰まりリスク集中
カラーモード: 全ノズル使用 → リスク分散
```

**4. R=G=B でニュートラルグレー**
```
RGB画像で R=G=B を維持
  ↓
カラーモード印刷でもグレー
  ↓
色かぶりなし
```

**Color Blocker統合**:
```
画像周囲を黒枠で覆う
  ↓
余白からの光漏れを防止
  ↓
コンタクトプリント時の精度向上
```

これはEDN（Easy Digital Negative）の手法と互換性があり、Version 2はこれを科学的に管理します。

## アプローチの違い

### 設計哲学の比較

| 観点 | Version 1 | Version 2 |
|------|-----------|-----------|
| **目標** | 完全リニア | 視覚最適化 |
| **基準** | 全体的な直線性 | 中間調基準（入力128→濃度0.70） |
| **カーブ** | シンプルなγカーブ | γ + 弱逆S |
| **測定** | 結果重視（1段階） | プロセス重視（2段階） |
| **哲学** | 「測定して補正」 | 「理解して設計」 |
| **印刷** | グレースケール | RGB + Color Blocker |
| **Dmax** | 1.6 | 2.1 |

### 測定アプローチの違い

**Version 1: 結果主義**
```
入力値 → [ブラックボックス] → プリント濃度

ブラックボックスの入出力を測定
  ↓
逆カーブで補正
```

**メリット**:
- シンプル
- 1回の測定

**デメリット**:
- 内部が見えない
- 原因不明の問題発生時に困る

---

**Version 2: プロセス主義**
```
入力値 → [プリンター] → ネガ濃度
            ↓ 測定①
         プリンター特性を理解

ネガ濃度 → [プラチナ] → プリント濃度
            ↓ 測定②
         プラチナ特性を理解

両方を理解した上で、統合的に補正カーブを設計
```

**メリット**:
- 各プロセスが理解できる
- 問題の原因特定が容易
- 精密な微調整が可能

**デメリット**:
- 測定が2回必要
- やや複雑

```
理想 = 視覚的に自然

```

| **ノズル** | Kのみ使用 | 全ノズル使用（分散） |

## 技術的詳細比較

### γ値の違い

**Version 1: γ=2.0**
```
D(x) = 1.60 - 1.50 × (x/255)^2.0

中間調:
  入力128 → (128/255)^2.0 = 0.251
  濃度 = 1.60 - 1.50×0.251 = 1.22
```

**Version 2: γ=1.8**
```
D(x) = 2.10 - 2.00 × (x/255)^1.8

中間調:
  入力128 → (128/255)^1.8 = 0.296
  濃度 = 2.10 - 2.00×0.296 = 1.51

  しかし、中間調基準設計により:
  入力128 → 濃度0.70（厳密に固定）
```

**違い**:
- Version 1: γから自動的に決まる
- Version 2: 中間調を先に決めて、カーブを設計

### 中間調配置の違い

**Version 1**: 中間調は結果的に決まる
```
入力128の位置は、γ=2.0から自動計算
明確な基準点なし
```

**Version 2**: 中間調を基準点として厳密に固定
```
入力128 → ネガ濃度0.70
これを不動のアンカーポイントとする
前後のトーンはこの基準点から配置
```

**効果**:
- Version 2は中間調の再現性が高い
- Zone V-VIの配置が保証される

**状況**:
- プリンターの制限
- ワークフローの制約

**対応**:
- Version 1のグレースケールアプローチ

#### 3. 迅速なキャリブレーションが必要な時

**状況**:
- 時間的制約
- 簡易キャリブレーション
- テスト目的

**対応**:
- 1回の測定で完了
- 手早く結果確認

### Version 2が優れる場面

#### 1. 最高品質を求める時

**用途**:
- 本番作品制作
- 展示用プリント
- コレクション制作
- 商業制作

**理由**:
- Dmax 2.1の深い黒
- 視覚的に最適化された階調
- 中間調の豊かさ

#### 2. 詳細な制御が必要な時

**状況**:
- 中間調の微調整
- シャドウディテール重視
- ハイライトの自然さ重視

**対応**:
- 2段階測定による精密理解
- 弱逆Sカーブによる微調整
- 中間調基準設計

#### 3. 問題解決が必要な時

**状況**:
- トラブルシューティング
- 階調の問題特定
- プリンター特性の理解
- プラチナ特性の理解

**対応**:
- 2段階測定で原因分離
- グラフで視覚的に確認
- 各段階を独立して調整

#### 4. 再現性が最重要な時

**状況**:
- シリーズ作品制作
- 長期プロジェクト
- 複数作品の統一感
- 商業制作

**対応**:
- 厳密な中間調基準（入力128→0.70）
- 測定データに基づく科学的管理
- LUTによる完全再現

## まとめ: 2つのバージョンの関係

### 進化の系譜

```
Version 1: 「測定して補正」
  ↓
  成果と課題の発見
  ↓
Version 2: 「理解して設計」
  ↓
  より高度な制御
  ↓
Version 3: 「AIで最適化」（未来）
```

### 相補的な関係

Version 1とVersion 2は、**対立ではなく補完**の関係:

- **Version 1**: シンプルさと理解しやすさ
- **Version 2**: 精密さと品質の追求

どちらも価値があり、状況に応じて使い分けることが重要。

### 共通の核心

両バージョンに共通する核心思想:

```
「測定できるものは、制御できる」

プラチナプリント制作を、
芸術と勘から、
芸術と科学へ。
```
