## 🎯 このプログラムについて

**Precision EDN v2** は、プラチナプリント用のデジタルネガを科学的にキャリブレーションするツールです。

### 何ができるか？

1. **測定データの解析**
   - プリンターの特性を測定
   - プラチナプリントの特性を測定
   - 2つを統合して最適なカーブを生成

2. **自動補正カーブ生成**
   - リニア化（線形化）カーブ
   - カスタムγカーブ（γ=1.8推奨）
   - 逆Sカーブ対応

3. **Photoshop用LUT出力**
   - CUBE LUT形式
   - Photoshopカーブ形式
   - グラデーションマップ用RGB値

### 対象ユーザー

- プラチナプリント/パラディウムプリントを行う写真家
- デジタルネガを使用するオルタナティブプロセス実践者
- 精密な濃度コントロールが必要な方

## 🔧 よくある質問

### Q1. `requirements.txt`って何？

**A:** Pythonで書かれたプログラムが必要とする外部ライブラリのリストです。

これがないと、他の人の環境で動きません！

### Q2. なぜ`requirements.txt`が必要？

**A:** Pythonプログラムは標準ライブラリ以外に、追加のライブラリ（パッケージ）を使用することがあります。

例えば、このプログラムは：
- `numpy` で数値計算
- `streamlit` でWebインターフェース
- `matplotlib` でグラフ描画

これらは**別途インストールが必要**です。

`requirements.txt`があれば、受け取った人は：
```bash
pip install -r requirements.txt
```
だけで、必要な全てのライブラリを一括インストールできます。

### Q3. `pip`って何？

**A:** Pythonのパッケージマネージャー（ライブラリをインストールするツール）です。

Pythonをインストールすると、通常は`pip`も一緒にインストールされます。

### Q4. なぜPythonファイルだけではダメなの？

**A:** Pythonファイル（`.py`）だけでは、そのコードが使っている外部ライブラリの情報がありません。

受け取った人の環境に必要なライブラリがインストールされていないと、実行時にエラーになります：

```
ModuleNotFoundError: No module named 'streamlit'
```

`requirements.txt`があれば、このエラーを防げます。

### Q5. 仮想環境って必要？

**A:** 必須ではありませんが、**強く推奨**します。

仮想環境を使うと：
- システムのPython環境を汚さない
- プロジェクトごとに異なるバージョンのライブラリを使える
- アンインストールが簡単

### 方法1: GitHub（推奨）

1. GitHubにリポジトリを作成
2. 全ファイルをプッシュ
3. URLを共有

- バージョン管理ができる
- 問題報告（Issue）を受け付けられる
- 更新が簡単

### 方法2: ZIP圧縮

```bash

zip -r precision_edn_v2.zip precision_edn_v2 \
  -x "*/__pycache__/*" \
  -x "*/.DS_Store" \
  -x "*/output/*"
```

**メリット:**
- シンプル
- Gitを知らない人にも配布しやすい

**デメリット:**
- バージョン管理できない
- 更新が面倒
