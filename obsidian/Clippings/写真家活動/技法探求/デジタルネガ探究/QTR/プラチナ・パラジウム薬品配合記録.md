## 🔬 濃度変更と化学的考察

### Na2PtCl6 20% → 5% 変更について

**重要な発見:**
- Na2PtCl6 5%は20%の **1/4の濃度**
- 同じPt金属量を得るには **4倍の量** が必要
- 1.020cc (5%) ≈ 0.255cc (20%) → **コスト削減効果なし**

**結論:**
```
✗ Na2 20% → 5%への変更のみではコスト削減不可
✓ Pt金属量そのものを削減することで真のコスト削減が可能
```

### K2PtCl4への変更は推奨しない理由

```
1. 化学的に全く異なる
   Pt(IV) → Pt(II)は根本的な変更

2. コントラスト大幅低下
   硬調 → 軟調への変化

3. QTRカーブが使えない
   v14カーブはNa2PtCl6で最適化済み

4. 露光時間の大幅変更
   すべてのパラメータ再調整が必要

5. 経年変化は同等
   どちらも500年以上の保存性

6. デジタルワークフローに不適
   自家調製は濃度管理が困難
```

### 歴史的配合の科学的解釈

**オーソドックスな配合（19世紀）:**
```
K2PtCl4 (Pt(II)) 5g / 25ml水
→ 軟調な階調の伝統的プラチナプリント
```

**フィルム時代のコントラストブースト:**
```
「Na2追加」= Na2PtCl6 (Pt(IV)) 追加
→ 第二白金でコントラスト増強
→ 科学的に正しい手法
```

**現代の標準（デジタルネガ時代）:**
```
Na2PtCl6 (Pt(IV)) 20%
→ 硬調な階調
→ デジタルネガに最適化
→ QTRカーブによる精密制御
```

## 🛡️ 経年変化（アーカイバル性）

### 500年保存の科学的根拠

**どちらの薬品も最終的に金属Ptに還元:**
```
Na2PtCl6 (Pt(IV)) → Pt金属
K2PtCl4 (Pt(II))  → Pt金属

最終形態: 両方とも「Pt金属」
→ 対イオン(Na+, K+)は洗浄で除去
→ 経年変化は同等（500年以上）
```

### 経年変化に影響する本当の要因

```
1. 紙の品質（最重要）
   - 中性紙・コットン紙使用

2. 洗浄・クリアリングの徹底
   - Fe残留の防止
   - 未反応Pt塩の除去

3. 保存環境
   - 湿度管理
   - 大気汚染物質の回避

4. 使用薬品の種類
   - Na2PtCl6 vs K2PtCl4: 影響は無視できる
```

**結論:**
```
✓ Na2PtCl6もK2PtCl4も経年変化は同等
✓ どちらも500年以上の保存性
✓ 適切な処理と保存環境が重要
```
