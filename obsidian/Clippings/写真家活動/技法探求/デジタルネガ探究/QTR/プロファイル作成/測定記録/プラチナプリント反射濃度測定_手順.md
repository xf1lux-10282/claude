# プラチナプリント反射濃度測定 - 手順書

## 目的

v7-NAとv8-HEのコントラスト差を**実際のプラチナプリントの反射濃度**で比較する。

---

## 準備するもの

### 1. テストチャート
- **グラデーションステップチャート**
- Input値: 0, 32, 64, 96, 128, 160, 192, 224, 232, 240, 248, 255
- 各ステップを明確に区切る

### 2. プリント条件
```
用紙: Pictorico OHP (同じロット)
感光剤: プラチナ/パラジウム (同じ調合)
露光: 同じUVライト、同じ時間
現像: 同じ温度、同じ時間
定着: 同じ条件
```

### 3. 測定機器
- 反射濃度計 (X-Rite i1Pro など)
- または分光測色計

---

## プリント手順

### Step 1: グラデーションチャートの作成

Photoshopで以下のチャートを作成:

```
+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+
|  0  | 32  | 64  | 96  |128  |160  |192  |224  |232  |240  |248  |255  |
+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+-----+
```

各ステップサイズ: 2cm × 2cm (測定しやすいサイズ)

### Step 2: デジタルネガ出力

**v7-NA用**:
1. QTRでPX1V-PtPd-xf1-v7-NAを選択
2. グラデーションチャートを出力
3. ファイル名: `gradient_v7-NA_YYYYMMDD.tif`

**v8-HE用**:
1. QTRでPX1V-PtPd-xf1-v8-HEを選択
2. 同じグラデーションチャートを出力
3. ファイル名: `gradient_v8-HE_YYYYMMDD.tif`

### Step 3: プラチナプリント

**重要**: 2枚のプリントを**完全に同じ条件**で処理する

1. 同じ日に感光剤を調合
2. 同じコーティング回数
3. 同じ露光時間
4. 同じ現像・定着・水洗時間
5. 同じ乾燥方法

---

## 測定手順

### Step 1: 測定準備

1. プリントを完全に乾燥させる (24時間以上)
2. 測定環境を整える (温度20-25℃、湿度40-60%)
3. 濃度計をキャリブレーション

**期待される結果**:

## 予想される結果

### Case 1: v8-HEのコントラストが実際に高い場合

```
Dmax:
v7-NA: 1.20D
v8-HE: 1.30D (+0.10D)
→ v8-HE-LC版を作成（K_maxを下げる）
```

### Case 2: デジネガ濃度とプリント濃度が非線形の場合

```
ネガ濃度差 ≠ プリント濃度差
→ 露光時間の調整が必要
→ または、カーブの全体的な再調整
```

### Case 3: ハイライト圧縮が効いている場合

```
Input 232-255:
v7-NA: 濃度差 0.15D (急激)
v8-HE: 濃度差 0.08D (緩やか) ✓
→ v8-HEの設計意図通り
→ ただしDmaxは調整が必要
```

---

## 次のステップ

測定結果に応じて:

1. **v8-HEのDmaxが高すぎる場合**
   → v8-HE-LC版を作成 (K_max調整)

2. **ハイライト圧縮が不十分な場合**
   → ハイライト域のカーブを再調整

3. **v7-NAとv8-HEの違いが見えない場合**
   → 露光時間を再検討

---

## 重要な注意点

### デジタルネガ濃度 ≠ プラチナプリント反射濃度

プラチナプリントでは:
- ネガの濃度範囲: 0.08D - 1.15D (透過濃度)
- プリントの濃度範囲: ? - ?D (反射濃度)

**この変換が非線形の可能性がある**

そのため:
1. まず実測する
2. 理論値と比較する
3. 必要なら露光カーブで補正する
