## 🔍 重要な発見

### 1. 紙白到達の問題（最重要）

**実測**: Input 255 = **0.09D**
**目標**: 0.07D（v7-NA基準）
**差分**: +0.02D（紙白未到達）

**原因分析**:
1. **K_max設定が不足**
   - v12: K_max = 33800
   - v7-NA: K_max = 33051（紙白到達実績）
   - K_maxを上げたにもかかわらず紙白未到達

2. **露光時間の可能性**
   - 想定6分で測定したか確認が必要
   - v7-NAは7分で0.07D達成
   - v12は6分では濃度過剰の可能性

**対策**:
- **対策A**: 露光時間を6分30秒〜6分45秒に延長（+10-12.5%）
- **対策B**: K_maxを34500程度に増加（+2.1%）
- **推奨**: まず対策Aを試行（露光時間調整が簡便）

2. **K_max増加の副作用**

3. **Zone境界変更の影響**
   - Zone 2終点: 232 → 240（+8ポイント）

## 🎯 v12の設計意図と実測の乖離

### 設計意図（理論値）

```
v12設計: ミッドトーン勾配 117.4 K/step（v9-SM比 -5.8%）
→ ミッドトーンを「緩やか」にして「薄く」する意図
```

### 実測結果

```
v12実測: ミッドトーン濃度 v7-NAより平均 +0.056D 濃い
→ 設計と逆に「濃く」なっている
```

### 乖離の理由

1. **K_max増加の影響**
   - K_maxを上げると、同じ露光時間では全体的に濃くなる
   - v7-NA: 33051 → v12: 33800（+2.3%増加）

2. **露光時間の最適化不足**
   - K_max増加に対応した露光時間の延長が不十分
   - 想定: 6分 → 実際の最適値: 6分45秒程度と推測

3. **Zone 2範囲拡大の影響**
   - Zone 2を232→240に延長したことで、より多くのInput値が緩やかな勾配域に
   - しかし露光不足により、この恩恵が活かされていない

## 📝 今後のカーブ開発への教訓

### 教訓1: K_max増加には露光時間延長が必須

**発見**:
- K_max +2.3%（33051 → 33800）では露光時間-14%（7分 → 6分）は急激すぎる
- K_max増加分に対して、露光時間は比例的に延長が必要

**新しい原則**:
```
K_max +2% → 露光時間 +3-5% 程度必要
（従来: K_max +10% → 露光時間 -15% は極端な例）
```

---

### 教訓2: 設計値と実測値の乖離要因

**要因**:
1. K_maxの絶対値が全体濃度に与える影響
2. 露光時間とK_maxの相互作用
3. Zone境界変更の副次的影響

**対策**:
- K_maxを変更する場合は、まず露光時間を固定して評価
- Zone境界変更は慎重に（影響が読みにくい）

---

### 教訓3: v7-NAの基準値の重要性

**再確認事項**:
- v7-NA: K_max=33051, 露光7分, 紙白0.07D
- この組み合わせは**実測で確立された基準点**
- 新カーブ開発時はこの基準から段階的に変更すべき
