### 設計方針

V7をベースに、段階的に濃度を削減する軟調化カーブ

- **Input 0-79**: V7のまま
- **Input 80-120**: 徐々に削減を増やす（0 → -1500 Quad）

- シンプルな設計でバンディングリスクを最小化

- 最大削減量: -1500 Quad = -0.0464D（Input 121-220）
- 段階的削減: Input 80-120で0から-1500 Quadへ線形変化
- 線形補間: Input 221-248でV7に戻る

**変換式:**
```
濃度 (D) = Quad値 × 1.73 / 55891
```

- Quad 1500 = 0.0464D
