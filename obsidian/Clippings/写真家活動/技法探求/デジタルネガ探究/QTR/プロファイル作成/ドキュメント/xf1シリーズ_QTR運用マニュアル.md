## 🎯 運用方針

### xf1シリーズの基本思想

**原則**:
> **用途に応じて適切なカーブを選択し、階調調整はPhotoshop側で行う**

**理由**:
1. 各カーブが特定の用途に最適化されている
2. 最大の階調情報を保持 = 最大の後処理柔軟性
3. Photoshopでコントラスト調整可能
4. 用途別の運用でシンプルな管理

### 現行カーブの位置づけ

**2026年4月4日時点の運用カーブ**:

| カーブ | 主な用途 | 特徴 | 露光時間 |
|--------|---------|------|---------|
| **xf1-v14** | 洋紙用、芸術的表現 | 最も豊かな階調（Zone 2: 115.0 K/step） | 6分45秒 |
| **v14-washi-v6** | 和紙用、ハイライト重視 | ハイライト拡張（240-255: 5000 K範囲） | 6分45秒 |
| **xf1-v28** | 紙白確実到達、K単色 | v26ベース + K[255]=40000 | **6分05秒** |
| **v28-multiink** | ディザリング解消、滑らかな階調 | v28 + GY/LGY 3%混合 | **6分05秒** |

## 🗂️ カーブ概要と使い分け

### xf1-v14（洋紙用、芸術的表現）⭐

**主要仕様**:

| 項目 | 値 |
|-----|-----|
| **K_max** | 33800 |
| **Zone 2勾配** | 115.0 K/step（最も豊かな階調） |
| **露光時間** | 6分45秒 |
| **実用域** | Input 0-255（全範囲） |
| **バンディング** | 完全消失（標準偏差0.000） |

**使用シーン**:
- コットン紙（Arches Platine 310gsm等）での通常プリント
- 最も豊かな階調表現が必要な作品
- Photoshopで柔軟にコントラスト調整したい場合

---

### v14-washi-v6（和紙用、ハイライト拡張）

**主要仕様**:

| 項目 | 値 |
|-----|-----|
| **K_max** | 32000 |
| **ハイライト範囲** | Input 240-255: 5000 K範囲（v14の4400 Kより1.14倍拡張） |
| **露光時間** | 6分45秒 |
| **設計思想** | 和紙のハイライト表現不足に対応 |

**使用シーン**:
- 和紙（楮、三椏、雁皮等）での繊細なハイライト表現
- Input 240-255の階調を確実に見せたい作品
- 洋紙でもハイライト重視の作品

**開発背景**:
- v14では和紙のInput 240-255が紙白になる問題
- ネガ思考: プリントで階調豊か → ネガで濃度差大きく → K値範囲拡張
- v1-v3の失敗（プリント思考）を経て、v5-v6で成功（ネガ思考）

---

### xf1-v28（紙白確実到達、K単色）

**主要仕様**:

| 項目 | 値 |
|-----|-----|
| **K_max** | 40000（v26の33500から+19.4%） |
| **Input 0-254** | v26と完全に同一 |
| **Input 255** | 40000（紙白確実到達） |
| **露光時間** | **6分05秒**（v26の6分45秒から-10%） |

**使用シーン**:
- 紙白（濃度0.09以下）の確実な到達が必要な作品
- シャドウバランス重視（v26のバランスを保持）
- K単色での純粋なモノトーン表現

**開発背景**:
- v27失敗: Input 8下げ（2520→2080）→ シャドウ悪化（K値↓ → プリント濃↑）
- v28: シンプル戦略 = v26の0-254保持 + K[255]のみ40000
- 測定誤差の理解: ±0.01-0.02はハンドコーティング誤差範囲

---

### v28-multiink（ディザリング解消、滑らかな階調）

**主要仕様**:

| 項目 | 値 |
|-----|-----|
| **K値** | v28と完全に同一 |
| **GY（LKチャンネル）** | K値の3% |
| **LGY（LLKチャンネル）** | K値の3% |
| **露光時間** | **6分05秒** |

**インク配分例**:

| Input | K値 | GY (3%) | LGY (3%) |
|-------|-----|---------|----------|
| 8 | 2520 | 76 | 76 |
| 128 | 16380 | 491 | 491 |
| 255 | 40000 | 1200 | 1200 |

**使用シーン**:
- ディザリングパターン（規則的なドット）が気になる作品
- より滑らかな階調表現が必要な作品
- わずかに暖色傾向が欲しい作品（グレーインク混合による）

**開発背景**:
- K単色（v28）: プリンターのディザリングパターンが目立つ
- マルチインク: K + GY 3% + LGY 3% = ドット配置ランダム化
- 3%の根拠: シャドウ域（Input 8: K=2520 → GY=76）で十分なディザリング効果

## 🖼️ デジタルネガ作成ワークフロー

### 1. Photoshopでの画像準備

#### Step 1-1: RAW現像
```
1. Lightroom/Photoshop Camera RAWでRAW現像
2. 基本的な露出・コントラスト調整
3. 色温度・トーン調整
```

#### Step 1-2: グレースケール変換
```
1. Photoshopで開く
2. 「イメージ > モード > グレースケール」
3. カラー情報を破棄
```

#### Step 1-3: 階調調整
```
1. レベル補正: ヒストグラムを確認、黒点・白点設定
2. トーンカーブ: 必要に応じてS字カーブ等でコントラスト調整
3. 覆い焼き・焼き込み: 局所的な調整
```

**カーブ別のポイント**:
- **v14**: 最も豊かな階調、積極的にコントラスト調整可能
- **v14-washi-v6**: ハイライト重視、Input 240-255を活用
- **v28/v28-multiink**: 紙白到達重視、シャドウバランス良好

#### Step 1-4: 解像度設定
```
1. 「イメージ > 画像解像度」
2. 解像度: 300-360 dpi
3. 最終プリントサイズに合わせる
   例: A4プリント → 210mm × 297mm × 360dpi
```

#### Step 1-5: 出力
```
1. 「ファイル > 別名で保存」
2. フォーマット: TIFF
3. ビット深度: 16bit（推奨）
4. 圧縮: なし（またはLZW）
5. 保存
```

#### Step 2-1: Print-Tool起動

```
/Applications/QuadToneRIP/QTR-Print-Tool.app
```

#### Step 2-2: 画像読み込み

```
1. 「Open Image」ボタンクリック
```

#### Step 2-3: プリント設定

```
1. 設定確認（特にカーブ名）
2. 「Print」ボタンクリック
```

#### Step 2-5: 乾燥
```
1. プリント完了後、OHPフィルムを取り出す

3. 乾燥確認後、使用可能
```

## 🎨 プラチナ/パラジウムプリントワークフロー

### 1. 感光液調合

**標準配合（1回分、A4サイズ）**:

| 成分 | 使用量 | 役割 |
|------|--------|------|
| シュウ酸第二鉄 (Fe(C₂O₄)₃) | 0.60 cc | 鉄塩（酸化剤） |
| パラジウム溶液 (PdCl₂系) | 0.675 cc | パラジウム供給源 |
| 塩化白金酸ナトリウム (Na₂PtCl₆) 20% w/v | 0.25 cc | プラチナ供給源 |

**調合手順**:
```
1. UV遮断照明下で作業
2. 清潔なビーカーで各成分を計量
3. スポイトで正確に計量（滴数カウント）
4. 均一に混合
5. すぐに使用（長時間放置しない）
```

---

### 2. 感光液塗布

**準備**:
```
- 印画紙: Arches Platine 310gsm（または使用する紙）
- 塗布具: ガラス棒またはホーク（刷毛）
- 作業場所: UV遮断照明、換気良好
```

**塗布手順**:
```
1. 印画紙をカット（プリントサイズ + 余白2-3cm）
2. 平らな作業台に固定
3. 感光液を均一に塗布
   - 塗布量: 約0.8-1.0 ml / A4サイズ
   - 塗布ムラに注意
   - 端まで均一に
4. 塗布完了
```

---

### 3. 乾燥

**自然乾燥**:
```
1. 暗所（UV遮断）で平置き
2. 乾燥時間: 30分〜1時間
3. 完全乾燥を確認
```

**ドライヤー使用**（時間短縮）:
```
1. 低温設定
2. 距離30cm以上離す
3. 均一に風を当てる
4. 乾燥時間: 10-15分
```

---

### 4. 露光

**露光条件（カーブ別）**:

| カーブ | 露光時間 | 備考 |
|--------|---------|------|
| **xf1-v14** | **6分45秒** | 最も豊かな階調 |
| **v14-washi-v6** | **6分45秒** | ハイライト拡張 |
| **xf1-v28** | **6分05秒** ★重要 | v26から-40秒 |
| **v28-multiink** | **6分05秒** ★重要 | v28と同じ |

**共通条件**:

| 項目 | 設定値 |
|------|--------|
| **光源** | PGI UV-Printer（365nm UV LED） |
| **距離** | 30cm（標準） |
| **湿度** | 40-60% RH |

**露光手順**:
```
1. UV露光機をセット
2. デジタルネガと印画紙を密着
   - ネガ: 乳剤面（印刷面）を下
   - 印画紙: 感光面を上
3. コンタクトフレームで密着（または重し）
4. タイマーセット: カーブに応じた時間（★重要）
5. 露光開始
6. 露光中は振動を避ける
7. 露光完了後、すぐに現像へ
```

**露光時間の微調整**:
```
- 白っぽい（露光不足）: +15秒
- 暗すぎる（露光過剰）: -15秒
- 微調整は15秒単位で実施
```

---

### 5. 現像・水洗・乾燥

#### Step 5-1: 現像

**現像液（クエン酸カリウム飽和溶液）**:
```
成分: クエン酸カリウム (K₃C₆H₅O₇)
濃度: 飽和溶液
温度: 20-24°C（室温）
使用量: 約1リットル/バット
```

**現像手順**:
```
1. 現像液バットを準備
2. 印画紙を現像液に浸ける
3. 穏やかに揺らしながら現像
4. 画像が完全に出現するまで（1-2分）
5. 完全に出現したら取り出す
```

#### Step 5-2: 1回目の水洗
```
1. 流水バットに移す
2. 流水で5分間水洗
3. 残留鉄塩を除去
```

#### Step 5-3: クリアリング

**クリアリング液（EDTA）**:
```
成分: EDTA 4Na
濃度: 1% w/v
温度: 20-24°C（室温）
使用量: 約500ml/バット
処理時間: 5分
```

**クリアリング手順**:
```
1. EDTA液バットを準備
2. 印画紙をEDTA液に浸ける
3. 穏やかに揺らす
4. 5分間処理
5. 取り出す
```

#### Step 5-4: 2回目の水洗
```
1. 流水バットに移す
2. 流水で20-30分水洗
3. 薬品を完全に除去
```

#### Step 5-5: 乾燥
```
1. 水を切る
2. 平らな場所で自然乾燥
   （または低温ドライヤー）
3. 完全乾燥まで待機（数時間〜1日）
```

## 🔧 トラブルシューティング

### 問題1: バンディング（境界線）が見える

**原因**:
- 古いカーブファイル使用
- QTRデータベース未更新
- Print-Toolキャッシュ問題

**対処**:
```bash
# 1. カーブ確認
ls -la /Library/Printers/QTR/quadtone/QuadP700/PX1V-PtPd-xf1-*.quad

# 2. データベース更新
/Library/Printers/QTR/bin/quadcurves QuadP700

# 3. キャッシュクリア＆再起動
defaults delete com.quadtonerip.QTR-Print-Tool 2>/dev/null
defaults delete com.quadtonerip.QTR-Print 2>/dev/null
killall "QTR Print Tool" 2>/dev/null
open /Applications/QuadToneRIP/QTR-Print-Tool.app

# 4. Print-Toolで正しいカーブが選択されているか確認
```

---

### 問題2: 紙白が到達しない（全体的に暗い）

**原因**:
- 露光時間不足
- カーブ選択ミス
- デジタルネガが薄すぎる
- 感光液の問題

**対処**:
```
1. カーブ確認: v28/v28-multiinkは6分05秒、v14系は6分45秒
2. 露光時間を+15秒延長してテスト
3. 紙白到達が重要ならv28/v28-multiinkに変更
4. Photoshopでハイライトを持ち上げる
5. 感光液の濃度・新鮮度を確認
6. UV光源の強度を確認
```

### 問題3: 全体的に白っぽい（露光過剰）

**原因**:
- 露光時間過剰
- カーブ選択ミス
- デジタルネガが濃すぎる

**対処**:
```
1. カーブ確認: v28系は6分05秒（6分45秒ではない）
2. 露光時間を-15秒短縮してテスト
3. Photoshopでハイライトを少し下げる
```

### 問題4: コントラストが高すぎる

**原因**:
- Photoshop調整が強すぎる
- v14-washi-v6のハイライト拡張が強すぎる

**対処**:
```
1. Photoshopでトーンカーブを調整
   - 中間調を持ち上げる（S字カーブを緩める）
2. ミッドトーンの明度を上げる
3. v14-washi-v6からv14に変更を検討
4. 再度デジタルネガ出力＆プリント
```

### 問題5: ディザリングパターンが見える

**原因**:
- K単色カーブ（v14/v14-washi-v6/v28）使用
- プリンターのディザリングアルゴリズムによる規則的なドット

**対処**:
```
1. v28-multiinkに変更（GY+LGY 3%混合でパターン解消）
2. 露光時間は同じ6分05秒を維持
3. わずかな暖色傾向を確認（グレーインク混合による）
```

### 問題6: Print-Toolでカーブが表示されない

**原因**:
- インストール不完全
- 拡張属性（@マーク）が残っている
- BOOST_K行欠落

**対処**:
```bash
# 1. カーブファイル存在確認
ls -la /Library/Printers/QTR/quadtone/QuadP700/PX1V-PtPd-xf1-*.quad

# 2. 拡張属性確認（@マークがあればNG）
# 出力に@マークがある場合、削除が必要
osascript -e 'do shell script "xattr -c /Library/Printers/QTR/quadtone/QuadP700/PX1V-PtPd-xf1-*.quad" with administrator privileges'

# 3. BOOST_K行の確認
grep "BOOST_K" /Library/Printers/QTR/quadtone/QuadP700/PX1V-PtPd-xf1-v14.quad
# 期待出力: # BOOST_K=0 - NO BOOST

# 4. データベース更新
/Library/Printers/QTR/bin/quadcurves QuadP700

# 5. キャッシュクリア
defaults delete com.quadtonerip.QTR-Print-Tool 2>/dev/null
defaults delete com.quadtonerip.QTR-Print 2>/dev/null
```


## ❓ FAQ

### Q1: どのカーブを使えばよいですか？

**A**: 用途に応じて選択してください。

| 用途 | 推奨カーブ |
|------|-----------|
| コットン紙、最も豊かな階調 | **xf1-v14** |
| 和紙、ハイライト重視 | **v14-washi-v6** |
| 紙白確実到達、K単色 | **xf1-v28** |
| ディザリング解消、滑らか | **v28-multiink** |

---

### Q2: 露光時間は厳守ですか？

**A**: はい、カーブごとに厳守してください。

| カーブ | 露光時間 | 重要度 |
|--------|---------|--------|
| v14系（v14, v14-washi-v6） | 6分45秒 | ★★★ |
| v28系（v28, v28-multiink） | **6分05秒** | ★★★★★ |

**特にv28系は重要**: v26から-40秒短縮（K値+19.4%に対応）

---

### Q3: v28とv28-multiinkはどう使い分けますか？

**A**: ディザリングパターンの有無で判断。

```
【v28（K単色）】
- プリンターのディザリングパターンが出る可能性
- 純粋なK単色表現
- 露光: 6分05秒

【v28-multiink】
- ディザリングパターン解消（GY+LGY 3%混合）
- より滑らかな階調
- わずかに暖色傾向
- 露光: 6分05秒（同じ）

→ まずv28でプリント、ディザリングが気になればv28-multiinkに変更
```

---

### Q4: Photoshopでどのように調整すればよいですか？

**A**: トーンカーブが最も効果的です。

```
【コントラスト高めたい場合】
Photoshop > トーンカーブ > S字カーブ
- ミッドトーンを下げる
- シャドウを少し落とす
- ハイライトを少し持ち上げる

【コントラスト低めたい場合】
Photoshop > トーンカーブ > 逆S字カーブ
- ミッドトーンを持ち上げる
- 全体的に明るく

【ハイライト重視の場合】
- v14-washi-v6を使用
- Input 240-255を積極的に活用
```

---

### Q5: v14で和紙プリントしたら、Input 240-255が紙白になりました

**A**: これは正常です。v14-washi-v6に変更してください。

```
【原因】
和紙はコットン紙よりハイライト域の階調表現幅が狭い
v14のInput 240-255: K値範囲4400 K → 和紙では不足

【解決策】
v14-washi-v6を使用（Input 240-255: K値範囲5000 K、1.14倍拡張）
露光時間は同じ6分45秒

【設計思想】
ネガ思考: プリントで階調豊か → ネガで濃度差大きく → K値範囲拡張
```
