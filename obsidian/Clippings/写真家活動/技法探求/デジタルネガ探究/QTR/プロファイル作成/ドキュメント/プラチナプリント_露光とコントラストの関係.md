# プラチナプリント: 露光とコントラストの関係

## 結論（重要）

**露光オーバー → コントラストが高くなる**

これは**正しい**です。

---

## 露光オーバーの影響

### 1. Dmax (最大濃度) への影響

```
露光不足:  Dmax = 1.00D (薄い、コントラスト低)
適正露光:  Dmax = 1.20D (バランス)
露光オーバー: Dmax = 1.40D (濃い、コントラスト高)
```

**理由**:
- プラチナ塩の還元が進む
- より多くの金属プラチナが析出
- シャドウ部がより濃くなる

---

### 2. ハイライト部への影響（非線形）

```
         紙白    ハイライト  ミッドトーン  シャドウ
露光不足:  0.05D → 0.15D → 0.60D → 1.00D
適正露光:  0.05D → 0.20D → 0.75D → 1.20D
オーバー:  0.05D → 0.22D → 0.85D → 1.40D
```

**重要な発見**:
- 紙白は変わらない（露光の影響を受けない）
- シャドウは大きく変化（露光に敏感）
- ハイライトは少し変化

**結果**:
```
濃度範囲 = Dmax - Dmin

露光不足:  1.00 - 0.05 = 0.95D (低コントラスト)
適正露光:  1.20 - 0.05 = 1.15D (標準)
オーバー:  1.40 - 0.05 = 1.35D (高コントラスト)
```

## あなたの仮説の検証

### **「v8-HEのコントラストが高い」原因は？**

#### 可能性1: QTRカーブのK_maxが高い
```
v7-NA: K_max = 33,051 → ネガ濃度 1.08D
v8-HE: K_max = 36,356 → ネガ濃度 1.15D (+0.07D)

→ ネガが濃い分、露光オーバー気味になる
→ プリントのコントラストが高くなる
```

#### 可能性2: 露光時間が同じため、v8-HEが相対的にオーバー露光
```
同じ露光時間で:
v7-NA用ネガ: 濃度 1.08D → 適正露光
v8-HE用ネガ: 濃度 1.15D → 露光オーバー気味

→ v8-HEでプリントすると、相対的に露光過多
→ コントラストが高くなる
```

## 重要な発見

### **v8-HEのコントラストが高い理由**

おそらく:

1. **ネガのK_maxが10%高い** (33,051 → 36,356)
2. **同じ露光時間を使用** (v7-NAと同じ)
3. **結果として露光オーバー気味** → コントラスト高

### **解決策**

#### 選択肢1: QTRカーブを調整（根本的）
```
v8-HE-LC版を作成:
- ハイライト圧縮は維持
- K_maxをv7-NAと同じ33,051に下げる
→ 同じ露光時間でコントラストが揃う
```

#### 選択肢2: 露光時間を調整（運用的）
```
v8-HE使用時は露光を10%短縮
例: v7-NA 8分 → v8-HE 7分12秒
→ プリントDmaxを揃える
```

#### 選択肢3: 両方を組み合わせ（最適）
```
1. v8-HE-LC版を作成（K_max調整）
2. 露光テストで最適化
3. ハイライト圧縮効果を確認
```
