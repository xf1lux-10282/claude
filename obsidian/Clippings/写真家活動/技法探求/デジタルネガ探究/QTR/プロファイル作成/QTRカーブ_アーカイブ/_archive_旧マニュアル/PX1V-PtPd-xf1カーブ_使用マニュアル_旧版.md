# PX1V-PtPd-xf1 QTRカーブ 使用マニュアル

**プラチナ/パラジウムプリント用デジタルネガカーブ**
**対応用紙**: Pictorico OHP Film
**対応プリンター**: Epson P700 (QuadToneRIP)
**作成日**: 2026年3月21日

---

## 概要

PX1V-PtPd-xf1シリーズは、プラチナ/パラジウムプリント用のデジタルネガを作成するためのQTRカーブです。線形補間により**バンディング（境界線）を完全に解消**し、滑らかなグラデーションを実現しています。

**利用可能なカーブ:**
- **PX1V-PtPd-xf1-v7-NA** (ナチュラル - 通常使用・推奨)
- **PX1V-PtPd-xf1-v5-HC** (ハイコントラスト - コントラスト重視)
- **PX1V-PtPd-xf1-v8-HE** (ハイライト強化 - 白飛び抑制・ゆっくり消える)

| 項目 | 仕様 |
|-----|------|

**特徴:**
- **Input 0-232**: v7-NAと完全同一（Base域保護）
- **Input 232-255**: ハイライト圧縮（ΔDを細かく）
- 白に向かって"ゆっくり消える"表現
- ハイライト情報量の増加
- v7-NAより高濃度まで使用（1.18D）

**向いている被写体:**
- ハイライトの微妙な階調が重要な作品
- 白飛びしやすい被写体（雪、白い花、空など）
- 明るい部分の質感を保持したい場合
- デリケートなハイライト表現

**技術詳細:**
- Base域（0-232）のGradient: 129-130 K/step（v7-NAと同一）
- Highlight域（232-255）のGradient:
  - 232→236: 270 K/step
  - 236→252: 225 K/step（均一）
  - 252→255: 535 K/step（急峻）
- Gradient比率: 2.38倍（バンディングリスク小）

**注意:**
- Input 252-255は急峻なGradient（実物確認推奨）
- テストプリントで効果を確認してから本番使用
- v7-NAとの違いは主にハイライト域（Input 233以降）

## 2. 使い分けの基準

### **基本方針**

```
通常使用 → xf1-v7-NA
↓
テストプリント
↓
【分岐】
├─ コントラストが物足りない → xf1-v5-HC
└─ ハイライトが白飛びする → xf1-v8-HE
```

### **選択チャート**

| 質問 | YES → | NO → |
|------|-------|------|
| ハイライトの白飛びを抑制したい？ | **xf1-v8-HE** | 次へ |
| 柔らかい階調が欲しい？ | **xf1-v7-NA** | 次へ |
| ハイライト域の繊細な表現が重要？ | **xf1-v8-HE** または **v7-NA** | 次へ |
| メリハリのある表現が欲しい？ | **xf1-v5-HC** | 次へ |
| グラフィカルな構図？ | **xf1-v5-HC** | **xf1-v7-NA** |

### **推奨用途別カーブ選択**

| 用途 | 推奨カーブ | 理由 |
|------|----------|------|
| **通常使用（デフォルト）** | **xf1-v7-NA** | バランスが良く、全範囲を活用 |
| **ハイライト重視作品** | **xf1-v8-HE** | 白飛び抑制、ゆっくり消える表現 |
| **高コントラスト作品** | **xf1-v5-HC** | メリハリ、ドラマティック |
| **雪景色・白い花** | **xf1-v8-HE** | ハイライトの微妙な階調保持 |
| **ポートレート** | **xf1-v7-NA** または **v8-HE** | 柔らかい階調 |
| **建築・グラフィック** | **xf1-v5-HC** | シャープなコントラスト |

**迷ったら → xf1-v7-NAを使ってください**
**ハイライトが重要なら → xf1-v8-HEを試してください**

### 3.2 プリント条件（基準値）

#### 薬品調合（1回分・A4サイズ）

| 薬品 | 濃度 | 使用量 | モル濃度 |

**モル比計算**:
- Na₂PtCl₆ 20% w/v = 200 g/L ÷ 561.9 g/mol ≈ 0.356 M × 0.25 cc → 約0.089 mmol Pt
- PdCl₂ 5% w/v = 50 g/L ÷ 177.3 g/mol ≈ 0.282 M × 0.675 cc → 約0.190 mmol Pd
- **実際のモル比**: Pt : Pd ≈ **1 : 2.1**（パラジウム過剰）

**塗布量**: 約0.8-1.0 ml / A4サイズ

## 8. トラブルシューティング

### **Q1. ネガにバンディング（境界線）が見える**

**A:** xf1-v5-HC/v7は線形補間でバンディングを解消しています。それでも見える場合:
- Print-Toolでカーブが正しく選択されているか確認
- QTRデータベースが最新か確認: `sudo /Library/Printers/QTR/bin/quadcurves`

---

### **Q2. ハイライトが飛ぶ（紙白になる）**

**xf1-v7-NAの場合:**
- Input 255が紙白の上限
- ヒストグラムで最も明るい部分がInput 255以下になるよう調整

**xf1-v5-HCの場合:**
- Input 240が紙白の上限
- ヒストグラムで最も明るい部分がInput 240以下になるよう調整

---

### **Q3. コントラストが強すぎる/弱すぎる**

**コントラストが強すぎる場合:**
- xf1-v5-HCを使用している → xf1-v7-NAに変更
- 画像編集時にカーブを調整してコントラストを下げる

**コントラストが弱すぎる場合:**
- xf1-v7-NAを使用している → xf1-v5-HCに変更
- 画像編集時にカーブを調整してコントラストを上げる

---

### **Q4. Input 240-255 の階調が見えない（xf1-v7-NA）**

**A:** これは正常な現象です。
- コットン紙の繊維による物理的限界
- Input 240付近で紙白に到達
- 240-255の微妙な差は繊維に埋もれて視覚的に区別不可
- デジタルネガの問題ではありません

## 12. よくある質問

### **Q: どちらのカーブを使えばいいかわからない**

**A:** 迷ったら **xf1-v7-NA** を使ってください。
- より安全（ハイライトが飛びにくい）
- より豊かな階調
- 後からコントラスト調整も可能

---

### **Q: 同じ画像でもカーブによって仕上がりが変わる？**

**A:** はい、変わります。
- xf1-v7-NA: 柔らかく、階調が豊か
- xf1-v5-HC: メリハリがあり、コントラストが強い

同じ画像でも作品の意図に応じて使い分けてください。

---

### **Q: プラチナプリントの露光時間は変わる？**

**A:** 基本的に同じです。
- 両方とも同じK値範囲でプリント濃度が決まる
- ただし、xf1-v5-HCは若干濃いネガになる可能性あり
- テストプリントで確認してください

---

### **Q: 他の用紙でも使える？**

**A:** このカーブはPictorico OHP Film専用です。
- 他の用紙では濃度が変わる可能性があります
- 用紙を変える場合は、新しいカーブを作成してください
