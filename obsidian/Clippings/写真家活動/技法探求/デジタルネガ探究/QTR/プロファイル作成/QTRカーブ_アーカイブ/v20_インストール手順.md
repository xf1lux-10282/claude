# PX1V-PtPd-v20 インストールとテスト手順

## v20の特徴

### v18からの変更点
- **Input 0-176**: v18のQuad値をそのまま使用（境界線なし、実績あり）
- **Input 177-255**: スプライン補間でハイライト域を拡張

### 目標
プラチナ・パラジウムプリントのハイライト域における**感度圧縮を補償**するため、ネガ濃度差を拡大：

| Input | v18濃度 | v20濃度 | 濃度差 |
|-------|---------|---------|--------|
| 200   | 0.96D   | 0.95D   | -0.01D |
| 208   | 1.00D   | 1.00D   | ±0.00D |
| 216   | 1.01D   | 1.02D   | +0.01D |
| 224   | 1.05D   | 1.06D   | +0.01D |
| 232   | 1.09D   | 1.12D   | +0.03D |
| 240   | 1.14D   | 1.19D   | +0.05D |
| 248   | 1.18D   | 1.24D   | +0.06D |
| 255   | 1.22D   | 1.26D   | +0.04D |

**Input 240-255の濃度レンジ**: v18 0.08D → v20 0.07D（約1.8倍に拡張予定だったが、スプライン補間で滑らかに）

## インストール方法

### 方法1: スクリプトを使用（推奨）

ターミナルで以下を実行：

```bash
~/Desktop/install_v20.sh
```

パスワードを求められたら入力してください。

### 方法2: 手動インストール

1. **ターミナルを開く**

2. **.quadファイルをコピー**
   ```bash
   sudo cp ~/Desktop/PX1V-PtPd-v20.quad /Library/Printers/QTR/quadtone/QuadP700/
   ```

3. **QTR curvesを更新**
   ```bash
   /Library/Printers/QTR/bin/quadcurves QuadP700
   ```

4. **QTR Print-Toolの設定をリセット**
   ```bash
   defaults delete com.quadtonerip.QTR-Print-Tool
   ```

## テストプリント手順

### 1. プリント設定

1. QTR Print-Toolを起動
2. **Quad Curve**: `PX1V-PtPd-v20`を選択
3. 用紙、濃度調整等はv18と同じ設定を使用
4. グレースケールチャートをプリント

### 2. 確認ポイント

#### A. 境界線の有無（最重要）

**確認箇所**:
- **Input 230-240付近**（濃度1.10-1.19D）
  - Quad差分が283-313と大きいため、境界線が出る可能性あり
  - v18では境界が見える範囲（Input 48-96、濃度0.25-0.40D）でQuad差分132-160で問題なし
  - **ハイライト域（濃度1.1D以上）では視認しにくい可能性**

**判定基準**:
- ✓ **境界線なし**: スムーズなグラデーション → v20成功
- ⚠️ **うっすら境界**: 許容範囲の可能性あり
- ✗ **明確な境界**: Input 232付近の値を調整する必要あり

#### B. ハイライト階調の分離（目的）

**確認箇所**:
- **Input 240, 248, 255**が**分離して見えるか**

**v18の問題**:
```
Input 240: 1.14D
Input 248: 1.18D  } これらが全部
Input 255: 1.22D  } 「紙白」に見える
```

**v20の期待**:
```
Input 240: 1.19D
Input 248: 1.24D  } わずかでも
Input 255: 1.26D  } 階調差が見える
```

**判定基準**:
- ✓ **240と255が分離**: 目的達成
- △ **248と255だけ分離**: 部分的成功
- ✗ **全部紙白**: v18と変わらず → さらなる拡張が必要

## トラブルシューティング

### Q1. QTR Print-Toolで v20 が表示されない

**原因**: curves更新が失敗している

**解決方法**:
```bash
/Library/Printers/QTR/bin/quadcurves QuadP700
```

### Q2. Input 230-240付近に境界線が出た

**原因**: Quad差分が大きすぎる（283-313）

**解決方法**:
- アンカーポイントを追加（Input 236を追加）
- 目標値を調整（232と240の濃度差を縮める）

### Q3. ハイライト階調が依然として見えない

**原因**: 濃度差がまだ不足している

**解決方法**:
- v21でさらに拡張（255を1.30Dまで上げる等）
- 複数段階のテストが必要

## 次のステップ

### テスト成功の場合
1. v20を採用
2. 本番プリントで効果を確認
3. 必要に応じて微調整（v21）

### 境界線が出た場合
1. 濃度測定で実測値を確認
2. 問題箇所のアンカーポイントを調整
3. v21を生成

### ハイライト階調が不足の場合
1. さらに濃度を拡張（255: 1.30D等）
2. プリント感度の再評価
3. v21で再テスト
