## Version 1からの変更点

### 1.1 主要パラメータの変更

| 項目 | Version 1 | Version 2 | 変更理由 |
|------|-----------|-----------|----------|
| ネガγ | 2.0 | 1.8 | 和紙+蛍光UV環境の推奨値(1.7〜1.9)に基づく |
| カーブ形状 | 完全γカーブ | 弱い逆Sカーブ | 視覚的な自然さ、optical dot gain補正 |
| 設計基準 | 全体リニア | 中間調基準 | Zone V-VI配置を厳密化 |
| 測定方法 | 1段階 | 2段階分離 | プリンター/プリント特性を分離把握 |

### 1.2 Version 1の成果と課題

**Version 1の成果**

- ネガDmin 0.1、Dmax 1.6の実現
- プラチナプリント Dmax 2.1の達成
- プリント露光レンジ 6.6EV の確認

**Version 1の課題**

- γ=2.0はやや強すぎる可能性(総括の推奨1.7〜1.9より高い)
- 完全γカーブは視覚的に眠く見える可能性
- 蛍光UV+和紙散乱の特性が未考慮
- optical dot gainの補正なし

### 1.3 Version 2の目標

1. **視覚的な自然さ** — 完全リニアではなく、人間視覚に最適化
2. **和紙+蛍光UV環境への完全適合** — 実測データに基づく設計
3. **中間調の豊かさ** — Zone V-VI の階調を最大化
4. **測定可能性** — 2段階測定による精密な検証

## 設計思想

### 2.1 プラチナプリントの本質

```
プラチナプリント = 階調の写真
```

**重要トーン順位**

```
1. 中間調(Zone V-VI)
2. シャドウ(Zone III-IV)
3. ハイライト(Zone VII-VIII)
```

### 2.2 和紙プラチナの光学特性

**和紙の3つの特性**

1. **紙内部散乱** → 中間調が豊か
2. **長繊維構造** → 柔らかいトーン
3. **光漏れやすい** → ネガDminが重要

**蛍光UVの特性**

- 365nm中心、350〜400nm分布
- UV散乱 → ハイライトが自然に持ち上がる
- エッジ拡散 → 微細階調が平滑化

**結果**

```
和紙散乱 + UV散乱 = 非常に滑らかなトーン
```

**ネガ設計への影響**

→ ネガカーブは「弱い逆Sカーブ」が最適

### 2.3 デジタルネガの3つの非線形

```
入力値
  ↓
① インク量 → ネガ濃度(プリンター非線形)
  ↓
② ネガ濃度 → 露光量(対数変換)
  ↓
③ 露光量 → プリント濃度(化学反応)
  ↓
プリント濃度
```

**Version 2のアプローチ**

これら3つを**2段階測定で分離把握**し、**1つの逆カーブで補正**する。

## 技術仕様

### 3.1 基本パラメータ

| 項目 | 仕様 | 根拠 |
|------|------|------|
| ネガγ | 1.8 | 総括推奨値1.7〜1.9の中央 |
| ネガDmin | 0.10 | 和紙での光漏れ防止 |
| ネガDmax | 1.55 | プリンターの線形領域内 |
| ネガ濃度レンジ | 1.45 | Dmax - Dmin |
| プリント露光レンジ | 6.6EV | 実測値 |
| カーブタイプ | 弱い逆Sカーブ | 視覚最適化 |
| 処理bit深度 | 16bit | 階調破壊防止 |

### 3.2 トーン配置(中間調基準設計)

| Zone | 入力値 | ネガ濃度 | 役割 |
|------|--------|----------|------|
| 0 (真黒) | 0 | 1.55 | Dmax |
| III (シャドウ) | 64 | 1.10 | ディテール開始 |
| V (中間調) | 128 | 0.70 | **基準点** |
| VII (明部) | 192 | 0.35 | ハイライト移行 |
| X (真白) | 255 | 0.10 | Dmin |

**設計の核心**

```
入力128 → ネガ濃度0.70
```

これをアンカーポイントとし、前後のトーンを配置する。

### 3.3 弱い逆Sカーブの定義

**カーブの3つの領域**

| 領域 | 入力範囲 | 特性 | 目的 |
|------|----------|------|------|
| シャドウ | 0-64 | わずかな持ち上げ | ディテール保持 |
| 中間調 | 64-192 | ほぼ直線(γ≈1.8) | 階調の豊かさ |
| ハイライト | 192-255 | 軽い圧縮 | 白の自然さ |

**逆Sカーブの強度**

- **強すぎるS** → 中間調が硬くなる
- **弱いS** → 視覚的に自然
- **完全直線** → 眠く見える

**Version 2の方針**

```
中間調は直線を維持
シャドウ・ハイライトのみ微調整
```

### 3.4 Optical Dot Gain補正

**インクジェットネガの特性**

```
ネガ = 透明フィルム + 微細インク粒子
```

**光の挙動**

```
UV光
  ↓
インク粒子で散乱
  ↓
透過光が拡散
  ↓
実効濃度 > 理論濃度
```

**結果**

- 中間調のコントラストが過剰になりやすい
- 特に50%付近で顕著

**Version 2での対応**

→ 中間調カーブを**わずかに緩める**(γを下げる)ことで補正

## カーブ設計

### 4.1 カーブ設計の方針

**3つの制約条件**

1. **入力128 → ネガ濃度0.70** (中間調基準)
2. **入力0 → ネガ濃度1.55** (Dmax)
3. **入力255 → ネガ濃度0.10** (Dmin)

**設計手順**

```
① 基準点(128, 0.70)を固定
② 直線γ≈1.8のカーブを引く
③ シャドウ・ハイライトを微調整
④ 32点カーブで実装
```

### 4.2 32点制御点の配置

**Version 2のカーブポイント設計**

制御点を3つの領域に分配:

| 領域 | 入力範囲 | 制御点数 | 間隔 |
|------|----------|----------|------|
| シャドウ | 0-64 | 8点 | 8ステップ |
| 中間調 | 64-192 | 16点 | 8ステップ |
| ハイライト | 192-255 | 8点 | 8ステップ |

**重要領域の密度**

- 中間調に**50%の制御点**を配分
- 階調変化が大きい部分を重点的に制御

### 4.3 Version 2カーブの数式表現

**基本γカーブ**

```
ネガ濃度 = Dmax - (Dmax - Dmin) × (入力値/255)^γ
```

**Version 2パラメータ**

```
Dmax = 1.55
Dmin = 0.10
γ = 1.8
```

**基本カーブ(補正前)**

```
D(x) = 1.55 - 1.45 × (x/255)^1.8
```

ここに**逆S補正**を加える:

```
D_v2(x) = D(x) + S(x)
```

**S(x) = 逆S補正関数**

- シャドウ(0-64): +0.02〜0.05 (持ち上げ)
- 中間調(64-192): ±0 (直線維持)
- ハイライト(192-255): -0.02〜-0.05 (圧縮)

## 測定プロトコル

### 5.1 測定の2段階分離

**目的**

プリンターとプラチナプリントの特性を分離して把握する。

**測定①: 入力値 → ネガ濃度**

目的: プリンターの線形領域とインク飽和点を知る

**測定②: ネガ濃度 → プリント濃度**

目的: プラチナプリントの実際のトーンカーブを知る

### 5.2 測定① — プリンター特性

**手順**

1. **グレーステップチャートを作成**
   - 入力値: 0, 8, 16, 24 ... 248, 255 (32ステップ)
   - 16bit TIFFで保存

2. **デジタルネガとして印刷**
   - プリンター: EPSON PX-1V
   - メディア: ピクトリコOHP
   - 設定: いつもの条件(Version 1と同じ)

3. **ネガ濃度を測定**
   - 濃度計でネガ濃度を測る
   - 各ステップ3箇所測定して平均

4. **データ記録**

| 入力値 | ネガ濃度 | 備考 |
|--------|----------|------|
| 0 | (測定値) | |
| 8 | (測定値) | |
| ... | ... | |
| 255 | (測定値) | |

**分析項目**

- Dmin (入力255での濃度)
- Dmax (入力0での濃度)
- 線形領域 (濃度変化が等間隔の範囲)
- 飽和領域 (濃度変化が鈍化する範囲)

### 5.3 測定② — プラチナプリント特性

**手順**

1. **測定①と同じネガを使用**

2. **コンタクトプリント**
   - 紙: 和紙
   - FO濃度: いつもの高濃度配合
   - 露光: Dmaxが出る適正露光
   - 現像: 標準プロトコル

3. **プリント濃度を測定**
   - 各ステップの濃度を測る
   - 各ステップ3箇所測定して平均

4. **データ記録**

| ネガ濃度 | プリント濃度 | 備考 |
|----------|--------------|------|
| (測定①の値) | (測定値) | |
| (測定①の値) | (測定値) | |
| ... | ... | |

**分析項目**

- プリントDmin (最も明るい部分)
- プリントDmax (最も暗い部分)
- ネガ濃度 → プリント濃度のカーブ形状
- 直線性の確認

### 5.4 データ統合と逆カーブ作成

**統合**

```
入力値 → ネガ濃度 → プリント濃度
```

この関係から

```
入力値 → プリント濃度
```

の実測カーブを得る。

**逆カーブの作成**

理想: **入力値 ∝ プリント濃度** (完全リニア)

実測カーブの**逆関数**を求める。

**補正カーブ**

```
補正カーブ = 理想カーブ - 実測カーブ
```

これを32点カーブとして実装。

### 5.5 測定環境の統一

**重要な統一項目**

| 項目 | 条件 |
|------|------|
| 温度 | 20〜25℃ |
| 湿度 | 40〜60% |
| 濃度計 | 同一機材、同一設定 |
| 測定位置 | 各ステップ中央部 |
| 測定回数 | 各3回、平均値 |

## 実装方法

### 6.1 処理フロー

```
RAW画像(16bit)
  ↓
グレースケール変換(16bit)
  ↓
【Version 2 トーンカーブ】
  ├ 32点制御カーブ
  ├ γ=1.8基準
  └ 弱い逆Sカーブ
  ↓
16bit TIFF ネガ画像
  ↓
反転(ネガ化)
  ↓
プリント出力
```

### 6.2 Photoshopでの実装

**手順**

1. **16bit画像で開く**
   - RAW → 16bit/チャンネル
   - ProPhoto RGBまたはGray Gamma 2.2

2. **グレースケール変換**
   - モード → グレースケール(16bit)

3. **カーブ調整**
   - レイヤー → 新規調整レイヤー → トーンカーブ
   - 32点の制御点を配置(4.4の表に基づく)

4. **反転**
   - イメージ → 色調補正 → 階調の反転

5. **保存**
   - 16bit TIFF
   - 圧縮なし

### 6.3 LUTでの実装

**LUTファイルの作成**

```python
# Version 2 カーブデータ(4.4の表)
curve_data = [
    (0, 1.55),
    (8, 1.52),
    (16, 1.47),
    # ... 32点すべて
    (255, 0.10)
]

# 256段階に補間
lut = interpolate_curve(curve_data, method='cubic')

# .cubeファイルとして書き出し
export_cube_lut(lut, "Precision_EDN_v2.cube")
```

**使用**

- Photoshop: カラールックアップで読み込み
- Lightroom: プロファイルとして適用
- RawTherapee: Film Simulationで適用

### 6.4 グラデーションマップでの実装

**手順(Photoshop)**

1. **グラデーションマップレイヤー作成**

2. **32点カラーストップを配置**
   - 位置0% → RGB(158, 158, 158) ※濃度1.55相当
   - 位置3.1% → RGB(161, 161, 161) ※濃度1.52相当
   - ...
   - 位置100% → RGB(230, 230, 230) ※濃度0.10相当

3. **保存**
   - グラデーションプリセットとして保存
   - "Precision_EDN_v2.grd"

### 6.5 16bit処理の注意点

**重要**

- 途中で8bitに落とさない
- レイヤー合成も16bitで
- 最終出力まで16bit維持

**階調破壊の防止**

```
RAW(16bit)
  ↓ (16bit)
グレー変換(16bit)
  ↓ (16bit)
カーブ(16bit)
  ↓ (16bit)
反転(16bit)
  ↓ (16bit)
TIFF保存(16bit)
```

## 付録B: トラブルシューティング

### B.1 プリントが眠く見える

**原因候補**

1. ネガγが低すぎる
2. 逆Sカーブが強すぎる
3. 露光不足

**対処**

- γを1.85に上げる
- 逆Sカーブを弱める
- 露光時間を延ばす

### B.2 ハイライトが濁る

**原因候補**

1. ネガDminが高い
2. 光漏れ
3. 紙のベース濃度

**対処**

- ネガDminを再測定
- 印刷設定確認
- 別の紙でテスト

### B.3 シャドウが潰れる

**原因候補**

1. ネガDmaxが高すぎる
2. 露光過多
3. シャドウ持ち上げ不足

**対処**

- カーブのシャドウ部を持ち上げ(+0.05 → +0.08)
- 露光時間を短縮
- 現像時間を確認

### B.4 中間調が硬い

**原因候補**

1. optical dot gainの過剰補正
2. γが高すぎる

**対処**

- 中間調カーブを緩める
- γを1.75に下げる
