## 1. 写真トーンの基本構造

プラチナプリントは次の変換システムです。

```
シーン光
  ↓
デジタル画像
  ↓
デジタルネガ
  ↓
UV露光
  ↓
化学反応
  ↓
プリント濃度
```

つまり

**入力画像 → プリント濃度** の変換システムです。

## 2. フィルム特性曲線の理論

### 2.1 特性曲線（H&Dカーブ）とは

写真材料には

**露光量 → 濃度**

の関係があります。

これを **特性曲線（Characteristic Curve）** または **H&D curve** と呼びます。

```
濃度
↑
|            ____ shoulder
|          /
|        /
|      /
|    /
|  /
|_/  toe
+----------------→ log露光
```

### 2.2 S字カーブの3つの領域

| 領域 | 名前 | 特徴 |
|------|------|------|
| toe | シャドウ | 立ち上がりが緩い |
| straight line | 中間調 | ほぼ直線 |
| shoulder | ハイライト | 飽和に向かう |

### 2.3 なぜS字になるのか

**Toe（シャドウ）**

- 露光が少ないと潜像がほとんどできない
- 銀ハロゲン結晶はある程度の光が当たらないと現像核が形成されない
- 結果：濃度がゆっくり立ち上がる

**Straight-line（中間調）**

- 露光が十分になると多くの結晶に潜像ができる
- この領域では **露光量 ∝ 現像される銀量**
- つまりほぼ比例関係

**Shoulder（ハイライト）**

- 露光が非常に多いとほとんどの結晶がすでに現像可能
- それ以上露光しても新しく現像される結晶はほぼ増えない
- 結果：濃度が飽和

### 2.4 ガンマ（γ）とコントラスト

**ガンマ（γ）** とは特性曲線の直線部分の **傾き** です。

```
γ = Δ濃度 / Δlog露光
```

つまり

**露光の変化が、どれだけ濃度差になるか** を表しています。

| 現像 | ガンマ | コントラスト |
|------|--------|--------------|
| 短い現像 | 低γ | 低コントラスト |
| 標準 | 標準γ | 標準 |
| 長い現像（プッシュ） | 高γ | 高コントラスト |

### 2.5 プッシュ現像とは

**プッシュ現像** とは

- 露光不足を現像で補う処理
- 現像時間を延ばす

**何が起きるか**

```
現像時間 ↑
  ↓
銀の生成量 ↑
  ↓
ネガ濃度 ↑
```

ただし **全部均等に濃くなるわけではない**。

| 部分 | 変化 |
|------|------|
| シャドウ | 少し濃くなる |
| 中間 | 濃くなる |
| ハイライト | 強く濃くなる |

つまり **コントラストが上がる**。

### 2.6 なぜプッシュしてもシャドウは救えないのか

フィルムの現像は **信号を作るのではなく、既にある信号を増幅する** だけです。

```
露光が足りないと
  ↓
潜像が存在しない
  ↓
現像をどれだけ延ばしても
  ↓
潜像なし → 現像されない
```

つまり **情報は生まれません**。

### 2.7 ゾーンシステムの本質

**ゾーンシステム** は単なる露出テクニックではなく、**フィルムの特性曲線を操作する理論** です。

| 操作 | 影響 |
|------|------|
| 露光 | シャドウを決める |
| 現像 | ハイライトを決める |

つまり

```
Expose for shadows
Develop for highlights
```

### 2.8 log露光を使う理由

写真では露光量を **log(E)** に変換します。

**理由**

1. **人間の視覚は対数的**
   - 人間は光の **比率** で明るさを感じる
   - 10 → 20 と 100 → 200 は同じ「明るさ2倍」

2. **EVとlog露光**
   - 1EV増える = 露光量 ×2
   - つまり `EV = log2(E)`

3. **等間隔になる**
   - 露光量: 1, 2, 4, 8, 16, 32 → 2倍ずつ増加
   - log: 0, 1, 2, 3, 4, 5 → 等間隔

### 2.9 写真のトーン変換

写真とは

```
光の物理量を直接写しているのではない

光
  ↓
対数変換
  ↓
非線形変換（Sカーブ）
  ↓
画像
```

という **数学的変換** です。

### 2.10 ネガ濃度の定義

写真では透過率の逆数を取り、対数を取ります。

```
D = log10(1/T)
```

これを **光学濃度（Optical Density）** と呼びます。

| 透過率 T | 濃度 D |
|----------|--------|
| 1.0 | 0 |
| 0.5 | 0.30 |
| 0.25 | 0.60 |
| 0.10 | 1.00 |
| 0.01 | 2.00 |

重要なのは

**濃度0.3ごとに光が半分** になることです。

## 3. プラチナプリントの特性

### 3.1 銀塩印画紙との決定的な違い

**銀塩印画紙**

- S字カーブ
- 潜像形成・現像核・結晶サイズという閾値反応を持つ

**プラチナプリント**

- ほぼ直線カーブ
- 連続的な化学反応（Fe³⁺ → Fe²⁺ → Pt/Pd還元）
- **光量 ∝ 生成金属量**

### 3.2 特性曲線の比較

**銀塩印画紙**

```
濃度
↑
|            ____ shoulder
|          /
|        /
|      /
|    /
|  /
|_/ toe
+----------------→ log露光
```

**プラチナプリント**

```
濃度
↑
|       /
|     /
|   /
| /
+----------------→ log露光
```

ToeとShoulderが **ほぼありません**。

### 3.3 階調の特徴

| プロセス | 特性曲線 | 階調 |
|----------|----------|------|
| 銀塩印画紙 | Sカーブ | トーン圧縮 |
| プラチナ | 直線 | トーン再現 |

つまり

- 銀塩：ハイライト圧縮・シャドウ圧縮
- プラチナ：階調が均等

### 3.4 ガンマの違い

| プロセス | γ |
|----------|---|
| 銀塩印画紙 | 1.5〜2 |
| プラチナ | 0.8〜1 |

つまりプラチナは **コントラストが低い**。

### 3.5 ネガ設計の違い

**銀塩写真**

```
シーン
  ↓
ネガ (低γ ≈ 0.55〜0.65)
  ↓
紙 (高γ ≈ 1.7〜2)
  ↓
プリント
```

**プラチナプリント**

```
シーン
  ↓
ネガ (高γ ≈ 1.4〜2.2)
  ↓
紙 (低γ ≈ 0.8〜1)
  ↓
プリント
```

設計が **逆** です。

### 3.6 中間調が美しい理由

プラチナプリントの中間調が美しい理由は5つあります。

1. **像が紙内部にある**
   - 銀塩：表面
   - プラチナ：紙繊維内部

2. **光が多重散乱する**
   ```
   光
     ↓
   紙繊維に侵入
     ↓
   金属粒子
     ↓
   散乱
     ↓
   再反射
   ```

3. **粒子が小さい**
   - 銀粒子：0.3〜1µm
   - Pt/Pd粒子：0.05〜0.2µm
   - 粒子が小さいほど光散乱が細かく、滑らかなトーン

4. **紙繊維が拡散する**
   - 和紙やコットン紙の繊維がランダム構造
   - 天然ディフューザーとして働く

5. **人間視覚が中間調に敏感**
   - 視覚のコントラスト感度は中間輝度で最大

### 3.7 プラチナプリントの本質

プラチナプリントは

**コントラストの写真ではなく、階調の写真** です。

つまり

**光のグラデーションを記録する技法** です。

## 4. デジタルネガの3つの非線形

デジタルネガでは途中に **3つの非線形** があります。

```
画像データ
  ↓
① インク量 → ネガ濃度（プリンタ非線形）
  ↓
② ネガ濃度 → 露光量（光透過非線形）
  ↓
③ 露光量 → プリント濃度（化学反応非線形）
  ↓
プリント濃度
```

### 4.1 非線形① インク量 → ネガ濃度

インクジェットプリンタでは

**インク量と濃度は比例しません**。

理由

- インクの重なり
- ドット構造
- インク拡散
- メディア吸収

例

| 入力 | 濃度 |
|------|------|
| 50% | 0.4 |
| 60% | 0.55 |
| 70% | 0.8 |

つまりカーブになります。

### 4.2 非線形② ネガ濃度 → 露光量

ネガ濃度は

```
D = log10(1/T)
```

つまり **濃度差 → 光量比** は指数関係です。

| 濃度 | 透過光 |
|------|--------|
| 0.3 | 50% |
| 0.6 | 25% |
| 0.9 | 12% |

### 4.3 非線形③ 露光量 → プリント濃度

プラチナプリントは直線に近いですが、実際には完全直線ではありません。

理由

- 紙の吸収
- コーティングムラ
- 金属生成効率
- 散乱光

### 4.4 リニア化とは

リニア化とは

**この3つの非線形を逆カーブで打ち消すこと** です。

```
逆カーブ
  ↓
非線形
  ↓
直線
```

## 5. プリンターの線形領域

### 5.1 プリンターの線形領域とは

インクジェットプリンタでは

```
入力値（0〜255）
  ↓
インク量
  ↓
ネガ濃度
```

という関係があります。

理想は **入力値 ∝ ネガ濃度** ですが、実際は

```
濃度
│        ______
│      /
│    /
│  /
│ /
└────────
入力
```

というカーブになります。

この中で **ほぼ直線** になっている部分が **プリンターの線形領域** です。

### 5.2 なぜ重要なのか

**線形領域では**

- 入力値の変化 → 濃度の変化 が安定

**ハイライト端・シャドウ端では**

- 入力を変えても濃度が変わらない（インク飽和）

この領域を使うと

- 階調が詰まる
- リニア化が壊れる

### 5.3 測定方法

**手順**

1. **0〜255のグレーステップを作る**
   - 例：0, 8, 16, 24 … 248, 255（32ステップでもOK）

2. **デジタルネガとして印刷**

3. **濃度計で測る**
   - `入力値 → ネガ濃度` の表を作る

**例**

| 入力値 | 濃度 |
|--------|------|
| 0 | 0.10 |
| 32 | 0.18 |
| 64 | 0.30 |
| 96 | 0.48 |
| 128 | 0.70 |
| 160 | 0.95 |
| 192 | 1.20 |
| 224 | 1.40 |
| 255 | 1.45 |

この場合 **192〜255** は濃度変化が小さい → **飽和領域**

**32〜192** がほぼ直線 → **線形領域**

### 5.4 デジタルネガ設計での使い方

理想的には

**線形領域の80〜90%だけを使う**

例：入力20〜200など

これで

- 階調破壊
- ハイライト詰まり

を防げます。

## 6. 和紙プラチナプリントの光学特性

### 6.1 和紙と西洋紙の構造の違い

**西洋紙（Arches Platineなど）**

- 短繊維コットン
- 比較的均一
- 光散乱少

**和紙**

- 長繊維（楮・三椏）
- ランダム構造
- 強い内部散乱

### 6.2 光の散乱

ネガを通ったUV光は

```
ネガ
  ↓
紙表面
  ↓
繊維内部
  ↓
多重散乱
```

この結果

**微細な濃度差 → 空間的に平均化**

されます。

### 6.3 トーンの特徴

| トーン | 特徴 |
|--------|------|
| シャドウ | 深いが柔らかい |
| 中間調 | 非常に豊か |
| ハイライト | 非常に滑らか |

つまり **中間調が劇的に増えます**。

### 6.4 その代わりの問題

光散乱があるので、ネガの微小階調が弱くなることがあります。

つまり

**ネガが低コントラストだと、さらに眠く見える** ことがあります。

### 6.5 和紙での理想ネガγ

| 紙 | ネガγ |
|----|-------|
| Arches Platine | 1.8〜2.0 |
| 和紙 | 1.9〜2.2 |

和紙の場合は **少し強め** に作る方が良いです。

### 6.6 Dmax 2.1 が出る理由

和紙 + 高FO濃度 の場合

```
還元量 ↑
  ↓
金属粒子密度 ↑
  ↓
Dmax ↑
```

### 6.7 和紙での重要な設計ポイント

和紙プラチナでは

**ネガのDmaxより重要なのはDmin** です。

理由：和紙は光漏れが起きやすいから。

**ネガDminが高いと → ハイライトが濁ります**

## 7. ネガ設計の実践

### 7.1 デジタルネガのガンマ

デジタルネガで **γ≈2 前後** が必要になる理由は

**プラチナプリントが直線特性だから** です。

```
プリントが直線
  ↓
コントラストが足りない
  ↓
ネガ側でコントラストを作る
```

### 7.2 ガンマの計算

最終的に

```
ネガγ × プラチナγ ≈ 視覚γ
```

にしたい。

例

```
ネガγ 2.0
  ×
プリントγ 1.0
  =
2.0
```

これが自然なトーンになります。

### 7.3 プリントレンジとネガγの関係

| プリントレンジ ΔD | ネガγ |
|-------------------|-------|
| 1.4 | 2.2 |
| 1.6 | 2.0 |
| 1.8 | 1.9 |
| 2.0 | 1.7〜1.8 |

つまり

**Dmax ≈ 2.1（ΔD ≈ 2.0）の環境では**

**ネガγ ≈ 1.7〜1.9** くらいが自然な範囲です。

### 7.4 理想的なネガ濃度配置

| 部分 | ネガ濃度 |
|------|----------|
| ベース+フォグ | 0.1 |
| シャドウ | 0.3 |
| 中間 | 0.7 |
| 明部 | 1.1 |
| ハイライト | 1.4〜1.6 |

重要なのは

**中間調がネガの直線領域の中心にあること** です。

### 7.5 プラチナプリントの露光設計

**銀塩写真**

```
Expose for shadows
Develop for highlights
```

**プラチナプリント**

```
設計基準 = 中間調
```

つまり

**Zone V〜VI を ネガ濃度 ≈ 0.7 付近に置く**

### 7.6 なぜハイライト基準で設計すると失敗するのか

プラチナプリントが再現できる露光差は

**4〜6EV 程度**

です。

ハイライトから設計すると

```
ハイライトにレンジを使う
  ↓
中間調が圧縮
  ↓
立体感・質感・空気感が弱い
  ↓
眠いプリント
```

### 7.7 理想の設計順序

プラチナプリントでは

```
① 中間調
② シャドウ
③ ハイライト
```

の順で設計します。

理由：**視覚感度は中間輝度に最も敏感だから**

### 7.8 弱いSカーブの必要性

完全リニアプリントは、物理的には正確でも **視覚的には眠く見えやすい**。

理由：**人間の視覚が線形ではないから**

多くのプラチナ作家は **緩やかなSカーブ** を使います。

**特徴**

| 部分 | 処理 |
|------|------|
| シャドウ | 少し持ち上げ |
| 中間調 | コントラスト強化 |
| ハイライト | 少し圧縮 |

### 7.9 32点補正について

**32点カーブの補間**

- カーブは直線補間ではなく **Bezier曲線** で繋がる
- 点 → 滑らかな曲線になる

**必要点数**

| カーブタイプ | 必要点 |
|--------------|--------|
| 単純γカーブ | 5〜6 |
| 弱Sカーブ | 10〜12 |
| 複雑補正 | 20〜30 |

つまり **32点は十分多い**。

### 7.10 256点制御の問題

256点を全部制御すると

```
測定ノイズ
  ↓
ギザギザカーブ
  ↓
トーン破壊
```

が起きます。

理想は

```
256測定
  ↓
近似
  ↓
32点カーブ
```

つまり **測定 ≠ 制御点** です。

### 7.11 階調破壊（tone break）

デジタル画像の階調は

- **8bit = 256階調**
- **16bit = 65536階調**

**カーブ処理で何が起きるか**

複数の処理を重ねると

```
256階調
  ↓
180階調
  ↓
130階調
  ↓
90階調
```

のように実際の階調数が減ります。

**防ぐ方法**

1. **処理を最小限にする**
   - 理想：画像 → 1つのトーンカーブ → プリント

2. **16bitで最後まで行う**
   - 8bit → 16bit変換ではなく
   - 最初から **16bit RAW** で処理

### 7.12 Optical Dot Gain

インクジェットネガは均一な濃度フィルターではなく

```
透明フィルム
  +
微細インク粒子
```

です。

光がネガを通るとき

```
インク粒子
  ↓
散乱
  ↓
透過光拡散
```

が起きます。これを **optical dot gain** と言います。

**結果**

| インク | 理論透過光 | 実際透過光 |
|--------|------------|------------|
| 50% | 50% | 35〜40% |

つまり **ネガ濃度が実際より強く働く** ので

**中間調コントラストが過剰** になることが多いです。

### 7.13 蛍光UV光源の特性

**蛍光UV（BL管）のスペクトル**

- 主ピーク：365nm付近
- 広がり：350〜400nm

**UV散乱の影響**

```
ハイライトが少し持ち上がる
中間調が柔らかくなる
エッジが少し拡散する
```

つまり **自然なショルダーが生まれる**。

これはむしろプラチナプリントに有利で

```
紙内部散乱
  +
UV散乱
  =
非常に滑らかなトーン
```

になります。

**ネガ設計への影響**

蛍光UV環境では、ネガカーブは多くの場合 **弱い逆Sカーブ** になります。

| トーン | 設計 |
|--------|------|
| シャドウ | ややフラット |
| 中間調 | 直線 |
| ハイライト | 軽く圧縮 |

## 8. 測定方法

### 8.1 工程を2つの関数に分解する

デジタルネガの精度を上げるには、全体を **2つの関数** に分解して測ります。

```
① 入力値 → ネガ濃度
② ネガ濃度 → プリント濃度
```

### 8.2 測定①：入力値 → ネガ濃度

**目的**

プリンタの **線形領域・飽和領域** を知る。

**手順**

1. 0〜255のグレーステップを作る
   - 例：16または32ステップ
   - 0, 8, 16, 24 … 248, 255

2. それをネガとして印刷（いつもの条件）

3. 濃度計でネガ濃度を測る

4. 次の表を作る

| 入力値 | ネガ濃度 |
|--------|----------|
| 0 | 0.10 |
| 32 | 0.18 |
| 64 | 0.30 |
| 96 | 0.48 |
| 128 | 0.70 |
| 160 | 0.95 |
| 192 | 1.20 |
| 224 | 1.35 |
| 255 | 1.38 |

**分かること**

- 濃度変化が止まるところ → **インク飽和**
- 直線に近いところ → **線形領域**

### 8.3 測定②：ネガ濃度 → プリント濃度

**目的**

プラチナプリントの **実際のトーンカーブ** を知る。

**手順**

1. 同じネガステップを使ってコンタクトプリント

2. 露光は適正（Dmaxが出る露光）

3. プリント濃度を測る

| ネガ濃度 | プリント濃度 |
|----------|--------------|
| 0.10 | 0.05 |
| 0.30 | 0.20 |
| 0.50 | 0.40 |
| 0.70 | 0.70 |
| 0.90 | 1.00 |
| 1.10 | 1.30 |
| 1.30 | 1.70 |
| 1.50 | 2.10 |

この表が **ネガ濃度 → プリント濃度** の関数です。

### 8.4 ここで分かること

この2つを組み合わせると

```
入力値 → ネガ濃度 → プリント濃度
```

つまり **入力値 → プリント濃度** が計算できます。

### 8.5 逆関数を作る

理想は **入力値 ∝ プリント濃度** なので

**逆カーブを作ります**。

```
補正カーブ = 逆(入力→プリント)
```

これを

- LUT
- グラデーションマップ
- トーンカーブ

で実装します。

### 8.6 この方法の利点

**普通のリニア化**

- 入力 → プリント だけ測る

**この方法**

- 入力 → ネガ
- ネガ → プリント

が分かるので **どこで歪んでいるかが分かる**。

| 問題 | 原因 |
|------|------|
| ハイライト詰まり | インク飽和 |
| 中間調急激 | 紙散乱 |
| シャドウ潰れ | 露光不足 |

### 8.7 和紙の場合の特徴

和紙では **ネガ濃度 → プリント濃度** が **弱いSカーブ** になります。

理由

- 紙内部散乱
- 感光層厚み
- 金属粒子分布

## 9. 露光思想とトーン配置

### 9.1 デジタルネガの本質

デジタルネガは **画像ではなく露光マップ** です。

つまり

**ネガ = 光量制御装置**

です。

### 9.2 プラチナプリントの本質

プラチナプリントは

**コントラストの写真ではなく、階調の写真** です。

特に **中間調** が最も重要になります。

### 9.3 ネガの直線部分を使う

「ネガの直線部分を使う」とは

**画像の重要なトーン（シャドウ〜ハイライト）を
フィルム特性曲線の straight-line region（直線部分）に収めること**

です。

**なぜ重要か**

直線部分では **露光量 ∝ 濃度** という性質があり、

**階調が正確に再現される** 領域だからです。

**toeを使うと**

- 濃度の立ち上がりが遅い
- 露光差 → 濃度差 が圧縮される
- 結果：**シャドウが潰れる**

**shoulderを使うと**

- 濃度が飽和に近づく
- 露光差 → 濃度差 が again 圧縮される
- 結果：**ハイライトが詰まる**

**理想のネガ**

```
重要トーン → straight line region に配置
```

```
濃度
↑
|           shoulder
|        /
|      /
|    /
|  /
|_/ toe
+----------------→ logE
      ↑        ↑
   シャドウ   ハイライト
```

### 9.4 プラチナプリントでは特に重要

プラチナプリントは **直線特性** なので

ネガが toe や shoulder を使うと

**そのままシャドウ潰れ・ハイライト詰まりになります**。

つまり

**プラチナではネガの直線部分を使うことが特に重要**。

### 9.5 ゾーンシステムとの関係

ゾーンシステムでは

| ゾーン | ネガ濃度 |
|--------|----------|
| Zone I | toe |
| Zone III | 直線開始 |
| Zone V | 中間 |
| Zone VII | 上部直線 |
| Zone VIII | shoulder手前 |

つまり **Zone III〜VII を straight line region に置く**。

### 9.6 写真家がやっていること

写真家は

```
露光 → シャドウ位置を決める
現像 → ハイライト位置を決める
```

つまり **曲線上で画像を配置する作業** をしています。

### 9.7 プラチナプリントのトーン構造

```
白
  ↓
柔らかいハイライト
  ↓
豊かな中間調
  ↓
深いシャドウ
```

ハイライトで勝負するプロセスではありません。

### 9.8 光の設計

プラチナに向く光

| 光 | 向き |
|----|------|
| 霧 | ◎ |
| 曇天 | ◎ |
| 逆光 | ◎ |
| 柔らかい斜光 | ◎ |
| 直射光 | △ |

つまり **グラデーションの光** です。

### 9.9 最終的な理解

プラチナプリントは

**コントラストの写真ではない**

**階調の写真である**

つまり

**光のグラデーションを記録する技法** です。

## まとめ

### デジタルネガ設計の核心

1. **3つの非線形を理解する**
   - インク → ネガ濃度
   - ネガ濃度 → 露光量
   - 露光量 → プリント濃度

2. **プリンターの線形領域を測る**
   - 入力値 → ネガ濃度のカーブ
   - インク飽和を知る

3. **2段階で測定する**
   - ①入力値 → ネガ濃度
   - ②ネガ濃度 → プリント濃度

4. **中間調を基準に設計する**
   - Zone V〜VI をネガ濃度 ≈ 0.7 に配置
   - ハイライト基準ではなく、中間調基準

5. **弱いSカーブを入れる**
   - 完全リニアは視覚的に眠い
   - シャドウ持ち上げ、中間強調、ハイライト圧縮

6. **和紙の特性を理解する**
   - 紙内部散乱 → 中間調が豊か
   - ネガγは少し強め（1.9〜2.2）
   - ネガDminを極端に低くする

7. **階調破壊を防ぐ**
   - 16bitで処理
   - 処理を最小限にする
   - 測定≠制御点（256測定 → 32点カーブ）

### 和紙プラチナプリント（Dmax ≈ 2.1、蛍光UV）の場合

| 項目 | 推奨値 |
|------|--------|
| ネガγ | 1.7〜1.9 |
| ネガ濃度レンジ | 1.6〜1.9 |
| プリント露光レンジ | 約6.6EV |
| 中間調配置 | ネガ濃度 ≈ 0.7 |
| カーブタイプ | 弱い逆Sカーブ |
| 制御点数 | 32点で十分 |
