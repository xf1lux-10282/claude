## 1. アルル/フェスティバル・ネットワーク(中心的ハブ)

これは**ヨーロッパ写真界で最も強力なネットワーキングの中心**です。

### 主要ハブ
レ・ランコントル・アルル(アルル国際写真フェスティバル)

### 主要な人間関係

**クリストフ・ヴィースナー(Christoph Wiesner)**
- ディレクター

頻繁に協働する相手:
- **タウス・ダーマニ(Taous Dahmani)** — アルルのルイ・ロデレール・ディスカバリー賞をキュレーション
- **カンタン・バジャック(Quentin Bajac)**
- **ナタリー・ヘルシュドルファー(Nathalie Herschdorfer)**
- **アンナ・テルグレン(Anna Tellgren)**

### なぜこれが重要か

ほぼすべてのヨーロッパの美術館キュレーターが毎年アルルに参加します。
多くの美術館展覧会は**アルルの展覧会として始まります**。

### 典型的な紹介チェーン

```
タウス・ダーマニ
     ↓
クリストフ・ヴィースナー
     ↓
カンタン・バジャック
     ↓
ヨーロッパの美術館展覧会
```

## 3. ロンドン写真ネットワーク

### 主要機関

- フォトグラファーズ・ギャラリー
- ヴィクトリア&アルバート美術館
- テート・モダン

### 主要な人間関係

**クレア・グラフィック(Clare Grafik)**
- 展覧会部門長

関係者:
- **シャーメイン・トー(Charmaine Toh)** — テートのキュレーター
- **タウス・ダーマニ(Taous Dahmani)** — フォトグラファーズ・ギャラリーのキュレーター
- **ドイツ取引所写真財団(Deutsche Börse Photography Foundation)**

**ドイツ取引所写真賞**は、共同展覧会プログラムを通じてロンドンとヨーロッパ、フランクフルトを結びつけています。

### ネットワーク構造

```
フォトグラファーズ・ギャラリー
          │
ドイツ取引所財団
          │
   Foam アムステルダム
```

この賞は**ヨーロッパにおける主要なキュレーター交流の場の一つ**となっています。

## 4. オランダ写真ネットワーク(極めて影響力が大きい)

### 主要機関

- Foam写真美術館アムステルダム
- オランダ写真美術館
- ハーグ写真美術館

### 主要人物

**ミリアム・クーイマン(Mirjam Kooiman)**
- Foamキュレーター

協働相手:
- **クラールチェ・ファン・ダイク(Claartje van Dijk)**
- **フリッツ・ギールストベルク(Frits Gierstberg)**
- **キム・クノッパース(Kim Knoppers)**

Foamのキュレーターたちは展覧会や研究プロジェクトを通じて頻繁に国際協働を行っています。

### 協働の例

ミリアム・クーイマンは、ドイツ取引所写真財団とマグナム展を共同キュレーションしました。

### ネットワークの影響力

```
Foam
 ├── PhotoElysée
 ├── フォトグラファーズ・ギャラリー
 └── ヨーロッパの写真集出版社
```

Foamは**写真集文化と新興写真家**とも深く結びついています。

## 戦略的洞察

**ヨーロッパの写真界は、約40〜50人の高度に結びついたキュレーターのみで構成されています。**

### 最も影響力のある5つのハブ

1. レ・ランコントル・アルル
2. Foam アムステルダム
3. フォトグラファーズ・ギャラリー ロンドン
4. Photo Elysée ローザンヌ
5. ジュ・ド・ポーム / MEP パリ

**各ハブで1人ずつにリーチできれば、ネットワークはあなたのために機能し始めます。**

## まとめ

この資料は、ヨーロッパの写真界における人脈構築の戦略的なロードマップを提供しています。約40〜50人の高度に結びついたキュレーターで構成されるこの世界では、5つの主要ハブ(アルル、Foam、フォトグラファーズ・ギャラリー、Photo Elysée、ジュ・ド・ポーム/MEP)それぞれで1人の関係者を得ることができれば、ネットワークが自然に機能し始めます。
