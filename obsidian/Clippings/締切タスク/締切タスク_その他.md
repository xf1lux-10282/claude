# 締切タスク｜その他

写真家活動・AP2Lab以外で**日付（締切）が決まっているタスク**の一覧です。
コンサルティング、広報、技術メモ関連など、ここに該当しない締切タスクをまとめます。

> 関連: [[締切タスク_写真家活動]] / [[締切タスク_AP2Lab]]

## 運用ルール

- **締切日が近い順**に並べる（進行中セクション内）
- 各タスクは「ステータス / 内容 / 関連ファイル / 返答予定日 / メモ」を記載
- 対応が済んだら **ステータスを更新**し、結果が確定したら下部の **## アーカイブ** へ移動する
- ステータス記号: 🔲未着手 / 🔄対応中 / 📤提出済み（結果待ち）/ ✅完了 / ❌見送り
- 日付は必ずシステムカレンダー（`date`）で確認してから記載する

### タスク記載フォーマット

```markdown
### [締切 YYYY-MM-DD] タスクタイトル

- **ステータス:** 🔲未着手 / 🔄対応中 / 📤提出済み / ✅完了 / ❌見送り
- **内容:** 何を、誰に、どこへ。簡潔に。
- **関連ファイル:**
  - [ファイル名](パス)
- **返答予定日 / 結果がわかる日:** YYYY-MM-DD（あれば）
- **メモ:** 補足・次にやること
```
