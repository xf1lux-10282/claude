## 1. 基本方針：役割分担

NAS と GitHub を競合させず、**ファイルの種類で住み分ける**。

```
┌─ テキストノート（.md 約620個）──→ [GitHub]
│   Obsidian Git で全端末同期・履歴・無料・iPhoneも可（既に稼働中）
│   ※ Claude作業のハブもGitHubのまま
│
└─ 大容量メディア（TIF/PSD/写真/PDF）──→ [UGREEN NAS]
    git に入らない重いファイルの正本・大容量・自己管理
    ※ .gitignore で画像/PDF/動画/officeは除外済み
```

**結論**: ノートと iPhone は GitHub で十分。Mac の全データ常時同期とメディアは NAS が担当。

## 2. iCloud 各機能の置き換え対応表

| iCloud機能 | NASでの代替 | 手段 |
|-----------|-------------|------|
| iCloud Drive（ファイル同期） | 常時双方向同期 | UGREEN純正Mac同期クライアント |
| デスクトップと書類フォルダ（複数Mac） | 複数Macで同じNASフォルダを双方向同期 | 同上（各Macに同期タスク） |

| 遠隔アクセス | 自宅外から安全に接続 | UGREENlink / Tailscale（任意） |

## 4. Mac の常時同期（iCloud Drive の代替）

- UGREEN には **Mac用デスクトップ同期クライアント**があり、フォルダを監視して**リアルタイム双方向同期**（Synology Drive / iCloud Drive と同等）。
- vault 本体をローカルに置き、NAS と常時同期 → オフライン編集可、つながった時に自動反映。
- **Docker 不要**（純正クライアントで完結）。より堅牢に組みたい場合は Syncthing（Docker・要 Container Manager）も選択肢。

### iCloud Drive → NAS 移行手順
1. iCloud Drive の「デスクトップと書類フォルダ」同期を**オフ**にする（二重同期を防ぐ）
2. vault を iCloud 配下からローカルの非iCloudパスへ移動
3. UGREEN純正クライアントで vault フォルダ ⇔ NAS の双方向同期タスクを作成
4. Obsidian の保管庫パスを新しいローカルパスに更新
5. 同期完了を確認

## 5. 複数Macでのデスクトップ／書類同期

UGREEN純正クライアントは**複数Macが同じNASフォルダを共有して双方向同期**できる。

```
[Mac A ~/Desktop, ~/Documents] ←双方向→ ┐
                                        ├─→ [NAS: Desktop/Documents 領域]（正本）
[Mac B ~/Desktop, ~/Documents] ←双方向→ ┘
```

設定：各Macに**同じNASユーザー**でサインインし、**同じNASフォルダ**を指す双方向同期タスクを作る。

### ⚠️ 複数台ならではの注意点
| 論点 | 対策 |
|------|------|
| iCloudと二重同期 | 先に iCloud「デスクトップと書類」を**オフ** |
| 初回マージ | A と B の中身が**合算**される。事前に整理してから接続 |
| 同時編集の競合 | 後保存が勝つ／競合コピー。同時編集が多いものは Google Docs 等を併用 |
| 削除の伝播 | 片方の削除が**もう片方からも消える**。NAS側のゴミ箱／スナップショット保護を有効化 |
| オンライン要件 | 反映は各MacがNASに繋がっている時。外出先は Tailscale 経由 |
| 不要ファイル | `.DS_Store` 等は除外設定 |

## 6. iPhone 写真の自動バックアップ（iCloud写真の代替）

- UGREEN写真／モバイル自動バックアップで、カメラロールを**自動バックグラウンドバックアップ**。
- HEIC / RAW / JPEG / MP4 / MOV 対応、**EXIF・GPS保持**、スクショ等の除外フィルタあり。
- 完全ローカル（外部クラウドに出ない）。写真家のRAW/フル解像度を 2TB NAS に集約する用途に好適。

### iCloud写真と"完全に同じ"ではない点
- **iPhoneストレージ最適化**（端末は軽量版・自動オフロード）は iCloud ほどシームレスでない（手動削除は可）。
- 閲覧は **UGREENアプリ経由**（Apple純正「写真」アプリとは別管理）。
- 外出先閲覧には遠隔アクセス（UGREENlink / Tailscale）が必要。
- 顔認識・AI検索は iCloud/Google Photos ほど成熟していない可能性（要・実機確認）。

## 7. iPhone ノート（どこでもアクセス）

- Obsidian の **Obsidian Git** プラグイン（iOSは Working Copy 併用）で GitHub と同期。
- メディアは持たずテキストのみ → 軽快。画像を見たい時は Obsidian/NASアプリで確認。

---

## 8. 遠隔アクセス（任意・"安心安全"の層）

| 方式 | 特徴 |
|------|------|
| UGREENlink | UGOS標準。手軽な遠隔接続 |
| Tailscale | Docker で導入。ポート開放不要・暗号トンネルで自宅LANに参加。外出先Mac/iPhoneを同期に含めるなら推奨 |

※ 現状のファイアウォールは「ローカルネットワークのみ許可」。遠隔を使う場合は Tailscale 経由（NAS直公開はしない）。
